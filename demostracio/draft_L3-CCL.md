## Reduction Theorem (gen = Rad2 ‚áî Core Completeness Lemma)

**Statement.** Define the Core Completeness Lemma (CCL): for every non-degenerate (a,c) in the band and every N, if Œ®(G,N) is satisfiable for some G‚àà{0,1}^W then Œ®(G*,N) is satisfiable for some G*‚ààF. **Proof.** By Lemma R+S, ‚Äò[1,N] colourable‚Äô ‚áî ‚ÄòŒ®(G,N) SAT for some G‚àà{0,1}^W‚Äô, and ‚Äòsome G‚ààF works‚Äô ‚áî ‚Äòfamily_decide(N) true‚Äô. ## Gaps
- MAIN GAP ‚Äî the Core Completeness Lemma (CCL), equivalently L3-completeness, is NOT proven in general. Precise open statement: for every non-degenerate (a,c) in the middle band and every N, if the fixed-gate 2-SAT Œ®(G,N) is satisfiable for SOME gate G‚àà{0,1}^W, then it is satisfiable for some ‚â§3-run gate G*‚ààF. Where the exchange argument stalls: fixing a gate G determines the per-colour forbidden-distance sets (D_0,D_1); the 2-SAT feasibility is a parity/alternation condition on walks in [1,N] whose steps alternate between D_0 and D_1 (a Cayley-graph bipartiteness condition). Changing G to reduce its run count changes (D_0,D_1) globally and is not monotone in any order I could establish, so a local run-reducing transposition that provably preserves satisfiability was not found. This walk-alternation structure is exactly the three-distance/Sturmian mechanism the project already identified as the reason there is no closed form, which is why a general proof is genuinely research-level and remained open in the project. STATUS OF VERIFICATION (my own, independent): (a) exhaustive brute force over ALL 2^m gates for a‚â§11 ‚Äî 2050 band parameter pairs, 6332 windows, checked at EVERY N‚â§gen ‚Äî family_decide == true_decide with ZERO mismatches, and at every extremal N=Rad2‚àí1 every valid gate is already ‚â§3 runs; (b) a fully independent pysat decider of the RAW equation (no window, gate, or family notion) equals gen on all 13792 SAT-computed CSV values (fullband 6192/6192 + midband 7600/7600) and on ~285 fresh out-of-sample values (172 random a‚àà{12..20}; bottom-of-band ranges a=13,14,15,17), zero mismatches; (c) all 1970 in-band SAT extremal witnesses reduce to ‚â§3-run gates.
- SECONDARY GAP ‚Äî the structural dichotomy that would likely power a proof is verified but unproven: (i) whenever the window is SINGLE-ARM (all owners on one side of lam, i.e. roughly N<lam, the deep-tail/Layer-2 regime), some 2-RUN gate is always valid (1847/1847 single-arm windows with m‚â•4 for a‚â§11, zero exceptions); (ii) whenever the window is V-SHAPED (straddles lam, bottom-of-band), a ‚â§3-run gate is always valid, with 3 runs genuinely required in 352 cases and 4 runs never required. A clean proof of (i) would settle the Layer-2 region; a proof of (ii) (the Sturmian region) is the hard core. Note: a natural stronger form of (i) ‚Äî that a global monotone block colouring 1^p0^q works in single-arm ‚Äî is FALSE (415 single-arm counterexamples for a‚â§11): the 2-run gate's valid extension is not itself a block colouring, so even the single-arm case needs the 2-SAT feasibility argument, not a one-line construction. A tempting cleaner conjecture (some distance-THRESHOLD gate always works) is also FALSE (104 counterexamples for a‚â§11, e.g. (7,24,15) where only 0111/1000 work), confirming the family must be indexed by owner-position contiguity (‚â§3 runs), not by distance.
- OUT OF SCOPE ‚Äî the Layer-2 pin predicate (Rad2 = lam‚àí1+[lam‚â§E1] with the stated E1) is a separate unproven closed form. The task explicitly targeted L3-completeness, so I did not attempt L2; it is independent of CCL (gen uses L2 only as a fast path where L3's decision procedure would also apply, so proving CCL suffices for gen's exactness even if the L2 shortcut were removed ‚Äî confirmed: setting MIDBAND_NO_L2 still yields gen==CSV on the checked ranges via L3).
