## Setup and Lemma Q0 (solution parametrization; normalization)

**Statement.** STANDING SETUP for all lemmas: fix integers d >= 2 and a with 1 <= a <= d; put c := a - d <= 0, N := (a+3)d + 1, E := d(a+1); note N - E = (a+3)d + 1 - (a+1)d = 2d + 1, and N >= 4d+1 (since a >= 1). A triple (x1,x2,x3) in [1,N]^3 is a SOLUTION if x1 + a*x2 - x3 = c; a 2-coloring chi: [1,N] -> {0,1} is VALID if no solution triple is monochromatic (entries may repeat). LEMMA: (i) the solutions in [1,N]^3 are exactly the triples T(x,k) := (x, k, x + a(k-1) + d) with x >= 1, k >= 1 and x + a(k-1) + d <= N; (ii) validity is invariant under the color swap 0 <-> 1, so if a valid coloring exists then one exists with chi(1) = 0.

**Proof.** (i) The equation x1 + a*x2 - x3 = c = a - d is equivalent to x3 = x1 + a(x2 - 1) + d. Write x := x1, k := x2, so every solution is T(x,k) and conversely. It remains to check that the three stated conditions already force all entries into [1,N]: x3 = x + a(k-1) + d >= 1 + 0 + d > 1; x <= x3 - d < N; and k <= N because a(k-1) = x3 - x - d <= N - 1 - d gives k - 1 <= (N-1-d)/a <= N - 1 (using a >= 1), so k <= N. (ii) The monochromaticity predicate chi(x1) = chi(x2) = chi(x3) is preserved when all colors are flipped, so chi is valid iff its swap is valid; if chi(1) = 1, replace chi by its swap. QED.

## Lemma Q1 (2-adic input: the existence condition yields gcd(a,2d) | d)

**Statement.** Assume v2(a) <= v2(c), where c = a - d and v2(0) := +infinity. Then v2(a) <= v2(d), and consequently g := gcd(a, 2d) divides d. (Conversely, if v2(a) > v2(d) then v2(a) > v2(c), so the hypothesis is exactly equivalent to v2(a) <= v2(d); numerically verified both ways for all 1 <= a <= d <= 500.)

**Proof.** Since v2(-x) = v2(x), we have v2(c) = v2(a - d) = v2(d - a). First claim: v2(a) <= v2(c) iff v2(a) <= v2(d). If v2(a) > v2(d) (in particular d != a, so both valuations are finite and distinct), the ultrametric equality gives v2(d - a) = min(v2(d), v2(a)) = v2(d) < v2(a), so v2(a) > v2(c) and the hypothesis fails. If v2(a) <= v2(d), then v2(d - a) >= min(v2(d), v2(a)) = v2(a) (this includes d = a, where v2(0) = +infinity), so the hypothesis holds. Hence the hypothesis forces e := v2(d) >= v2(a). Second claim: g | d. Write d = 2^e * m with m odd. Then v2(2d) = e + 1 > v2(a), so v2(g) = min(v2(a), v2(2d)) = v2(a) <= e. The odd part of g divides the odd part of 2d, which equals m. A positive integer divides d = 2^e * m if and only if its 2-adic valuation is at most e and its odd part divides m; g satisfies both conditions, so g | d. QED.

## Lemma Q2 (basic forcing rules)

**Statement.** Let chi be valid on [1,N] with chi(1) = 0. Then: (i) chi(d+1) = 1; (Z) for every x in [1, N-d]: chi(x) = 0 implies chi(x+d) = 1; (O) for every x in [1, 2d+1]: chi(x) = 1 implies chi(x+E) = 0.

**Proof.** (Z) T(x,1) = (x, 1, x + a*0 + d) = (x, 1, x+d) is a solution triple, in range because x >= 1 and x + d <= N. Its first two entries have colors chi(x) = 0 and chi(1) = 0; if chi(x+d) = 0 the triple would be monochromatic, contradicting validity. Hence chi(x+d) = 1. (i) Apply (Z) at x = 1: the range condition 1 <= N - d holds since N = (a+3)d + 1 >= 4d + 1; the triple used is T(1,1) = (1,1,d+1) (repeated entries are permitted). Hence chi(d+1) = 1. (O) T(x, d+1) = (x, d+1, x + a*d + d) = (x, d+1, x + E) is a solution triple; it lies in range iff x + E <= N, i.e. x <= N - E = 2d + 1. Its second entry has color chi(d+1) = 1 by (i); if chi(x) = 1 and chi(x+E) = 1 the triple would be monochromatic. Hence chi(x+E) = 0. QED.

## Lemma Q3 (pairing on [1, d+1]; the pattern p)

**Statement.** Let chi be valid on [1,N] with chi(1) = 0. Then chi(s + d) = 1 - chi(s) for EVERY s in [1, d+1]. Consequently, defining phi(s) := chi(s) for s in [1,d] (so phi(1) = 0) and letting p: Z -> {0,1} be the unique 2d-periodic function with p(s) = phi(s) and p(s + d) = 1 - phi(s) for s in [1,d], the function p is antisymmetric (p(u + d) = 1 - p(u) for all u in Z) and chi(u) = p(u) for all u in [1, 2d+1]. (The instance s = d+1 of the pairing is the unique step of the whole proof that uses the last cell N; it is exactly what fails at length N-1, and it yields chi(2d+1) = 0.)

**Proof.** Fix s in [1, d+1]. NOT BOTH 0: T(s, 1) = (s, 1, s+d) is in range (s + d <= 2d + 1 <= N). If chi(s) = chi(s+d) = 0 then, together with chi(1) = 0, it is monochromatic - contradiction. NOT BOTH 1: suppose chi(s) = 1 = chi(s+d). Rule (O) at x = s (range: s <= d+1 <= 2d+1) gives chi(s + E) = 0. Rule (O) at x = s + d (range: s + d <= 2d + 1, i.e. s <= d+1; and x + E = s + d + E <= E + 2d + 1 = N) gives chi(s + d + E) = 0. Rule (Z) at x = s + E (range: s + E <= N - d iff s <= N - E - d = d + 1, which holds) gives, from chi(s+E) = 0, that chi(s + E + d) = 1 - contradicting chi(s + d + E) = 0. Hence exactly one of chi(s), chi(s+d) equals 1, i.e. chi(s+d) = 1 - chi(s). CONSEQUENCES: p is well defined because every residue class mod 2d has a unique representative in [1, 2d], of the form s or s + d with s in [1, d]. Antisymmetry: if u == s (mod 2d), s in [1,d], then u + d == s + d and p(u+d) = 1 - phi(s) = 1 - p(u); if u == s + d, then u + d == s (mod 2d) and p(u+d) = phi(s) = 1 - (1 - phi(s)) = 1 - p(u). Finally chi = p on [1, 2d+1]: on [1, d] this is the definition of phi; for u in [d+1, 2d], pairing at s = u - d in [1, d] gives chi(u) = 1 - phi(u-d) = p(u); and at u = 2d + 1, pairing at s = d + 1 gives chi(2d+1) = 1 - chi(d+1) = 1 - 1 = 0 (using Q2(i)), which equals p(2d+1) = p(1) = phi(1) = 0. QED.

## Lemma Q4 (far zone, from the pairing alone - no window lemma)

**Statement.** Let chi be valid on [1,N] with chi(1) = 0, and let p be as in Lemma Q3. Then chi(E + u) = 1 - p(u) for every u in [1, 2d+1]; equivalently chi(z) = 1 - p(z - E) for every z in [E+1, N] (note E + (2d+1) = N).

**Proof.** By Lemma Q3, chi(u) = p(u) is known for all u in [1, 2d+1]. CASE 1: p(u) = 1. Then chi(u) = 1 and rule (O) at x = u (range: u <= 2d+1) gives chi(E + u) = 0 = 1 - p(u). CASE 2: p(u) = 0. Observe u != d + 1, since p(d+1) = 1 - phi(1) = 1; so either u <= d or d + 2 <= u <= 2d + 1. Sub-case u <= d: put v := u + d <= 2d <= 2d+1. Antisymmetry gives p(v) = 1 - p(u) = 1, so Case 1 applied at v yields chi(E + v) = chi(E + u + d) = 0. The triple T(E+u, 1) = (E+u, 1, E+u+d) is in range (x3 = E + u + d <= E + 2d = N - 1 <= N). Its entries at positions 2 and 3 have colors chi(1) = 0 and chi(E+u+d) = 0; if chi(E+u) = 0 the triple is monochromatic - contradiction. Hence chi(E+u) = 1 = 1 - p(u). Sub-case u in [d+2, 2d+1]: put w := u - d in [2, d+1]. Antisymmetry gives p(w) = 1 - p(w + d) = 1 - p(u) = 1, so Case 1 at w (range: w <= d+1 <= 2d+1) yields chi(E + w) = 0. The triple T(E+w, 1) = (E+w, 1, E+w+d) = (E+w, 1, E+u) is in range (x3 = E + u <= E + 2d + 1 = N; x1 = E + w >= 1). Its entries at positions 1 and 2 have colors chi(E+w) = 0 = chi(1); if chi(E+u) = 0 the triple is monochromatic - contradiction. Hence chi(E+u) = 1 = 1 - p(u). QED.

## Lemma Q5 (transport)

**Statement.** Let chi be valid on [1,N] with chi(1) = 0. Let k0 in [1,N] satisfy chi(k0) = 1 and put D := a(k0 - 1) + d. Then for every y >= 1 with chi(y) = 1 and y + D + d <= N: chi(y + d) = 0 and chi(y + 2d) = 1.

**Proof.** Four solution triples, each with its range check (entries may repeat). STEP 1: T(y, k0) = (y, k0, y + D); range: x3 = y + D <= N - d < N by hypothesis. Since chi(y) = chi(k0) = 1, monochromaticity would follow from chi(y + D) = 1; hence chi(y + D) = 0. STEP 2: T(y + D, 1) = (y + D, 1, y + D + d); range: x3 = y + D + d <= N by hypothesis. Since chi(y + D) = 0 = chi(1), we get chi(y + D + d) = 1. STEP 3: T(y + d, k0) = (y + d, k0, y + d + D); range: x1 = y + d >= 1 and x3 = y + D + d <= N. Since chi(k0) = 1 = chi(y + d + D), monochromaticity would follow from chi(y + d) = 1; hence chi(y + d) = 0. STEP 4: T(y + d, 1) = (y + d, 1, y + 2d); range: x3 = y + 2d <= y + D + d <= N, using D = a(k0 - 1) + d >= d. Since chi(y + d) = 0 = chi(1), we get chi(y + 2d) = 1. QED.

## Lemma Q6 (one-step window extension - Case A; replaces the inductive window lemma)

**Statement.** Let chi be valid on [1,N] with chi(1) = 0, and suppose phi(s0) = 1 for some s0 in [2, d]. Then chi(u) = p(u) for every u in [1, a + 2d]. (Because a <= d, the required extension beyond the pairing zone [1, 2d+1] is at most a - 1 <= d - 1 further cells, each reached by a SINGLE application of transport - no induction is needed, in contrast to the q >= 1 regime where a >= 2d+1.)

**Proof.** For u in [1, 2d+1] this is Lemma Q3. So let u in [2d+2, a+2d] (nonempty only if a >= 2) and write u = s + 2d with s := u - 2d in [2, a]; note s in [1, d] since a <= d, and p(u) = p(s). Put D0 := a(s0 - 1) + d; since 2 <= s0 <= d, we have a + d <= D0 <= a(d-1) + d = E - a. CASE phi(s) = 1: apply transport (Lemma Q5) with k0 = s0 (legitimate: chi(s0) = phi(s0) = 1) at y = s. Room: y + D0 + d <= a + (E - a) + d = E + d <= E + 2d + 1 = N. Transport gives chi(s + 2d) = 1 = phi(s) = p(u). CASE phi(s) = 0: by Lemma Q3, chi(s + d) = 1 - phi(s) = 1. Apply transport with k0 = s0 at y = s + d. Room: y + D0 + d <= (a + d) + (E - a) + d = E + 2d = N - 1 <= N. Transport gives chi(y + d) = chi(s + 2d) = 0 = phi(s) = p(u). In both cases chi(u) = p(u). QED.

## Lemma Q7 (translation pigeonhole in Z/2d)

**Statement.** Let d >= 1 and let p: Z -> {0,1} be 2d-periodic and antisymmetric (p(u + d) = 1 - p(u) for all u). Let a >= 1 satisfy gcd(a, 2d) | d, and set eps := p(d). Then there exists u in [1, 2d] with p(u) = 1 - eps and p(u + a) = eps. (This is purely combinatorial - independent of the coloring. The divisibility hypothesis cannot be dropped: if gcd(a,2d) does not divide d, i.e. v2(a) > v2(d), the pattern p(u) = bit_e(u) with e = v2(d) is antisymmetric and admits no such u; this is the parity obstruction that makes Rad2 nonexistent there.)

**Proof.** Regard p as a function on Z/2d (representatives [1, 2d]) and put P := { v in Z/2d : p(v) = eps }. Suppose for contradiction that no such u exists. Then for every v in Z/2d: p(v + a) = eps implies p(v) = eps; that is, P - a is a subset of P, where P - a := { v : v + a in P }. Translation by a is a bijection of the finite set Z/2d, so |P - a| = |P|, and a subset of P with the same finite cardinality equals P: P - a = P. Iterating, P - ja = P for every integer j >= 0. The set { ja mod 2d : j >= 0 } is the subgroup of Z/2d generated by a, which equals g*(Z/2d) with g := gcd(a, 2d); hence P is invariant under translation by every multiple of g. Since g | d, in particular P - d = P, equivalently P + d = P. Now d is in P, because p(d) = eps by definition; therefore 2d = d + d is in P, i.e. p(2d) = eps. But antisymmetry gives p(2d) = p(d + d) = 1 - p(d) = 1 - eps, and eps != 1 - eps in {0,1} - a contradiction. Hence the required u exists; choose its representative in [1, 2d]. QED. (Verified exhaustively: all d <= 13, ALL a in [1, 2d] with gcd(a,2d) | d, ALL 2^d antisymmetric patterns - 240,630 instances - plus 1,638 random instances with d up to 1000; zero failures. Necessity of the gcd hypothesis also verified.)

## Lemma Q8 (endgame, Case A: phi not identically 0)

**Statement.** Under the standing setup with the existence condition v2(a) <= v2(c): there is no valid coloring chi of [1,N] with chi(1) = 0 and phi(s0) = 1 for some s0 in [2, d]. Explicitly, with eps := phi(d) and u given by Lemma Q7, the solution triple (a + u, d, E + u) is monochromatic of color eps.

**Proof.** Suppose chi were such a coloring; let phi, p be as in Lemma Q3 and set eps := phi(d) = p(d). By Lemma Q1, gcd(a, 2d) | d, so Lemma Q7 provides u in [1, 2d] with p(u) = 1 - eps and p(u + a) = eps. Consider T(a + u, d) = (a + u, d, (a + u) + a(d - 1) + d) = (a + u, d, E + u), using (a + u) + a(d-1) + d = u + ad + d = u + E. RANGES: x1 = a + u >= a + 1 >= 2; x2 = d in [1, N]; x3 = E + u <= E + 2d = N - 1 <= N. FORCED COLORS: (1) x1 = a + u <= a + 2d, so Lemma Q6 (applicable since s0 exists) gives chi(a + u) = p(a + u) = p(u + a) = eps by the choice of u. (2) chi(d) = phi(d) = eps, since d in [1, d]. (3) x3 = E + u with u in [1, 2d], a subset of [1, 2d+1], so Lemma Q4 gives chi(E + u) = 1 - p(u) = 1 - (1 - eps) = eps. All three entries have color eps: the triple is monochromatic, contradicting validity. QED. (Worked instance: a = 3, d = 4, N = 25, phi = 0100: eps = 0, u = 8, triple (11, 4, 24); note x1 = 11 = a + 2d uses the LAST cell of the extended window, so the range a + 2d in Lemma Q6 is exactly what is needed.)

## Lemma Q9 (endgame, Case B: phi identically 0)

**Statement.** Under the standing setup (d >= 2, 1 <= a <= d; NO 2-adic hypothesis needed here): there is no valid coloring chi of [1,N] with chi(1) = 0 and phi(s) = 0 for all s in [1, d]. Explicitly, the solution triple (d + 1 - a, 2, 2d + 1) is monochromatic of color 0. (This kill needs only the pairing of Lemma Q3 - no window extension and no far zone.)

**Proof.** Suppose chi were such a coloring. Consider T(d + 1 - a, 2) = (d + 1 - a, 2, (d + 1 - a) + a + d) = (d + 1 - a, 2, 2d + 1); the equation check: (d+1-a) + 2a - (2d+1) = a - d = c. RANGES: x1 = d + 1 - a in [1, d], since 1 <= a <= d gives 1 <= d + 1 - a <= d; x2 = 2 <= N; x3 = 2d + 1 <= N (N >= 4d + 1). FORCED COLORS: (1) chi(d + 1 - a) = phi(d + 1 - a) = 0, since d + 1 - a in [1, d] and phi is identically 0. (2) chi(2) = phi(2) = 0, using d >= 2 so that 2 in [1, d] (this is the only place d >= 2 is indispensable, and it is why the single parameter point (a, c) = (1, 0), d = 1, is excluded). (3) chi(2d + 1) = 0 by Lemma Q3 (the pairing instance s = d + 1: chi(2d+1) = 1 - chi(d+1) = 0). All three entries have color 0: monochromatic, contradicting validity. QED.

## Theorem (upper bound of Conjecture 1) and tightness remark

**Statement.** THEOREM: Let a >= 1 and c <= 0 with v2(a) <= v2(c) and d := a - c >= 2. Then every 2-coloring of [1, (a+3)(a-c) + 1] contains a monochromatic solution of x1 + a*x2 - x3 = c; hence Rad2(a; c) <= (a+3)(a-c) + 1. Combined with the Dwivedi-Tripathi lower bound Rad2(a; c) >= (a+3)(a-c) + 1 (their Theorem 2, citable), Rad2(a; c) = (a+3)(a-c) + 1. IN PARTICULAR the target statement holds: for every a >= 3 and c <= 0 with v2(a) <= v2(c) (there d = a - c >= a >= 3), every 2-coloring of [1, (a+3)(a-c)+1] contains a monochromatic solution - i.e. Conjecture 1 of Dwivedi-Tripathi (Integers 20 (2020) #A36) is TRUE in all its open cases. (The only parameter point with c <= 0 and v2(a) <= v2(c) not covered is (a,c) = (1,0), where d = 1; that is the classical Schur equation x1 + x2 - x3 = 0 with Rad2(1;0) = 5 = (1+3)(1)+1, known and outside the target.) TIGHTNESS: at length N - 1 the unique failing step is the pairing instance s = d+1 (it uses cell N = E + 2d + 1); everything else, including the whole of Case A, survives at N - 1, so every valid coloring of [1, N-1] with chi(1) = 0 has phi identically 0 - in agreement with all 925 machine-found extremal witnesses, which after normalization are exactly the 3-block coloring 0^d 1^{(a+1)d} 0^d.

**Proof.** Since c <= 0, a <= a - c = d, and d >= 2 by hypothesis; so the standing setup applies with N = (a+3)d + 1. Suppose some 2-coloring chi of [1, N] has no monochromatic solution. By Lemma Q0(ii), normalize chi(1) = 0. Let phi := chi restricted to [1, d] (Lemma Q3), so phi(1) = 0. Exactly one of the following holds: (a) phi(s0) = 1 for some s0 in [2, d] - impossible by Lemma Q8, whose 2-adic input gcd(a, 2d) | d is supplied by Lemma Q1 from the hypothesis v2(a) <= v2(c); or (b) phi(s) = 0 for all s in [1, d] - impossible by Lemma Q9. Hence no valid coloring of [1, N] exists, i.e. Rad2(a; c) <= N = (a+3)(a-c) + 1. The reverse inequality is Dwivedi-Tripathi Theorem 2 (also re-verified here: the 3-block coloring 0 on [1,d], 1 on [d+1,(a+2)d], 0 on [(a+2)d+1,(a+3)d] is solution-free at length N-1; checked mechanically for all admissible pairs with d <= 40). For the target statement take a >= 3: then d = a - c >= a >= 3 >= 2. For the tightness remark: at length N-1, rules (Z) and (O) hold with ranges shrunk by one cell (x <= N-1-d and x <= 2d respectively), which still supports the pairing on s in [1, d] (its worst range use is s + d + E <= E + 2d = N - 1), the far zone for u in [1, 2d] (the p(u)=0, u <= d sub-case uses E+u+d <= E+2d = N-1; u = d+1 is always in the p=1 case), and the Case-A extension (worst room (a+d) + (E-a) + d = N - 1) and kill (x3 = E + u <= E + 2d = N - 1) - so Case A still eliminates every phi not identically 0 at length N-1; only the Case-B kill dies, exactly because chi(2d+1) is no longer forced when cell N is absent. QED.

Two scope notes for the record: (1) the single parameter point (a, c) = (1, 0) (d = 1) is outside the argument because Lemma Q9 needs 2 in [1, d]; it is the classical Schur case, Rad2(1;0) = 5, consistent with the formula and already known (DT T3/T4) - and it is not in the target since a >= 3. - Independence caveat, stated honestly: the proof was constructed from scratch for the q = 0 regime and its overall shape differs from the q >= 1 write-up (pairing extended to s = d+1 via a 3-triple argument; far zone derived from pairing alone with no window lemma; no inductive window lemma - a single transport step suffices because a <= d; Case B killed with no window at all, explaining tightness transparently). This should be counted as convergent evidence rather than a fully disjoint second proof of the Case-A endgame.
