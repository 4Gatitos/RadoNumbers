# CCL-V (V-core of the Core Completeness Lemma): for every middle-band pair (a,c) (a<c<a(K+2), a∤c, v2(a)<=v2(c)) with N = Rad2(a;c)-1 and W(N) V-shaped with m>=4 owners (N>=lam, both valley owners lam-1, lam active with distances t = c mod a and mu = a-t), some valid colouring of [1,N] has a gate (restriction to W) with <= 3 runs.

**status**: complete

## Summary

CCL-V is PROVEN — in a form far stronger than required: for EVERY colourable N (not only the extremal one) whose window is V-shaped with m>=4, EVERY valid colouring chi has gate x g^(m-2) y where g = chi(lam-1) = chi(lam) — i.e. the valley pair is equal-coloured and every non-outermost owner carries the valley colour; such a word has at most 3 runs, hence lies in F. Since [1, Rad2-1] is colourable by definition, its own valid colouring supplies the F-gate: CCL-V follows, and with the already-proven Theorem SA (single-arm), Lemma V0 (trichotomy) and the trivial m<=3 case, CCL holds IN FULL; by Theorem E, gen(a,c) = Rad2(a;c) for every middle-band pair, closing the last open regime of the problem. The proof rests on two new seedless infeasibility lemmas whose only arithmetic input is the 2-adic existence condition. (1) Lemma PAIR: if v2(P) != v2(Q) and N >= P+Q+max(P,Q), no 2-colouring of [1,N] simultaneously avoids 1-pairs at distance P and 0-pairs at distance Q; proof: the forced-1 propagation map F(x) = x+(P+Q) / x-(max-min) is total, enters the window of the top 2·max positions, acts there as the rotation by P+Q on Z_{2max}, and its orbit — a full coset of gcd(P+Q, 2max), which divides max exactly because v2(P) != v2(Q) — contains two positions at exact distance max: a forbidden monochromatic pair. Applied to (P,Q) = (t,mu) (v2(t) != v2(mu) always, since v2(t)=v2(mu) would force v2(a)>v2(t)), and using that m>=4 guarantees N >= a+max(t,mu)+1, this kills every split valley (Lemma VAL) — including all 63 corner instances with N<2a. (2) Lemma WINDOW: for v2(a)<=v2(t), no two subsets u,v of a window of a+1 consecutive integers, each avoiding distances t and mu, can cover the window; proof: the walk y -> y+t (else y-mu) closes into an explicit cycle of length n = a/gcd(a,t) inside the window whose edges are distance-t/mu pairs, n is ODD exactly because v2(a)<=v2(t), and an odd n-cycle cannot be covered by two independent sets (2·(n-1)/2 < n). The converse also holds (v2(a)>v2(t) makes the window graph bipartite and the system feasible), so the 2-adic hypothesis is exactly the driver. (3) Lemma INTERIOR: in a valid colouring with equal valley colour g, any owner k with chi(k) != g and d_k + a <= N-1 yields, via u(y)=chi(y), v(y)=chi(y+d_k) on [1,a+1], two {t,mu}-independent window sets (1s avoid t and mu — the two valley owners) that cover the window (0s avoid d_k) — contradicting WINDOW. Every non-outermost arm owner satisfies d_k + a <= N-1 (the next-outward owner is active), which yields Theorem V-RIGID and the corollaries. Everything was mechanically verified end-to-end: PAIR orbit certificates on 1064 (P,Q) grids + SAT infeasibility at the exact threshold + feasibility converse; WINDOW odd-cycle/bipartite dichotomy on 44,849 (a,t) pairs (a<=300); the gate law on all 1203 extremal V-core instances a<=13 (all 2^m gates: valid ones exactly of form x g^(m-2) y, zero violations), on all 744 colourable V-core instances at EVERY N for a<=9, and on 174 fresh out-of-sample extremal instances at a=14..17 with Rad2 recomputed from raw SAT. A consolidated verifier was added to the repository (demostracio/verify_vcore_ccl.py — all checks pass).

## Lemma 2AD (2-adic facts)

**Statement.** Let a >= 3, t in [1,a-1], mu = a-t, and suppose v2(a) <= v2(t) (the existence condition transported to the band by the proven Lemma V2 of draft_assalt_CCL-transfer.md). Then: (i) v2(t) != v2(mu); (ii) a != 2t; (iii) n := a/gcd(a,t) is odd and n >= 3; (iv) v2(a) = min(v2(t), v2(mu)).

**Proof.** (i) If v2(t) = v2(mu) = e, write t = 2^e t', mu = 2^e m' with t', m' odd; then a = t + mu = 2^e (t'+m') with t'+m' even, so v2(a) >= e+1 > e = v2(t), contradicting v2(a) <= v2(t). (ii) a = 2t gives v2(a) = v2(t)+1 > v2(t). (iii) Let g1 = gcd(a,t); v2(g1) = min(v2(a), v2(t)) = v2(a), so v2(n) = v2(a) - v2(g1) = 0: n is odd. n = 1 would mean a = g1 | t with t < a, impossible; odd and != 1 gives n >= 3. (iv) By (i) the valuations of t and mu differ, so v2(t+mu) = min(v2(t), v2(mu)); and t + mu = a. [Exhaustively machine-checked for all a <= 2000.]

## Lemma PAIR (seedless two-distance infeasibility)

**Statement.** Let P, Q >= 1 be integers with v2(P) != v2(Q) (hence P != Q), and let N >= P + Q + max(P,Q). Then no chi: [1,N] -> {0,1} satisfies simultaneously: (no p with chi(p) = chi(p+P) = 1, p+P <= N) and (no p with chi(p) = chi(p+Q) = 0, p+Q <= N). Moreover the 2-adic hypothesis is exactly right: if v2(P) = v2(Q) = e, the colouring by alternating blocks of size 2^e satisfies both constraints on [1,N] for every N.

**Proof.** Complementation chi -> 1-chi swaps the two constraints with P and Q exchanged, so WLOG the 1-avoided distance is W := max(P,Q) and the 0-avoided is w := min(P,Q). Put S := W + w = P + Q, B := W - w >= 1. All-0 is impossible: (1, 1+w) would be a 0-pair at distance w (1+w <= N). So chi(x0) = 1 for some x0. Forcing: (F1) if chi(x)=1 and x <= N-S, then x+W <= N forces chi(x+W)=0 (1-pair at W), and x+S <= N forces chi(x+S)=1 (0-pair at w on (x+W, x+S)); (F2) if chi(x)=1 and x >= N-S+1, then x-W >= N-S+1-W >= 1 (using N >= S+W) forces chi(x-W)=0, and the pair (x-W, x-B) forces chi(x-B)=1 (1 <= x-B < x <= N). Hence F(x) := x+S (x <= N-S), x-B (x > N-S) is total on [1,N] and propagates the value 1. The window Omega := [N-2W+1, N] (nonempty and >= 1 since N >= S+W > 2W) is F-invariant: from x <= N-S, x+S in [N-2W+1+S, N] subset Omega; from x > N-S, x-B >= N-(S+B)+1 = N-2W+1. The orbit of x0 enters Omega (repeated +S steps; the first iterate exceeding N-S lies in (N-S, N] subset Omega because S <= 2W). On Omega, which contains exactly one representative of each residue mod 2W, F(x) === x + S (mod 2W), i.e. F restricted to Omega is the rotation by S of Z_{2W}; the forward orbit of the entry point e is the full coset {x in Omega : x === e (mod g')}, g' := gcd(S, 2W). Divisibility: odd(g') | odd(2W) = odd(W); and v2(S) = min(v2(P), v2(Q)) <= v2(W) (unequal valuations, ultrametric equality), while v2(2W) = v2(W)+1, so v2(g') = v2(S) <= v2(W); hence g' | W. Now [N-2W+1, N-W] is a block of W >= g' consecutive integers, so it contains w* === e (mod g'); then w*+W is in Omega and w*+W === w* (mod g') since g' | W. Both lie in the orbit, so chi(w*) = chi(w*+W) = 1 — a forbidden 1-pair at distance W. Contradiction. For the converse: if v2(P)=v2(Q)=e, colour chi(x) = floor((x-1)/2^e) mod 2; any pair at distance P = 2^e P' (P' odd) joins blocks of opposite parity, so it is never monochromatic, and likewise for Q: both constraints hold for every N. [Mechanically verified: orbit certificate replayed step-by-step, including every range inequality, for all P,Q <= 40 with v2 mismatch at N = P+Q+max and N = P+Q+max+7; infeasibility SAT-confirmed at the exact threshold for all P,Q <= 25; converse feasibility SAT-confirmed for all P,Q <= 40 with equal v2.]

## Lemma VAL (valley equality)

**Statement.** Let (a,c) be a non-degenerate middle-band pair with Rad2 existing, and let N satisfy N >= lam with W(N) V-shaped with m >= 4 owners (so both valley owners lam-1 and lam are active, by the proven Lemma V0(iii)). Then every valid colouring chi of [1,N] has chi(lam-1) = chi(lam).

**Proof.** First, m >= 4 forces N >= a + max(t,mu) + 1: if L >= 2 and R >= 2 then both owner lam-2 (distance t+a) and owner lam+1 (distance mu+a) are active, so N-1 >= a + max(t,mu); if R = 1 then L >= 3 and owner lam-3 is active: N >= 2a+t+1 > 2a >= a + max(t,mu) + 1; if L = 1 then R >= 3 and symmetrically N >= 2a+mu+1. (Exhaustive; L,R >= 1 by Lemma V0(iii).) Now suppose chi(lam-1) != chi(lam). Complementing chi if necessary, assume chi(lam-1) = 1, chi(lam) = 0. Owner lam-1 (distance t, colour 1) forbids 1-pairs at distance t; owner lam (distance mu, colour 0) forbids 0-pairs at distance mu. By Lemma 2AD(i), v2(t) != v2(mu), and N >= a + max(t,mu) + 1 > t + mu + max(t,mu). Lemma PAIR with (P,Q) = (t,mu) gives a contradiction. [Corner check: this covers all 63 V-core extremal instances with N < 2a; no separate argument needed.]

## Lemma WINDOW (odd-cycle window covering)

**Statement.** Let a >= 3, t in [1,a-1], mu = a-t, v2(a) <= v2(t). There do NOT exist sets u, v, each a subset of a window of a+1 consecutive integers (say {0,1,...,a}), such that (A) neither u nor v contains a pair of elements at distance t or at distance mu, and (B) u union v = {0,...,a}. Conversely, if v2(a) > v2(t) such u, v exist (the window graph is bipartite).

**Proof.** Let g1 = gcd(a,t) and n = a/g1; by Lemma 2AD(iii) n is odd, n >= 3. Define the walk y_0 = t, y_{i+1} = y_i + t if y_i + t <= a, else y_i - mu. By induction every y_i lies in [1,a]: in the second branch y_i > a - t = mu so y_i - mu in [1,t]. In both branches y_{i+1} === y_i + t (mod a) (since -mu === t). Hence y_i === (i+1)t (mod a); the residues t, 2t, ..., nt are pairwise distinct mod a (the order of t in (Z_a,+) is n), and each residue has exactly one representative in [1,a], so y_0, ..., y_{n-1} are distinct; y_n === (n+1)t === t and y_n in [1,a] force y_n = y_0: the walk closes. Consecutive walk points differ by exactly t or mu and all lie in [0,a]: C := (y_0, ..., y_{n-1}) is a cycle of ODD length n >= 3 in the graph G on {0..a} whose edges are pairs at distance t or mu. Now suppose u, v satisfy (A)+(B). Then u' := u ∩ V(C) and v' := v ∩ V(C) are independent sets of the n-cycle C (its edges are distance-t/mu pairs, forbidden inside u and inside v), so |u'| <= floor(n/2) = (n-1)/2 and likewise |v'|; but (B) gives |u'| + |v'| >= n. Hence n <= n-1, a contradiction. Converse: if v2(a) > v2(t), then n is even and the graph G is bipartite (machine-checked; the parts of the bipartition give u, v satisfying (A)+(B)). [Mechanically verified: the odd-cycle construction (ranges, distinctness, closure, edge distances) for all admissible (a,t) with a <= 300 — 44,849 pairs together with the bipartite converse; independently, SAT confirms (A)+(B) infeasible for ALL admissible (a,t) with a <= 80 and feasible for all non-admissible ones.]

## Lemma INTERIOR (interior owners carry the valley colour)

**Statement.** Setting of Lemma VAL, with chi valid and g := chi(lam-1) = chi(lam). Let k be any owner (either arm) with distance D := d_k such that D + a <= N - 1. Then chi(k) = g. In particular every NON-OUTERMOST arm owner satisfies this (the next owner outward on its arm is active with distance D + a <= N-1), so all owners except possibly the two outermost ones have colour g.

**Proof.** Suppose chi(k) = 1 - g; complementing, assume g = 1. Three constraint families hold: no 1-pair at distance t (owner lam-1, colour 1), no 1-pair at distance mu (owner lam, colour 1), no 0-pair at distance D (owner k, colour 0). Also N >= D + a + 1. Define u := {y-1 : y in [1,a+1], chi(y) = 1} and v := {y-1 : y in [1,a+1], chi(y+D) = 1}, subsets of {0..a} (note y + D <= a+1+D <= N). (A): a pair of u at distance t or mu would be a monochromatic 1-pair of chi at that distance inside [1,a+1] — forbidden; a pair of v at distance t or mu would be a 1-pair of chi inside [D+1, D+a+1] subset [1,N] — forbidden. (B): if y-1 is in neither u nor v then chi(y) = chi(y+D) = 0, a 0-pair at distance D — forbidden; so u union v = {0..a}. This contradicts Lemma WINDOW (its hypothesis v2(a) <= v2(t) is the existence condition). For the 'in particular': the window is a set of consecutive integers (proven Lemma W(ii) of draft_L3-CCL.md); if the owner at position lam-1-h (left arm, h <= L-2) is not outermost, the owner at lam-2-h is in the window, hence active, and its distance is exactly (t+ah) + a <= N-1; symmetrically on the right arm with D = mu+ah.

## Theorem V-RIGID (gate rigidity in the V-core)

**Statement.** Let (a,c) be a non-degenerate middle-band pair with Rad2 existing, and N >= lam with m >= 4 owners (V-shaped window). Then EVERY valid colouring chi of [1,N] has gate (reading the owner interval left to right) equal to x g^(m-2) y for some x,y in {0,1}, where g = chi(lam-1) = chi(lam). Such a word has at most 3 maximal runs, hence lies in the canonical family F of the generator.

**Proof.** Lemma VAL gives chi(lam-1) = chi(lam) = g. The owner positions form the interval [k_lo, k_hi] containing lam-1, lam; every owner except the leftmost (k_lo) and rightmost (k_hi) is a non-outermost arm owner, so Lemma INTERIOR gives it colour g. The gate is therefore chi(k_lo), g, g, ..., g, chi(k_hi) = x g^(m-2) y with m-2 >= 2 central g's covering both valley owners. Its run count is at most 3 (x-block, g-block, y-block), and F is exactly the words with <= 3 runs (proven Lemma F of draft_L3-CCL.md). [Verified: all valid gates — enumerated exhaustively by SAT with gate-blocking clauses — have exactly this form in: all 1203 extremal V-core instances for a <= 13; all 744 colourable V-core instances at EVERY N for a <= 9; and 174 out-of-sample extremal instances for a = 14..17 with Rad2 freshly computed by raw SAT. Zero violations.]

## Corollary CCL-V, full CCL, and gen = Rad2

**Statement.** (1) CCL-V: if N = Rad2(a;c) - 1 and W(N) is V-shaped with m >= 4, then some valid colouring of [1,N] has a gate with <= 3 runs — indeed every valid colouring does. (2) CCL in full: for every non-degenerate middle-band pair and every N, if the fixed-gate 2-SAT Psi(G,N) is satisfiable for some gate G, it is satisfiable for some G* in F. (3) gen(a,c) = Rad2(a;c) for every middle-band pair; combined with the project's proven coverage ({a|c}, {c<=0}, {0<c<a}, {c >= a(K+2)}, deep tail), the 2-colour Rado number Rad2(a; c) of x1 + a*x2 - x3 = c is now determined (proven-exact generator) on the entire parameter space where it exists.

**Proof.** (1) [1, Rad2-1] admits a valid colouring by the definition of Rad2; by Theorem V-RIGID its gate lies in F, and by the soundness Lemma S (draft_L3-CCL.md) that colouring is a model of Psi(gate, N). (2) By the proven trichotomy (Lemma V0 of draft_assalt_G2-pin.md): N <= lam-1 is the single-arm regime, where Theorem SA (proven, ibid.) provides explicit <= 2-run gates or vacuity; N >= lam with m <= 3 makes every gate a word of length <= 3, automatically in F; N >= lam with m >= 4 is the V-core, where any satisfiable gate yields a valid colouring whose own gate is in F by Theorem V-RIGID. (3) Theorem E (proven, ibid.) reduces gen = Rad2 to CCL at the single extremal length, which is (1); the coverage synthesis is Theorem DT-final and Coverage of draft_assalt_CCL-transfer.md. Note the proof of (2) delivers strictly more than CCL-V required: rigidity holds at every colourable N, not merely the extremal one.

## Gaps
- No gaps remain in CCL-V itself: the chain 2AD -> PAIR -> VAL, 2AD -> WINDOW -> INTERIOR -> V-RIGID -> CCL-V is complete and self-contained given the previously proven project results listed next.
- Declared dependencies (all previously proven and verified in the repository, none conjectural): Lemma R/S/F/W of demostracio/draft_L3-CCL.md (owner-window reformulation, fixed-gate 2-SAT soundness, F = <=3-run words, window consecutiveness); Lemma V0, Theorem SA, Lemma M, Theorem E of demostracio/draft_assalt_G2-pin.md (trichotomy, single-arm CCL, monotonicity, extremal reduction); Lemma V2 of demostracio/draft_assalt_CCL-transfer.md (v2(a)<=v2(c) iff v2(a)<=v2(t)). The final corollary (3) additionally inherits the dependencies already declared in draft_assalt_CCL-transfer.md for the non-V regimes (DT2020 Theorems 1 and 6, nota/sec_ub1-2, sec_lb, sec_conj).
- Minor scope notes, not gaps: (a) the PAIR threshold N >= P+Q+max(P,Q) is sufficient but not always tight (94 of the small (P,Q) systems are still feasible one below it — irrelevant since the V-core always provides N >= a+max(t,mu)+1); (b) V-RIGID says nothing about GLOBAL run counts of valid colourings, which remain genuinely quasiperiodic (the documented (7,16,15) obstruction is consistent: only the gate is rigid); (c) a >= 3 is automatic (a = 2 forces t = 1 and v2(a) > v2(t), so Rad2 does not exist); (d) empirically the valley-split system is feasible for some seeds when N <= a+max(t,mu)-1, so the m >= 4 hypothesis (which supplies the range) cannot simply be dropped from VAL.
