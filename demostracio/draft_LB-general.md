## Summary

THEOREM (lower bound, proven in full). Let d >= 2 and let a >= d+1 satisfy v2(a) <= v2(a-d). Write a = 2dq + r with r in [1, 2d] (Lemma 0 shows r != 2d under the hypothesis, so r in [1, 2d-1]), and set N* = (a+3)d + a - min(r-1, d-1) - 1. Then the extremal family coloring chi of [1, N*] contains no monochromatic solution (x, k, x + a(k-1) + d) of x1 + a*x2 - x3 = a - d; hence Rad2(a; a-d) >= N* + 1 = (a+3)d + a - min(r-1, d-1).

METHOD. (1) The family coloring is rewritten in a "chunk-offset normal form": with t = max(d, 2d-r), jump points p_j = ja + t (j = 1..d), J(x) = #{j : x > p_j} and offset w(x) = x - a*J(x), one has chi(x) = 1 iff (w(x)-1) mod 2d < d (Lemma 2; the d "defects" of the construction -- long (d+r)-runs in regime r <= d, short s-runs in regime r > d -- are exactly the d phase jumps, each shifting the phase by -r == -a (mod 2d)). (2) For any in-range triple, w(y) - w(x) = n*a + d where n = k - 1 - (J(y) - J(x)), and the w-ranges of the chunks force the trichotomy n in {-1, 0, 1} (Lemma 4); in particular k <= d+2. (3) Case n = 0: phases of x and y differ by exactly d, so chi(x) != chi(y) -- never monochromatic. Case n = -1: forces k = DeltaJ <= d, hence chi(k) = 1, while the offset window pins chi(x) = 0 (regime 1) or chi(y) = 0 (regime 2). Case n = +1: forces x <= t and either (sub-case away from the last chunk) chi(x) = 1 and chi(y) = 0 directly, or (sub-case in the last chunk) k = d + 2, whence chi(k) = 0 but chi(x) = 1 (regime 2), or chi(x) = 0 forces chi(y) = 1 (regime 1). Every case is contradictory, uniformly in q >= 1 (regime 1) / q >= 0 (regime 2) and in all edge values r = 1, r = d, r = d+1, r = 2d-1, a = d+1. The single use of d >= 2 is chi(d+2) = 0 (d+2 <= 2d); for d = 1 that step fails and indeed (1, 3, 2a+2) is monochromatic in the analogous construction, matching Theorem 1's value 2a+2 sitting one below the general formula. All lemmas and every intermediate inequality were verified computationally on thousands of (a, d) pairs before being written up (see computational_checks).

## Lemma 0 (normalization)

**Statement.** Let d >= 2 and a >= d+1 (equivalently 0 < c < a for c = a - d). Write a = 2dq + r with q >= 0 and r in [1, 2d] (this representation is unique). Then: (i) if r = 2d then v2(a) > v2(a-d), so the existence condition v2(a) <= v2(a-d) fails; hence under that condition r in [1, 2d-1]. (ii) If r <= d ("regime 1") then q >= 1. (iii) If r >= d+1 ("regime 2"), set s = r - d; then s in [1, d-1] and q >= 0. Throughout, define t = max(d, 2d - r), so t = 2d - r in regime 1 (t in [d, 2d-1]) and t = d in regime 2; and define N* = (a+3)d + a - min(r-1, d-1) - 1, so N* = (a+3)d + a - r in regime 1 and N* = (a+2)d + a in regime 2. Finally, the solutions of x1 + a*x2 - x3 = a - d with values in [1, N] are exactly the triples (x, k, y) with y = x + a(k-1) + d and x, k, y in [1, N].

**Proof.** (i) If r = 2d then 2d | a; write a = 2dm with m >= 1. Then v2(a) = v2(2dm) >= v2(2d) = v2(d) + 1. Also a - d = d(2m - 1) with 2m - 1 odd, so v2(a - d) = v2(d). Hence v2(a) >= v2(d) + 1 > v2(d) = v2(a-d). (ii) If r <= d and q = 0 then a = r <= d, contradicting a >= d + 1; so q >= 1. In particular a = 2dq + r >= 2d + 1 in regime 1. (iii) r in [d+1, 2d-1] gives s = r - d in [1, d-1]; nothing forces q >= 1 here (e.g. a = d + s with q = 0 occurs whenever d + 1 <= a <= 2d - 1). For the formulas for N*: in regime 1, min(r-1, d-1) = r - 1 (as r <= d), so N* = (a+3)d + a - (r-1) - 1 = (a+3)d + a - r; in regime 2, min(r-1, d-1) = d - 1 (as r >= d+1), so N* = (a+3)d + a - (d-1) - 1 = (a+2)d + a. For the solution set: x3 = x1 + a*x2 - (a - d) = x1 + a(x2 - 1) + d, so with x = x1, k = x2, y = x3 the solutions are precisely the stated triples; conversely every such triple is a solution. QED.

## Lemma 1 (the family coloring: well-definedness, run colors, total length)

**Statement.** Keep the notation of Lemma 0 and assume r in [1, 2d-1]. Define the family coloring chi as the concatenation of runs of constant color, colors alternating with the first run of color 1 (so the run with index j gets color 1 if j is odd, color 0 if j is even), with run-length list: REGIME 1 (r <= d): [d] x (2q+1); then for i = 1, ..., d: one run [d+r] followed by [d] x (2q-1) if i < d, and by [d] x (2q+1) if i = d. REGIME 2 (r > d, s = r - d): [d] x (2q+2), [s]; then for i = 1, ..., d-1: [d] x (2q+1), [s]; finally [d] x (2q+2), [s]. Then: (a) all run lengths are >= 1; (b) the total length is exactly N*, so chi : [1, N*] -> {0,1}; (c) in regime 1 the i-th long (d+r)-run has run index 2qi + 2 (even), hence color 0; in regime 2 all s-runs except the last have odd index, hence color 1, and the final s-run has even index, hence color 0.

**Proof.** (a) Regime 1: d >= 2, d + r >= 3, and 2q - 1 >= 1 because q >= 1 (Lemma 0(ii)); the multiplicities 2q+1, 2q-1 are >= 1. Regime 2: d >= 2, s in [1, d-1] (Lemma 0(iii)), and 2q+1 >= 1, 2q+2 >= 2 for q >= 0. (b) Regime 1: using 2dq = a - r, the total is 2*d(2q+1) + d(d+r) + (d-1)*d(2q-1) = (4dq + 2d) + (d^2 + dr) + (2qd^2 - d^2 - 2qd + d) = 2qd^2 + dr + 3d + 2dq. On the other hand N* = (a+3)d + a - r = (2dq + r + 3)d + 2dq = 2qd^2 + rd + 3d + 2dq. Equal. Regime 2: using 2dq = a - d - s, we have d(2q+2) = a + d - s and d(2q+1) = a - s, so the total is 2(a + d - s) + (d-1)(a - s) + (d+1)s = 2a + 2d - 2s + (d-1)a - (d-1)s + (d+1)s = (d+1)a + 2d + s(-2 - (d-1) + (d+1)) = (d+1)a + 2d = ad + a + 2d = (a+2)d + a = N*. Equal. (c) Regime 1: the number of runs strictly before the i-th long run is (2q+1) + (i-1)(1 + (2q-1)) = (2q+1) + (i-1)2q, so its index is 2qi + 2, which is even; hence color 0. Regime 2: the initial group contributes 2q+3 runs, so the first s-run has index 2q+3 (odd: color 1); each subsequent group ([d] x (2q+1), [s]) contributes 2q+2 runs (even), so the s-run closing the i-th middle group (1 <= i <= d-1) has index (2q+3) + i(2q+2), odd + even = odd: color 1. The final group's first d-run has index (2q+3) + (d-1)(2q+2) + 1, which is even (odd + even + 1); its last d-run has index (2q+3) + d(2q+2), odd; the final s-run has index (2q+3) + d(2q+2) + 1, even: color 0. QED.

## Lemma 2 (chunk-offset normal form)

**Statement.** Keep the notation of Lemmas 0-1. Define the jump points p_j = ja + t for j = 1, ..., d, the jump count J(x) = #{ j in [1,d] : x > p_j }, the offset w(x) = x - a*J(x), and alpha(u) = 1 if u mod 2d in [0, d-1], alpha(u) = 0 if u mod 2d in [d, 2d-1]. Then: (I) chi(x) = alpha(w(x) - 1) for every x in [1, N*]. (II) (w-ranges) J(x) = 0 iff x in [1, a+t], and then w(x) = x in [1, a+t]; for j in [1, d-1], J(x) = j iff x in [ja+t+1, (j+1)a+t], and then w(x) in [t+1, a+t]; J(x) = d iff x in [da+t+1, N*], and then w(x) in [t+1, a+t+d]. In particular w(x) <= a+t+d always, and w(x) >= t+1 whenever J(x) >= 1.

**Proof.** We use two trivial facts. FACT A: alpha(u + d) = 1 - alpha(u) for every integer u (write u = 2dm + v, v in [0, 2d): if v < d then (u+d) mod 2d = v + d in [d, 2d); if v >= d then (u+d) mod 2d = v - d in [0, d)). FACT B: if u == 0 (mod 2d) then alpha(u) = alpha(u+1) = ... = alpha(u+d-1) = 1, and if u == d (mod 2d) then those d values all have alpha = 0. Throughout we use a == r (mod 2d).

(II) first. Since 0 < p_1 < p_2 < ... < p_d and, in both regimes, N* - p_d = N* - da - t = a + d > 0 (regime 1: N* - da - (2d - r) = (ad + a + 3d - r) - ad - 2d + r = a + d; regime 2: N* - da - d = (ad + a + 2d) - ad - d = a + d), the sets {x : J(x) = j} are exactly the stated intervals, and w(x) = x - ja on the j-th of them, giving the stated w-ranges ([ja+t+1, (j+1)a+t] - ja = [t+1, a+t]; [da+t+1, N*] - da = [t+1, a+t+d]).

(I), REGIME 1 (t = 2d - r, q >= 1). By Lemma 1, [1, N*] decomposes into consecutive segments S0 = [1, a+d-r] (the initial 2q+1 d-runs; length d(2q+1) = 2dq + d = a + d - r); for i = 1..d the long run L_i = [ia+d-r+1, ia+2d] (length d+r; indeed the segment before L_1 ends at a+d-r and each later L_i starts one past the end of T_{i-1} below), and the tail T_i = [ia+2d+1, (i+1)a+d-r] for i <= d-1 (length a - d - r = d(2q-1)) and T_d = [da+2d+1, N*] (length N* - da - 2d = a + d - r = d(2q+1)). Consecutiveness: end(S0)+1 = a+d-r+1 = start(L_1); end(L_i)+1 = ia+2d+1 = start(T_i); end(T_i)+1 = (i+1)a+d-r+1 = start(L_{i+1}); end(T_d) = N*. So the segments partition [1, N*].

J-values per segment: p_j = ja + 2d - r. On S0: x <= a+d-r < a+2d-r = p_1, so J = 0. Split L_i at p_i (note ia+d-r+1 <= p_i <= ia+2d-1 since d >= 1 and r >= 1): on L_i^- = [ia+d-r+1, p_i] we have p_{i-1} = (i-1)a + 2d - r < ia + d - r + 1 iff 2d < a + d + 1 iff d <= a, true; so J = i - 1. On L_i^+ = [p_i + 1, ia + 2d]: J = i (for i < d, ia+2d <= p_{i+1} = ia + a + 2d - r iff r <= a, true). On T_i: p_i < ia+2d+1 and (for i < d) (i+1)a + d - r < p_{i+1}, so J = i; on T_d, J = d.

Phases: write u(x) = w(x) - 1. On S0: u = x - 1 runs over [0, d(2q+1) - 1]; S0 consists of the d-runs R_m = [(m-1)d+1, md], m = 1..2q+1, colored 1 for m odd, 0 for m even (Lemma 1, global indices 1..2q+1). For x in R_m, u = (m-1)d + v with v in [0, d-1]; if m is odd, (m-1)d == 0 (mod 2d) and Fact B gives alpha = 1; if m is even, (m-1)d == d and alpha = 0. Match. On L_i^-: u = x - 1 - (i-1)a in [a+d-r, a+2d-r-1], d consecutive integers whose first is == r + d - r = d (mod 2d); by Fact B, alpha = 0 on all of L_i^-. On L_i^+: u = x - 1 - ia in [2d-r, 2d-1], a subset of [d, 2d-1] because r <= d; so alpha = 0. Hence alpha = 0 on the whole long run, matching its color 0 (Lemma 1(c)). On T_i: u = x - 1 - ia in [2d, 2d + M - 1] with M = d(2q-1) (i < d) or d(2q+1) (i = d); u - 2d starts at 0, and T_i consists of consecutive d-runs whose first has global index 2qi + 3 (odd, color 1: it is the run following the i-th long run, whose index is 2qi+2 by Lemma 1(c)); exactly as for S0, alpha matches the alternating colors on all of T_i (blocks aligned: u == 0 (mod 2d) at the start).

(I), REGIME 2 (t = d, s = r - d in [1, d-1], q >= 0). Segments: C0b = [1, a+d-s] (2q+2 d-runs; length d(2q+2) = (a - d - s) + 2d = a+d-s), C0s = [a+d-s+1, a+d] (first s-run); for i = 1..d-1: Cib = [ia+d+1, (i+1)a+d-s] (2q+1 d-runs; length a - s = d(2q+1)), Cis = [(i+1)a+d-s+1, (i+1)a+d] (s-run); Fb = [da+d+1, da+a+2d-s] (2q+2 d-runs; length a+d-s = d(2q+2)), Fs = [da+a+2d-s+1, N*] (final s-run; N* = da + a + 2d, length s). These are consecutive and partition [1, N*].

J-values: p_j = ja + d. On C0b u C0s = [1, a+d] = [1, p_1]: J = 0. On Cib u Cis = [ia+d+1, (i+1)a+d] = [p_i + 1, p_{i+1}]: J = i. On Fb u Fs = [p_d + 1, N*]: J = d.

Phases (u = w - 1; a == r = d + s mod 2d): On C0b: u = x - 1 in [0, d(2q+2) - 1] with d(2q+2) = 2d(q+1); aligned d-blocks starting at phase 0: alpha alternates 1,0,1,... per run, matching global run indices 1..2q+2 (odd = 1). On C0s: u = x - 1 in [a+d-s, a+d-1]; a + d - s = 2d(q+1) == 0 (mod 2d), so u mod 2d in [0, s-1], a subset of [0, d-1] (s <= d-1): alpha = 1, matching color 1 (Lemma 1(c), index 2q+3 odd). On Cib: u = x - 1 - ia in [d, d + d(2q+1) - 1]; aligned blocks starting at phase d: alpha alternates 0,1,0,..., matching colors, since the first run of Cib has even global index (2q+3) + (i-1)(2q+2) + 1 (odd + even + 1), i.e. color 0. On Cis: u = x - 1 - ia in [d + a - s, d + a - 1]; d + a - s == d + (d+s) - s = 2d == 0 (mod 2d), so u mod 2d in [0, s-1]: alpha = 1, matching color 1 (index (2q+3) + i(2q+2), odd). On Fb: u = x - 1 - da in [d, d + 2d(q+1) - 1]; aligned blocks starting at phase d: alpha alternates 0,1,...,1 (2q+2 runs, ending with alpha = 1), matching colors (first run even index (2q+3)+(d-1)(2q+2)+1, color 0; last run odd index, color 1). On Fs: u = x - 1 - da in [d + 2d(q+1), d + 2d(q+1) + s - 1], so u mod 2d in [d, d+s-1], a subset of [d, 2d-1] (s <= d-1): alpha = 0, matching the final s-run's color 0 (index (2q+3) + d(2q+2) + 1, even). QED.

## Lemma 3 (colors of the small integers 1..d+2)

**Statement.** With chi as above (d >= 2, r in [1, 2d-1]): chi(k) = 1 for 1 <= k <= d, and chi(k) = 0 for k in {d+1, d+2}.

**Proof.** Since a >= d+1 and t >= d, we have a + t >= 2d + 1 > d + 2 (using d >= 2 for 2d + 1 >= d + 3, and in fact 2d+1 > d+2 iff d > 1). Hence every k <= d + 2 satisfies k <= a + t = p_1, so J(k) = 0 and w(k) = k (Lemma 2(II)); also d + 2 < a + t <= N*, so chi(k) is defined. By Lemma 2(I), chi(k) = alpha(k - 1). For 1 <= k <= d: k - 1 in [0, d-1], so alpha = 1. For k in {d+1, d+2}: k - 1 in {d, d+1}, and d + 1 <= 2d - 1 because d >= 2, so k - 1 in [d, 2d-1] and alpha = 0. QED. (This is the only place the hypothesis d >= 2 is used, and it is sharp: for d = 1, chi(d+2) = chi(3) = 1, and the proof of the Theorem breaks precisely in Case n = +1, sub-case J(y) = d -- see the Remark.)

## Lemma 4 (jump-count trichotomy)

**Statement.** Let x, y in [1, N*] with y = x + a(k-1) + d for some integer k >= 1. Put DeltaJ = J(y) - J(x) and n = k - 1 - DeltaJ. Then w(y) = w(x) + n*a + d, DeltaJ >= 0, and n in {-1, 0, +1}. Consequently k = n + 1 + DeltaJ <= d + 2 for every in-range triple.

**Proof.** Identity: w(y) - w(x) = (y - a*J(y)) - (x - a*J(x)) = (y - x) - a*DeltaJ = a(k-1) + d - a*DeltaJ = n*a + d. DeltaJ >= 0 because y > x (as a(k-1) + d > 0) and J is non-decreasing.

Suppose n >= 2. Then w(y) = w(x) + n*a + d >= 1 + 2a + d (w(x) >= 1 by Lemma 2(II)). But w(y) <= a + t + d always (Lemma 2(II)), so 2a + d + 1 <= a + t + d, i.e. a + 1 <= t. In regime 1, a >= 2d + 1 (Lemma 0(ii)) while t = 2d - r <= 2d - 1, so a + 1 >= 2d + 2 > t: contradiction. In regime 2, t = d < d + 2 <= a + 1: contradiction.

Suppose n <= -2. Then DeltaJ = k - 1 - n >= k + 1 >= 2. Hence J(y) = J(x) + DeltaJ >= 2 >= 1, so w(y) >= t + 1 (Lemma 2(II)); and J(x) = J(y) - DeltaJ <= d - 2 <= d - 1, so w(x) <= a + t (Lemma 2(II)). Then t + 1 <= w(y) = w(x) + n*a + d <= (a + t) - 2a + d = t - (a - d) <= t - 1, since a >= d + 1: contradiction.

So n in {-1, 0, 1}. Finally DeltaJ <= J(y) <= d gives k = n + 1 + DeltaJ <= 1 + 1 + d = d + 2. QED.

## THEOREM (lower bound of the conjecture, all d >= 2)

**Statement.** Let d >= 2 and a >= d+1 with v2(a) <= v2(a - d), and let r, t, N*, chi be as in Lemmas 0-2. Then chi is a 2-coloring of [1, N*] containing NO monochromatic triple (x, k, y) with y = x + a(k-1) + d and x, k, y in [1, N*]. Consequently Rad2(a; a - d) >= N* + 1 = (a+3)d + a - min(r-1, d-1), i.e. the conjectured value is a lower bound for Rad2(a; a-d) for every such pair (a, d).

**Proof.** By Lemma 0(i), r in [1, 2d-1], so Lemmas 1-4 apply. Suppose, for contradiction, that x, k, y = x + a(k-1) + d all lie in [1, N*] and chi(x) = chi(k) = chi(y). Let DeltaJ = J(y) - J(x) and n = k - 1 - DeltaJ; by Lemma 4, n in {-1, 0, 1} and w(y) = w(x) + n*a + d. Write alpha, w, J as in Lemma 2 and recall chi(z) = alpha(w(z) - 1) for all z in [1, N*].

CASE n = 0. Then w(y) = w(x) + d, so chi(y) = alpha(w(x) - 1 + d) = 1 - alpha(w(x) - 1) = 1 - chi(x) by Fact A (Lemma 2). Hence chi(x) != chi(y): contradiction.

CASE n = -1, i.e. DeltaJ = k. Then k = DeltaJ <= J(y) <= d, so chi(k) = 1 by Lemma 3; the common color is 1. Also J(y) = J(x) + k >= 1, so w(y) >= t + 1; and J(x) = J(y) - k <= d - 1, so w(x) <= a + t (Lemma 2(II)). From w(y) = w(x) - (a - d): w(x) >= w(y) + a - d >= a + t + 1 - d and w(y) <= (a + t) - (a - d) = t + d. Now:
- Regime 1 (t = 2d - r): w(x) - 1 lies in [a + t - d, a + t - 1], d consecutive integers whose first satisfies a + t - d = a + d - r == r + d - r = d (mod 2d); by Fact B (Lemma 2), alpha(w(x) - 1) = 0, i.e. chi(x) = 0 != 1: contradiction.
- Regime 2 (t = d): w(y) - 1 lies in [t, t + d - 1] = [d, 2d - 1], already inside [0, 2d), so alpha(w(y) - 1) = 0, i.e. chi(y) = 0 != 1: contradiction.

CASE n = +1, i.e. DeltaJ = k - 2. Since DeltaJ >= 0, k >= 2. Here w(y) = w(x) + a + d. Since w(y) <= a + t + d always (Lemma 2(II)), w(x) <= t; but every z with J(z) >= 1 has w(z) >= t + 1, so J(x) = 0 and x = w(x) in [1, t]. Hence DeltaJ = J(y) = k - 2.
- Sub-case J(y) <= d - 1: then w(y) <= a + t (Lemma 2(II)), so x = w(y) - a - d <= t - d.
  * Regime 2: t = d gives x <= 0, impossible.
  * Regime 1: x in [1, t - d] = [1, d - r] (nonempty only if r <= d - 1). Then chi(x) = alpha(x - 1) with x - 1 in [0, d - r - 1], a subset of [0, d - 1]: chi(x) = 1. And w(y) - 1 = x + a + d - 1 == (x - 1) + r + d (mod 2d), with (x - 1) + r + d in [r + d, 2d - 1], a subset of [d, 2d - 1]: chi(y) = 0. So chi(x) != chi(y): contradiction.
- Sub-case J(y) = d: then k - 2 = d, so k = d + 2 and chi(k) = 0 by Lemma 3 (here d + 2 <= 2d, i.e. d >= 2, is used); the common color is 0, so chi(x) = 0 is forced.
  * Regime 2: x in [1, t] = [1, d], so chi(x) = alpha(x - 1) with x - 1 in [0, d - 1]: chi(x) = 1 != 0: contradiction.
  * Regime 1: x in [1, t] = [1, 2d - r], so x - 1 in [0, 2d - r - 1], all inside [0, 2d); chi(x) = 0 requires (x - 1) in [d, 2d - r - 1]. If r = d this interval is empty, so chi(x) = 1 != 0: contradiction. If r <= d - 1, then x in [d + 1, 2d - r], and w(y) - 1 = x + a + d - 1 == (x - 1) + r + d (mod 2d), with (x - 1) + r + d in [2d + r, 3d - 1]; subtracting 2d, the residues lie in [r, d - 1], a subset of [0, d - 1]: chi(y) = 1 != 0: contradiction.

All cases are contradictory, so chi has no monochromatic triple. Since chi 2-colors [1, N*] with no monochromatic solution of x1 + a*x2 - x3 = a - d (Lemma 0, solution parametrization), Rad2(a; a - d) > N*, i.e. Rad2(a; a - d) >= N* + 1 = (a+3)d + a - min(r-1, d-1). QED.

## Remark (sharpness of d >= 2; consistency with Theorem 1)

**Statement.** For d = 1 (where r = 1, regime 1, t = 1) the same construction and the whole proof go through EXCEPT the single step chi(d + 2) = 0 in Lemma 3 / Case n = +1, sub-case J(y) = d, which requires d + 2 <= 2d. For d = 1 one has chi(3) = 1 and the triple (x, k, y) = (1, 3, 2a + 2) is monochromatic of color 1 in the length-(2a+2) construction. This is the unique failure mode, and it is consistent with Theorem 1: Rad2(a; a-1) = 2a + 2 = ((a+3)*1 + a - 0) - 1 + 1, exactly one below what the general formula would predict at d = 1.
