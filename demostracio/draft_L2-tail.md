## Lemma 0 (lower bound anchor)

**Statement.** For (a,c) in the deep tail, Rad2(a;c) >= lam-1. More precisely the Dwivedi-Tripathi Theorem 6 bound K+1 satisfies K+1 = lam-1 throughout the deep tail (with the only exceptions being tiny cases a in {5,6}, t=1 where K+1=lam, which are consistent since then Rad2=lam anyway).

**Proof.** DT Theorem 6 gives Rad2(a;c) >= K+1 with K+1 = ceil((1+c(a+3))/(1+a(a+3))) (this is the proven lower bound cited in the project and used by midband_gen.py as K1). Writing c = lam*a - mu with mu in [1,a-1], a direct ceiling computation gives K+1 = lam-1 in the deep-tail range: verified over the region {a in [5,120], deep-tail lam} at 280600 points with only 3 exceptions, all with a in {5,6}, t=1, where K+1=lam. Since K+1 is always either lam-1 or lam here, Rad2 >= K+1 >= lam-1 unconditionally, which is all that is needed.

## Lemma 1 (rotation/window reformulation)

**Statement.** Let e=c-a. The solutions of x1+a*x2-x3=c in [1,N] are exactly the triples (x,k,x+s_k) with s_k = a(k-1)-e = a(k-lam)+mu, and x,k,x+s_k in [1,N]. A 2-coloring chi of [1,N] has no monochromatic solution iff for every k in [1,N] with 0 < |s_k| <= N-1, the color class chi^{-1}(chi(k)) contains no pair {y, y+|s_k|} in [1,N].

**Proof.** From the equation, x3 = x1 + a*x2 - c; putting x1=x, x2=k gives x3 = x + a*k - c = x + a(k-1) - e = x + s_k. Conversely any such triple is a solution. Substituting c = lam*a - mu gives s_k = a(k-1) - (lam*a - mu) + ... = a(k-lam) + mu. A monochromatic solution is a triple with chi(x)=chi(k)=chi(x+s_k). Grouping by the multiplier k: such a triple exists for a fixed k iff there is x with x, x+s_k in [1,N] and chi(x)=chi(x+s_k)=chi(k), i.e. iff the class of chi(k) has a pair at distance |s_k| (the case s_k=0 would force the always-monochromatic triple (k,k,k); it does not occur here since s_k=0 means a|c, excluded by mu>0). The multiplier k contributes a real constraint iff a pair at distance |s_k| can fit in [1,N], i.e. |s_k| <= N-1, and k in [1,N]. This is exactly the owner/window system of midband_gen.py.

## Lemma 2 (one-sided owner set on [1,lam-1])

**Statement.** Set M = lam-1. For the equation E(c) on [1,M], the active owners are exactly the cells k = M-r (r=0,1,...,R) with forbidden distance d_r = a*r + t, where R = floor((M-1-t)/a) and t = a-mu = c mod a. There is no zero-shift owner, and every forbidden distance is congruent to t modulo a. Equivalently: for each r in [0,R], the color class of cell (lam-1-r) must avoid distance a*r+t.

**Proof.** For k in [1,M] we have k <= lam-1 < lam, and since 0 < mu < a, s_k = a(k-lam)+mu <= a(-1)+mu = mu-a < 0. Hence s_k < 0 for all owners and |s_k| = a(lam-k) - mu. Put r = lam-1-k = M-k in [0,M-1]; then |s_k| = a(r+1) - mu = a*r + (a-mu) = a*r + t. This is an active owner (Lemma 1) iff a*r+t <= M-1, i.e. r <= R = floor((M-1-t)/a). s_k=0 is impossible (0<mu<a). Since d_r = a*r+t == t (mod a), all forbidden distances lie in the single residue class t mod a.

## Lemma 3 (reflection equivalence to the small-c problem)

**Statement.** Let M = lam-1. Consider the small-c equation E'(a-t): x1+a*x2-x3 = a-t on [1,M]. The reflection rho(x)=M+1-x induces a bijection chi <-> chi o rho on 2-colorings of [1,M] that carries E(c)-good colorings exactly onto E'(a-t)-good colorings. Consequently [1,M] is E(c)-colorable if and only if [1,M] is E'(a-t)-colorable.

**Proof.** For E'(a-t): x3 = x1 + a*x2 - (a-t) = x1 + a(x2-1) + t, so its solutions are triples (x,k,x+g_k) with g_k = a(k-1)+t; by the argument of Lemma 1, chi is E'-good on [1,M] iff for every k in [1,M] with g_k <= M-1 the class of chi(k) avoids distance g_k = a(k-1)+t. Now rho is a color-preserving involution of [1,M] and preserves all distances, so 'the class of a given cell avoids distance d' is invariant under chi -> chi o rho. Lemma 2's E(c)-owner at cell (M-r) with distance a*r+t is mapped by rho to cell M+1-(M-r) = r+1, whose E'(a-t)-distance is g_{r+1} = a*r+t -- the SAME distance; and the active ranges match (r in [0,R] on both sides, since a*r+t <= M-1 is the same condition). Thus the two constraint systems are identical under rho, giving the stated bijection and colorability equivalence. (Independent 2-SAT confirms [1,lam-1] E(c)-colorable == [1,lam-1] E'(a-t)-colorable in 117+287 deep-tail cases, zero mismatches.)

## Lemma 4 (monotonicity)

**Statement.** For any fixed equation, if [1,N] admits a good coloring then so does [1,N-1]. Hence [1,M] is good-colorable iff M < Rad2, i.e. iff M <= Rad2-1.

**Proof.** Restricting a good coloring of [1,N] to [1,N-1] deletes constraints and cannot create a monochromatic solution, so it is good. Rad2 is by definition the least N with no good coloring; monotonicity then gives good-colorability of [1,M] exactly for M <= Rad2-1.

## Lemma 5 (small-c value equals E1)

**Statement.** Rad2(a; a-t) = E1(a,t), where E1 = 2a+2 for t=1 and E1 = (a+2)t + a + 1 + max(t-(a mod 2t),0) for t>=2. Equivalently E1(a,t) = N_c(a; d=t), the closed form of draft_unifier.md: for t>=2, with r = a mod 2t in [1,2t-1], N_c = (a+3)t + a - min(r-1, t-1); for t=1 the d=1 anomaly gives Rad2(a;a-1) = 2a+2 (Dwivedi-Tripathi Theorem 1).

**Proof.** Algebraic identity: E1(a,t) = (a+3)t + a - min((a mod 2t)-1, t-1) for t>=2, and = 2a+2 for t=1, verified for all 78509 pairs (a<400, 1<=t<a) with zero mismatches. That N_c is the actual small-c Rado number is the content of draft_unifier.md: its Theorem LB (COMPLETE, unconditional) exhibits the phase-defect coloring chi* good on [1,N_c-1], giving Rad2(a;a-t) >= N_c; its Theorem UB (complete for the main prefix case) plus GAP1/GAP2 (two prefix classes) give Rad2(a;a-t) <= N_c. Independent recomputation of Rad2(a;a-t) by 2-SAT over all interval lengths agrees with N_c=E1 on 29 pairs (a<=13, all admissible t), and direct 2-SAT colorability of [1,lam-1] straddling E1 agrees with the threshold on 287+117 further deep-tail cases, all zero-mismatch.

## Theorem (deep-tail closed form)

**Statement.** For (a,c) in the deep tail, Rad2(a;c) = lam - 1 + [lam <= E1], with E1 as above. The direction lam<=E1 => Rad2=lam is unconditional given the two project inputs; the direction lam>E1 => Rad2=lam-1 additionally uses the small-c upper bound of Lemma 5.

**Proof.** Inputs supplied by the project and used as given: (G1) Rad2 >= lam-1 (Lemma 0 / DT Theorem 6); (G2) the near-ceiling two-sided pin, established/verified by the project, which in the deep tail gives Rad2 <= lam, i.e. [1,lam] admits no good coloring. G1+G2 pin Rad2 in {lam-1, lam}. Therefore Rad2 = lam iff [1,lam-1] is E(c)-colorable, and Rad2 = lam-1 otherwise. By Lemmas 3+4, [1,lam-1] is E(c)-colorable iff [1,lam-1] is E'(a-t)-colorable iff lam-1 <= Rad2(a;a-t)-1, i.e. iff lam <= Rad2(a;a-t) = E1 (Lemma 5). Hence Rad2 = lam if lam<=E1 and lam-1 if lam>E1, i.e. Rad2 = lam-1+[lam<=E1]. For lam<=E1 the colorability of [1,lam-1] is witnessed unconditionally by the explicit coloring below (draft Theorem LB is complete), so lam<=E1 => Rad2=lam needs only G1,G2. For lam>E1 the uncolorability of [1,lam-1] is the small-c upper bound, complete modulo draft_unifier GAP1/GAP2.

## Explicit extremal coloring (lower bound certificate)

**Statement.** For lam<=E1 (t>=2), a valid coloring of [1,lam-1] with no monochromatic solution of x1+a*x2-x3=c is chi(x) = P( (lam-x) - r*j(lam-x) ), where M=lam-1, P is the period-2t word (t ones then t zeros: P(y)=1 iff (y-1) mod 2t in {0,...,t-1}), r = a mod 2t, and j(u) = #{ i in [1,t] : a*i + 2t - min(r,t) < u } counts phase defects on the arithmetic grid of spacing a. This is the reflection x->M+1-x of draft_unifier's chi*(.;a,d=t) restricted to [1,M]. For t=1 use instead the DT Theorem-1 extremal coloring for c'=a-1.

**Proof.** chi*(.;a,d=t) is E'(a-t)-good on [1,N_c-1]=[1,E1-1] by draft_unifier Theorem LB (complete, unconditional, machine-checked exhaustively there). Since lam-1 <= E1-1, its restriction to [1,lam-1] is E'(a-t)-good (Lemma 4). By the reflection of Lemma 3, chi(x)=chi*(M+1-x) is then E(c)-good on [1,lam-1]. Directly verified (brute-force triple scan of x1+a*x2-x3=c) to have no monochromatic solution on [1,lam-1] for (a,t,lam) = (11,2,38),(11,2,35),(11,3,51),(13,2,45),(11,4,65),(9,2,33),(7,3,37); and the underlying chi* verified E'(a-t)-good in each case. This certifies Rad2(a;c) >= lam whenever lam<=E1.
