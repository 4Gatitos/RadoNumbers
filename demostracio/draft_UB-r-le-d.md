# Draft: UB-r-le-d

**status**: complete

## Summary

THEOREM (upper bound, regime r <= d). Let d >= 2, let a satisfy v2(a) <= v2(a-d), write a = 2dq + r with r in [1,2d], and assume 1 <= r <= d. Set N := (a+3)d + a - (r-1) = (d+1)a + 3d + 1 - r. Then EVERY 2-coloring of [1,N] contains a monochromatic solution (x1,x2,x3) = (x, k, x + a(k-1) + d) of x1 + a*x2 - x3 = a - d. Hence Rad2(a; a-d) <= (a+3)d + a - min(r-1,d-1) in this regime, matching the conjecture (the reverse inequality being witnessed by the already machine-verified extremal family at length N-1).

The proof is UNCONDITIONAL and complete — no gaps. Its architecture generalizes the Theorem-1 template but replaces the counting step with a rigidity ("transport") mechanism: (0) WLOG chi(1)=0; then chi(d+1)=1. (1) A four-triple Transport Lemma: for any k0 with chi(k0)=1 and D = (k0-1)a+d, a 1 at y with y <= N-D-d forces chi(y+d)=0, chi(y+2d)=1. (2) Pairing: chi(s+d) = 1-chi(s) for all s in [1,d]; this defines phi(s)=chi(s) and its 2d-periodic antisymmetric extension p (p(u+d)=1-p(u), p(1)=0). (3) Window Lemma: with s* = min{s in [2,d+1] : p(s)=1} and X* = N - ((s*-1)a+d), the coloring is FORCED to equal p on all of [1,X*]. (4) Far-Zone Lemma: chi(z) = 1 - p(z-E) on [E+1,N], where E = d(a+1). (5) Endgame, two cases. Case A (some phi(s)=1, s in [2,d]): the window then reaches beyond a+2d, so the d+2-... the k=d rule couples window to far zone along a FULL residue sweep mod 2d; a pigeonhole/translation-invariance argument in Z_{2d} — using gcd(r,2d) | d, which is EXACTLY equivalent to the existence condition v2(r) <= v2(d) — produces u in [1,2d] with p(u+r) = phi(d) and p(u) = 1-phi(d), making the triple (a+u, d, E+u) monochromatic. Case B (phi identically 0): the single explicit triple (d+1-r, 2, a+2d+1-r) is monochromatic of color 0, because a+2d+1-r = N-E is ≡ 1 (mod 2d) and lies in the window. Both cases contradict mono-freeness. QED.

Every lemma and every range inequality was machine-validated: SAT-entailment of all forced values for all phi-patterns on six (a,d) instances; all arithmetic side conditions on all 1492 admissible pairs with a <= 120; the Case-A combinatorial claim exhaustively for d <= 12 and randomly to d = 40; full end-proof instantiation on 40060 (a,d,phi) cases with zero failures; independent brute force (2^20 colorings) for (a,d)=(5,2); and DPLL UNSAT@N / SAT@N-1 for 22 cases up to d=6. The proof also explains the empirical structure: at length N-1 the extremal colorings realize phi = 0 and evade the Window Lemma by exactly one unit of room, which is why the bound is tight.

## Lemma 0 (Setup, normalization, basic inequalities)

**Statement.** Let d >= 2, 0 < c < a, d = a - c, a = 2dq + r with r in [1, 2d], and assume r <= d. Set N := (a+3)d + a - (r-1) and E := d(a+1). Then: (i) q >= 1 and a >= 2d+1; (ii) N = (d+1)a + 3d + 1 - r and N - E = 2d(q+1) + 1 >= 4d + 1; (iii) the solutions of x1 + a x2 - x3 = a - d are exactly the triples T(x,k) := (x, k, x + a(k-1) + d), and such a triple lies in [1,N]^3 iff x >= 1, k in [1,N], and x + a(k-1) + d <= N; (iv) the property of being a mono-solution-free ('valid') coloring is invariant under swapping the two colors, so in any proof by contradiction we may assume chi(1) = 0.

**Proof.** (i) If q = 0 then a = r <= d (regime assumption), while 0 < c means d = a - c <= a - 1, i.e. d < a — contradiction. Hence q >= 1 and a = 2dq + r >= 2d + 1. (ii) Since r <= d, min(r-1, d-1) = r-1, so N = ad + 3d + a - r + 1 = (d+1)a + 3d + 1 - r. Then N - E = (d+1)a + 3d + 1 - r - d(a+1) = a + 2d + 1 - r = 2dq + r + 2d + 1 - r = 2d(q+1) + 1, and q >= 1 gives N - E >= 4d + 1. (iii) x1 + a x2 - x3 = a - d is equivalent to x3 = x1 + a(x2 - 1) + d; since x3 > x1 >= 1 automatically, the only binding conditions are x1 >= 1, x2 in [1,N], x3 <= N. (iv) The defining condition 'chi(x1) = chi(x2) = chi(x3) for some solution triple' is preserved by the swap 0 <-> 1; hence chi is valid iff its swap is valid, and we may normalize chi(1) = 0. QED.

## Lemma 1 (2-adic lemma: the existence condition forces gcd(r,2d) | d)

**Statement.** Let e := v2(d). If v2(a) <= v2(a-d) (the existence condition for Rad2(a; a-d)), then v2(r) <= e, and consequently gamma := gcd(r, 2d) divides d.

**Proof.** Write d = 2^e m with m odd. Suppose for contradiction v2(r) >= e+1, say r = 2^{e+1} r'. First, v2(2dq) = e + 1 + v2(q) >= e+1, so 2^{e+1} divides both terms of a = 2dq + r, giving v2(a) >= e+1. Second, r - d = 2^e (2r' - m) with 2r' - m odd, so v2(r-d) = e exactly. Then a - d = 2dq + (r - d) is a sum of a term of 2-adic valuation >= e+1 and a term of valuation exactly e, so v2(a-d) = e. Hence v2(a) >= e+1 > e = v2(a-d), contradicting the existence condition. Therefore v2(r) <= e. Now let gamma = gcd(r, 2d). Its 2-part: v2(gamma) = min(v2(r), v2(2d)) = v2(r) <= e (as v2(r) <= e < e+1 = v2(2d)). Its odd part divides the odd part of 2d, which is m. A positive integer divides d = 2^e m iff its 2-adic valuation is <= e and its odd part divides m; gamma satisfies both, so gamma | d. QED. (Verified numerically for all 9341 admissible pairs with a <= 300.)

## Lemma 2 (First forced color)

**Statement.** Let chi be a valid coloring of [1,N] with chi(1) = 0. Then chi(d+1) = 1.

**Proof.** The triple T(1,1) = (1, 1, 1+d) is a solution triple, and 1 + d <= N by Lemma 0(ii) (N >= E + 4d + 1 > d + 1). Its first two entries have color chi(1) = 0; if chi(d+1) = 0 the triple would be monochromatic. Hence chi(d+1) = 1. QED.

## Lemma 3 (Transport Lemma)

**Statement.** Let chi be a valid coloring of [1,N] with chi(1) = 0. Let k0 in [1,N] satisfy chi(k0) = 1 and put D := (k0 - 1)a + d. Then for every y in [1, N - D - d] with chi(y) = 1: chi(y + d) = 0 and chi(y + 2d) = 1.

**Proof.** All four triples used below are genuine solution triples inside [1,N]; ranges are checked in brackets. Step 1: T(y, k0) = (y, k0, y + D) [x3 = y + D <= N - d <= N]. Since chi(y) = chi(k0) = 1, monochromaticity would follow from chi(y+D) = 1; hence chi(y + D) = 0. Step 2: T(y + D, 1) = (y + D, 1, y + D + d) [x3 <= N by hypothesis y <= N - D - d]. Since chi(y+D) = chi(1) = 0, we get chi(y + D + d) = 1. Step 3: T(y + d, k0) = (y + d, k0, y + d + D) [x1 = y + d >= 1; x3 = y + D + d <= N]. Since chi(y + d + D) = 1 = chi(k0), monochromaticity would follow from chi(y+d) = 1; hence chi(y + d) = 0. Step 4: T(y + d, 1) = (y + d, 1, y + 2d) [x3 = y + 2d <= y + D + d <= N, using D = (k0-1)a + d >= d]. Since chi(y+d) = chi(1) = 0, we get chi(y + 2d) = 1. QED. (Note: repeated values among x1, x2, x3 are permitted by the problem statement, so no genericity conditions are needed.)

## Lemma 4 (Pairing Lemma; definition of phi and p)

**Statement.** Let chi be valid on [1,N] with chi(1) = 0. Then for every s in [1, d]: chi(s + d) = 1 - chi(s). Define phi(s) := chi(s) for s in [1,d] (so phi(1) = 0), and define p : Z -> {0,1} by p(u) = phi(s) if u ≡ s (mod 2d) with s in [1,d], and p(u) = 1 - phi(s) if u ≡ s + d (mod 2d) with s in [1,d]. Then p is 2d-periodic, antisymmetric (p(u + d) = 1 - p(u) for all u), p(1) = 0, and chi(u) = p(u) for all u in [1, 2d].

**Proof.** Fix s in [1,d]. Not both 0: the triple T(s, 1) = (s, 1, s + d) lies in [1,N] (s + d <= 2d < N); if chi(s) = chi(s+d) = 0, then together with chi(1) = 0 it is monochromatic — contradiction. Not both 1: apply the Transport Lemma (Lemma 3) with k0 = d + 1 (chi(d+1) = 1 by Lemma 2), D = E = d(a+1), at y = s. The room condition s <= N - E - d holds since N - E - d = 2d(q+1) + 1 - d = 2dq + d + 1 >= d >= s. If chi(s) = 1, Lemma 3 yields chi(s + d) = 0. Hence exactly one of chi(s), chi(s+d) is 1, i.e. chi(s+d) = 1 - chi(s). The stated properties of p follow directly: 2d-periodicity by construction; antisymmetry: if u ≡ s (s in [1,d]) then u + d ≡ s + d and p(u+d) = 1 - phi(s) = 1 - p(u); if u ≡ s + d then u + d ≡ s (mod 2d) and p(u+d) = phi(s) = 1 - (1 - phi(s)) = 1 - p(u). Finally chi(u) = p(u) on [1,2d] is exactly phi's definition plus the pairing just proved. QED.

## Lemma 5 (Window Lemma)

**Statement.** Let chi be valid on [1,N], chi(1) = 0, with phi, p as in Lemma 4. Define s* := min{ s in [2, d+1] : p(s) = 1 } (well-defined since p(d+1) = 1 - phi(1) = 1), D* := (s* - 1)a + d, and X* := N - D*. Then chi(u) = p(u) for every u in [1, X*]. (Note X* >= N - E = 2d(q+1) + 1 >= 4d + 1, since D* <= D_{d+1} = E.)

**Proof.** First, chi(s*) = 1: if s* <= d this is phi(s*) = p(s*) = 1 by definition of phi; if s* = d+1 it is Lemma 2. So the Transport Lemma applies with k0 = s*, D = D*, and room condition y <= N - D* - d = X* - d. Fix s in [1,d]; let y0 := s if phi(s) = 1, else y0 := s + d. By Lemma 4, chi(y0) = 1 and y0 in [1, 2d]. CLAIM (induction on j >= 0): (a) if y0 + 2dj <= X* + d then chi(y0 + 2dj) = 1; (b) if y0 + 2dj + d <= X* then chi(y0 + 2dj + d) = 0. Base j = 0: (a) is the seed. (b): if y0 + d <= X* then y0 <= X* - d, so Transport at y = y0 gives chi(y0 + d) = 0. Inductive step: assume (a) at j, i.e. chi(y_j) = 1 for y_j = y0 + 2dj <= X* + d (if the hypothesis of (a) at j fails, the hypotheses at all larger j and of (b) at >= j fail too, and there is nothing to prove). For (b) at j: if y_j + d <= X* then y_j <= X* - d, Transport at y_j gives chi(y_j + d) = 0. For (a) at j+1: if y_j + 2d <= X* + d then y_j <= X* - d, Transport at y_j gives chi(y_j + 2d) = 1. This proves the claim. COVERAGE: let u in [1, X*]. Its residue mod 2d equals that of y0 or of y0 + d, where {y0, y0+d} ≡ {s, s+d}. Case u ≡ y0 (mod 2d): since y0 in [1,2d], the least positive representative is y0 itself, so u = y0 + 2dj for some j >= 0 with u <= X* < X* + d; by (a), chi(u) = 1 = p(u) (as p(u) = p(y0) = chi(y0) = 1 by construction). Case u ≡ y0 + d (mod 2d): if y0 = s (phi(s) = 1), the least positive representative is s + d = y0 + d, so u = y0 + d + 2dj with u <= X*; by (b), chi(u) = 0 = p(u) (p(u) = p(s+d) = 1 - phi(s) = 0). If y0 = s + d (phi(s) = 0), the least positive representative is s: either u = s <= 2d, a seed with chi(s) = phi(s) = 0 = p(u) (Lemma 4), or u = s + 2dj = y0 + d + 2d(j-1) for some j >= 1 with u <= X*, and (b) gives chi(u) = 0 = p(u). In all cases chi(u) = p(u). QED. (Machine-validated by SAT-entailment of every window fact, for every phi-pattern, on (a,d) in {(5,2),(9,2),(6,2),(7,3),(9,3)}.)

## Lemma 6 (Far-Zone Lemma)

**Statement.** Let chi be valid on [1,N], chi(1) = 0, with p, s*, X* as above, and E = d(a+1). Then chi(z) = 1 - p(z - E) for every z in [E+1, N].

**Proof.** Let u := z - E in [1, N - E], where N - E = 2d(q+1) + 1. Note u <= N - E <= X* (Lemma 5 note), so chi(u) = p(u) is known. Case 1: p(u) = 1. Then chi(u) = 1, and the triple T(u, d+1) = (u, d+1, u + E) = (u, d+1, z) lies in [1,N] (z <= N by hypothesis). Since chi(u) = chi(d+1) = 1 (Lemma 2), monochromaticity would follow from chi(z) = 1; hence chi(z) = 0 = 1 - p(u). Case 2: p(u) = 0. Sub-case u <= N - E - d: then u + d <= N - E and p(u + d) = 1 (antisymmetry), so by Case 1 applied to u + d, chi(z + d) = chi(E + u + d) = 0. The triple T(z, 1) = (z, 1, z + d) lies in [1,N] (z + d = E + u + d <= N). If chi(z) = 0, then chi(z) = chi(1) = chi(z+d) = 0 is monochromatic — contradiction; hence chi(z) = 1 = 1 - p(u). Sub-case u > N - E - d, i.e. u >= 2dq + d + 2: then u - d >= 2dq + 2 >= 1 and p(u - d) = 1, so by Case 1 applied to u - d (which is <= N - E), chi(z - d) = chi(E + (u-d)) = 0; also z - d >= E + 1 since u - d >= 1... more precisely u >= d + 1 (indeed u >= 2dq + d + 2 > d + 1), so z - d = E + (u - d) is in [E+1, N]. The triple T(z - d, 1) = (z - d, 1, z) lies in [1,N]. If chi(z) = 0, then chi(z-d) = chi(1) = chi(z) = 0 is monochromatic — contradiction; hence chi(z) = 1 = 1 - p(u). QED. (Machine-validated by SAT-entailment of every far-zone fact, for every phi-pattern, on the same six instances.)

## Lemma 7 (Endgame, Case A: phi not identically zero)

**Statement.** Assume the setting of Lemmas 0-6, with the existence condition v2(a) <= v2(a-d), and suppose s* <= d (i.e. phi(s) = 1 for some s in [2,d]). Then chi cannot be valid. More precisely, set c := phi(d). There exists u in [1, 2d] with p(u + r) = c and p(u) = 1 - c, and for any such u the solution triple (a + u, d, E + u) is monochromatic of color c.

**Proof.** EXISTENCE OF u. View p as a function on Z_{2d} (residues represented by [1, 2d]) and let P := { v in Z_{2d} : p(v) = c }. Suppose no u as stated exists; then for every u in Z_{2d}: p(u + r) = c implies p(u) = c, i.e. P - r ⊆ P, where P - r := { v : v + r in P }. Translation by r is a bijection of Z_{2d}, so |P - r| = |P|, forcing P - r = P. Iterating, P - jr = P for all j >= 0, so P is invariant under translation by every element of the subgroup <r> = gamma·Z_{2d}, where gamma = gcd(r, 2d). By Lemma 1, gamma | d, so d ∈ <r> and P - d = P. Now d ∈ P because p(d) = phi(d) = c. Hence 2d = d + d ∈ P (using P = P + d), i.e. p(2d) = c. But antisymmetry gives p(2d) = p(d + d) = 1 - p(d) = 1 - c. Contradiction; the required u exists. THE TRIPLE. Fix such a u in [1,2d] and consider T(a + u, d) = (a + u, d, a + u + (d-1)a + d) = (a + u, d, E + u). Ranges: x1 = a + u >= a + 1 >= 1; x3 = E + u <= E + 2d <= N by Lemma 0(ii) (N - E >= 4d + 1). Forced colors: (1) x1 = a + u <= a + 2d <= X*: indeed s* <= d gives D* <= (d-1)a + d, so X* = N - D* >= N - (d-1)a - d = 2a + 2d + 1 - r, and 2a + 2d + 1 - r - (a + 2d) = a + 1 - r >= a + 1 - d > 0 by Lemma 0(i). Hence by the Window Lemma chi(a + u) = p(a + u) = p(u + r), because a - r = 2dq ≡ 0 (mod 2d). By choice of u, chi(a+u) = c. (2) chi(d) = phi(d) = c by definition. (3) x3 = E + u ∈ [E+1, E+2d] ⊆ [E+1, N], so by the Far-Zone Lemma chi(E + u) = 1 - p(u) = 1 - (1 - c) = c. Thus all three entries have color c: the triple is monochromatic, contradicting validity of chi. QED. (The existence claim was verified exhaustively for all admissible (d, r) with d <= 12 and all 2^{d-1} patterns phi, and randomly up to d = 40; the full instantiation — triple membership, ranges, and forced colors — was verified for 40060 (a,d,phi) cases with a <= 120, zero failures.)

## Lemma 8 (Endgame, Case B: phi identically zero) and Theorem (upper bound)

**Statement.** Assume the setting of Lemmas 0-6 and suppose s* = d + 1 (i.e. phi(s) = 0 for all s in [1, d]). Then the solution triple (d+1-r, 2, a+2d+1-r) is monochromatic of color 0, so chi cannot be valid. THEOREM: consequently, for every d >= 2 and every a with v2(a) <= v2(a - d), writing a = 2dq + r with 1 <= r <= d, every 2-coloring of [1, (a+3)d + a - (r-1)] admits a monochromatic solution of x1 + a x2 - x3 = a - d; that is, Rad2(a; a-d) <= (a+3)d + a - (r-1) = (a+3)d + a - min(r-1, d-1).

**Proof.** CASE B KILL. Here X* = N - D_{d+1} = N - E = a + 2d + 1 - r = 2d(q+1) + 1. Consider T(d+1-r, 2) = (d+1-r, 2, (d+1-r) + a + d) = (d+1-r, 2, a + 2d + 1 - r). Ranges: x1 = d+1-r >= 1 since r <= d; x3 = a + 2d + 1 - r = X* <= N. Forced colors: (1) chi(d+1-r) = phi(d+1-r) = 0, since d+1-r ∈ [1,d] and phi ≡ 0. (2) chi(2) = phi(2) = 0, using d >= 2 so that 2 ∈ [1,d]. (3) x3 = X* is in the window, and X* - 1 = 2d(q+1) ≡ 0 (mod 2d), so X* ≡ 1 (mod 2d) and, by the Window Lemma, chi(X*) = p(X*) = p(1) = phi(1) = 0. All three entries have color 0: monochromatic, contradicting validity. THEOREM ASSEMBLY. Suppose some 2-coloring chi of [1, N], N = (a+3)d + a - (r-1), has no monochromatic solution. By Lemma 0(iv) assume chi(1) = 0. Lemmas 2 and 4 define phi and p; s* = min{ s in [2, d+1] : p(s) = 1 } is well-defined. If s* <= d, Lemma 7 yields a monochromatic solution triple — contradiction. If s* = d+1, then p(s) = 0 for all s in [2, d], and phi(1) = 0, so phi ≡ 0 on [1,d]; the Case B kill yields a monochromatic triple — contradiction. Hence no such chi exists and Rad2(a; a-d) <= N. Since r <= d gives min(r-1, d-1) = r-1, this is exactly the conjectured upper bound. QED. (Remark on tightness: at length N-1 the same derivations lose exactly one unit of room — the Window Lemma then forces chi = p only up to X* - 1 — and the machine-verified extremal family, which realizes phi ≡ 0 and inserts its d long runs beyond the shortened window, shows the bound cannot be improved.)
