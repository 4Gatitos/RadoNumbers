# Draft: C2-reduction

**status**: complete

## Summary

Conjecture 2 of Dwivedi–Tripathi is proved CONDITIONALLY on Conjecture 1 (C1-general): for every a >= 1 and every c with v2(a) <= v2(c) and c >= a(K+2), where M := a(a+3)+1, X := 1+c(a+3), K+1 := ceil(X/M), we have Rad2(a;c) = K+1. The proof is a reflection (complementary-coloring) reduction generalising D-T's own Theorem 7 argument: the involution sigma(x) = N+1-x on [1,N] with N = K+1 carries solutions of x1+a*x2-x3 = c to solutions of the same equation with constant c' = a(K+2)-c; the hypothesis c >= a(K+2) makes c' <= 0; the ultrametric inequality transfers the 2-adic existence condition (v2(c') >= v2(a)); and the exact ceiling identity (a+3)(a-c')+1 = (K+1) - s with s = (K+1)M - X in [0, M-1] shows the C1 value at c' fits inside [1, K+1], so every 2-coloring of [1,K+1] reflects to one containing a monochromatic c'-solution, which reflects back to a monochromatic c-solution. The matching lower bound Rad2(a;c) >= K+1 is D-T Theorem 6 (citable); it is additionally re-derived self-containedly by reflecting D-T Theorem 2's valid coloring through sigma on [1,K] (using c'' := a(K+1)-c < a, valid for ALL c > a, and K <= (a+3)(a-c''), which is exactly KM <= X-1). Boundary case c = a(K+2): then c' = 0, d' = a, and the required instance Rad2(a;0) = a(a+3)+1 is UNCONDITIONALLY known (D-T Theorem 4), so the entire boundary family needs no conjecture; the family is nonempty since c = a(M+1) gives X = M^2 and c = a(K+2) exactly. No circularity: K is a closed-form function of (a,c) (no iteration), and C1 is invoked only at c' <= 0 < c, strictly outside the regime being proved. The hypothesis is non-vacuous and asymptotically automatic: c >= 2aM implies c >= a(K+2) (exact proof; verified exhaustively for a <= 12). All steps verified by exact SAT computation on 307 (a,c) pairs with a in [1,10], including 220+ exact-boundary cases (c'=0), 76+ interior cases (c' in [-10,-1], slack s in [0,28]), ~150 cases with v2(c) > v2(a) strictly, and edge cases a=1, a=2, a=4 (v2=2), a=8 (v2=3): zero failures.

## Notation and Standing Hypotheses (N0)

**Statement.** Throughout, a >= 1 is an integer and E_c denotes the equation x1 + a*x2 - x3 = c, whose solutions are unrestricted triples in the given interval (repeated values allowed). Rad2(a;c) is the least N such that every 2-coloring of [1,N] admits a monochromatic solution of E_c in [1,N]^3; by the 2-adic criterion (D-T / this project's existence lemma), Rad2(a;c) exists iff v2(a) <= v2(c), with v2(0) = +infinity. Set M := a(a+3) + 1 = a^2 + 3a + 1 (odd for all a, and M >= 5). For c > a set X := 1 + c(a+3) and K := ceil(X/M) - 1, i.e. K+1 = ceil(X/M). C1-general is the assumed statement: for every integer c* <= 0 with v2(a) <= v2(c*), Rad2(a;c*) = (a+3)(a - c*) + 1. Basic facts used without further comment: (F1) for integers p and q >= 1, ceil(p/q) is the unique integer m with (m-1)q < p <= mq; (F2) a 2-coloring of [1,N] restricted to [1,N0], N0 <= N, is a 2-coloring of [1,N0], and any solution of E_c in [1,N0]^3 is a solution in [1,N]^3; (F3) 'Rad2(a;c) > L' is equivalent to 'there exists a 2-coloring of [1,L] with no monochromatic solution of E_c in [1,L]^3' — this equivalence holds verbatim also when Rad2(a;c) does not exist (interpret Rad2 = +infinity: if no N works, then for every L a valid coloring of [1,L] exists by definition of the minimum).

**Proof.** Definitional; (F1) is the characterisation of the ceiling; (F2) is immediate from [1,N0] being a subinterval of [1,N] and monochromaticity/solutionhood being properties of the triple and the colors of its entries only; (F3): Rad2 is defined as a minimum over N of a monotone property (if every coloring of [1,N] has a mono solution, the same holds for every N' >= N, since a coloring of [1,N'] restricts to [1,N] and by (F2) the mono solution found there lives in [1,N']^3). Hence Rad2 > L iff the property fails at N = L iff a valid (solution-free) coloring of [1,L] exists; if the property fails at every N, Rad2 does not exist and valid colorings exist at every L.

## Lemma 1 (Reflection Identity)

**Statement.** Fix an integer N >= 1 and let sigma: [1,N] -> [1,N], sigma(x) = N + 1 - x. Then sigma is an involutive bijection, and for every triple (x1,x2,x3) in [1,N]^3 and every integer c: (x1,x2,x3) solves E_c if and only if (sigma(x1), sigma(x2), sigma(x3)) solves E_{c^}, where c^ := a(N+1) - c. Moreover, for every 2-coloring chi of [1,N], setting chi' := chi o sigma (i.e. chi'(y) = chi(N+1-y)), a triple (y1,y2,y3) in [1,N]^3 is chi'-monochromatic iff (sigma(y1),sigma(y2),sigma(y3)) is chi-monochromatic. Consequently: chi admits a monochromatic solution of E_c in [1,N]^3 if and only if chi' admits a monochromatic solution of E_{c^} in [1,N]^3. The map c |-> a(N+1) - c is an involution, matching sigma being one.

**Proof.** sigma maps [1,N] to itself bijectively: for x in [1,N], N+1-x is in [1,N], and sigma(sigma(x)) = N+1-(N+1-x) = x. Equation transform (exact computation): with yi = N+1-xi, y1 + a*y2 - y3 = (N+1-x1) + a(N+1-x2) - (N+1-x3) = (N+1)*(1 + a - 1) - (x1 + a*x2 - x3) = a(N+1) - (x1 + a*x2 - x3). Hence y1 + a*y2 - y3 = c^ = a(N+1) - c iff x1 + a*x2 - x3 = c. Since sigma is an involution, the same identity applied to (y1,y2,y3) shows the converse direction with the involuted constant a(N+1) - c^ = c; so the correspondence is a bijection between the solution set of E_c in [1,N]^3 and the solution set of E_{c^} in [1,N]^3 (componentwise sigma). Monochromaticity: chi'(y1) = chi'(y2) = chi'(y3) means chi(sigma(y1)) = chi(sigma(y2)) = chi(sigma(y3)), which is exactly chi-monochromaticity of the sigma-image triple; conversely by involutivity. Combining: if chi has a chi-mono E_c-solution (x1,x2,x3), then (sigma(x1),sigma(x2),sigma(x3)) is an E_{c^}-solution (shown above, applied to xi = sigma(yi)) and is chi'-mono because chi'(sigma(xi)) = chi(sigma(sigma(xi))) = chi(xi); and symmetrically. No boundary issues: all six triple entries stay in [1,N] because sigma is a bijection of [1,N]. This was verified computationally (exhaustive solution-set comparison) for all 307 tested (a,c,N) instances.

## Lemma 2 (Exact Ceiling Arithmetic)

**Statement.** Let a >= 1, c > a, X = 1 + c(a+3), M = a(a+3)+1, K+1 = ceil(X/M). Define s := (K+1)M - X, c' := a(K+2) - c, c'' := a(K+1) - c. Then, with exact integer arithmetic: (i) K >= 1, hence a(K+2) >= 3a and the hypothesis c >= a(K+2) forces c >= 3a (consistent with, and stronger than, c > a); (ii) s is an integer with 0 <= s <= M-1; (iii) KM <= X - 1, equivalently KM <= c(a+3); (iv) (a+3)(a - c') + 1 = (K+1) - s; in particular (a+3)(a - c') + 1 <= K + 1, with equality iff M divides X; (v) (a+3)(a - c'') = X - 1 - KM + K >= K, i.e. K <= (a+3)(a - c''); (vi) c'' < a holds for EVERY c > a (no hypothesis on c beyond c > a); (vii) if c >= a(K+2) then c' <= 0 and c'' = c' - a <= -a < 0; moreover a - c' = c - a(K+1) >= a >= 1, so d' := a - c' >= a, consistent with c' <= 0.

**Proof.** (i) c > a >= 1 gives X = 1 + c(a+3) >= 1 + (a+1)(a+3) = a^2 + 4a + 4 > a^2 + 3a + 1 = M, so X >= M + 1 and K + 1 = ceil(X/M) >= 2, i.e. K >= 1. Then a(K+2) >= a*3 = 3a. (ii) By (F1), (K+1-1)M < X <= (K+1)M, so 0 <= (K+1)M - X = s and s = (K+1)M - X < (K+1)M - KM = M, so s <= M - 1 (s integer). (iii) From (F1)'s strict left inequality: KM < X, and both sides integers, so KM <= X - 1 = c(a+3). (iv) a - c' = a - a(K+2) + c = c - a(K+1). Compute (a+3)(a - c') + 1 = (a+3)(c - a(K+1)) + 1 = c(a+3) + 1 - a(a+3)(K+1) = X - (M-1)(K+1) = X - M(K+1) + (K+1) = (K+1) - s. Since s >= 0, this is <= K+1; equality iff s = 0 iff M | X. (v) a - c'' = c - aK, so (a+3)(a - c'') = c(a+3) - a(a+3)K = (X - 1) - (M-1)K = (X - 1 - KM) + K >= 0 + K = K, using (iii). (vi) c'' < a iff a(K+1) - c < a iff aK < c. From (iii), aK <= a * c(a+3)/M = c * (M-1)/M * (a(a+3)/(M-1)) — more directly: aK * M <= a * c(a+3) = c(M-1) < cM, so aK < c since M >= 1. (Exact: KM <= c(a+3) implies aKM <= ac(a+3) = c(M-1) = cM - c < cM, divide the strict integer inequality aKM < cM by M > 0.) (vii) c >= a(K+2) gives c' = a(K+2) - c <= 0 and c'' = c' - a <= -a < 0 directly (note c' - c'' = a(K+2) - a(K+1) = a). Also a - c' = c - a(K+1) >= a(K+2) - a(K+1) = a >= 1. All identities in (ii)-(v) were confirmed numerically (slack identity (a+3)(a-c')+1 = K+1-s, and 0 <= s <= M-1, and K <= (a+3)(a-c'')) on all 307 tested pairs, with s realized across [0, 54] including s = 0 and s = M-1 endpoints.

## Lemma 3 (2-adic Transfer of the Existence Condition)

**Statement.** Let a >= 1 and c be integers with v2(a) <= v2(c), and let n be any integer (in our application n = K+2 or n = K+1). Then c* := a*n - c satisfies v2(c*) >= v2(a); in particular v2(a) <= v2(c*), so the existence criterion for Rad2(a;c*) holds. This covers c* = 0 (v2(0) = +infinity >= v2(a)).

**Proof.** The 2-adic valuation is ultrametric: for integers u, v, v2(u - v) >= min(v2(u), v2(v)), valid with the convention v2(0) = +infinity (if u = v both sides are handled: LHS = infinity). Proof of the ultrametric inequality for completeness: write m = min(v2(u), v2(v)) (if u = v = 0 the claim is trivial); then 2^m | u and 2^m | v, hence 2^m | (u - v), so v2(u - v) >= m. Now v2(a*n) = v2(a) + v2(n) >= v2(a) (with v2(a*n) = +infinity if n = 0, still >= v2(a)), and v2(c) >= v2(a) by hypothesis. Therefore v2(c*) = v2(a*n - c) >= min(v2(a*n), v2(c)) >= v2(a). Verified numerically on all 307 tested pairs (rado_exists(a, c') returned True in every case, including c' = 0 and cases with v2(c) > v2(a) strictly, e.g. a = 2 with 4 | c, a = 4 with 8 | c, a = 8 with 16 | c).

## Lemma 4 (Lower Bound: Rad2(a;c) >= K+1 for all c > a)

**Statement.** For every a >= 1 and every integer c > a: there exists a 2-coloring of [1, K] with no monochromatic solution of E_c in [1,K]^3; equivalently (by N0(F3)) Rad2(a;c) >= K + 1 = ceil((1 + c(a+3))/(1 + a(a+3))). No 2-adic hypothesis is needed. [This is Dwivedi-Tripathi, Integers 20 (2020) #A36, Theorem 6, cited as proven; we additionally give a self-contained re-derivation from their Theorem 2 via reflection, so the lower bound does not depend on reading Theorem 6's construction.]

**Proof.** Primary justification: cite D-T Theorem 6 verbatim (Rad2(a;c) >= ceil((1+c(a+3))/(1+a(a+3))) for all c > a). Self-contained re-derivation from D-T Theorem 2: Theorem 2 states that for every integer cc < a, Rad2(a;cc) >= (a+3)(a - cc) + 1; by N0(F3) this means exactly: there exists a 2-coloring chi0 of [1, L], L := (a+3)(a - cc), with no monochromatic solution of E_{cc} in [1,L]^3 (their proof exhibits an explicit such coloring; the statement is non-vacuous and holds whether or not Rad2(a;cc) is finite). Apply this with cc := c'' = a(K+1) - c. By Lemma 2(vi), c'' < a for every c > a, so Theorem 2 applies; note c'' may be negative or of any residue — Theorem 2 has no other hypotheses. By Lemma 2(v), K <= (a+3)(a - c'') = L. Restrict chi0 to [1, K] (valid by N0(F2)): chi0|[1,K] has no monochromatic E_{c''}-solution in [1,K]^3, since such a solution would also be one in [1,L]^3. Now apply Lemma 1 with N := K and define chi := chi0|[1,K] o sigma_K, sigma_K(x) = K+1-x. By Lemma 1, chi has a monochromatic E_{c^}-solution in [1,K]^3 with c^ = a(K+1) - c'' = a(K+1) - (a(K+1) - c) = c if and only if chi0|[1,K] has a monochromatic E_{c''}-solution in [1,K]^3 — which it does not. (Direction check: Lemma 1 applied with the pair (chi0|[1,K], c'') and its reflected pair (chi, c^): the two constants c'' and c^ = a(K+1) - c'' are exchanged by the involution, and the lemma is an iff, so absence transfers.) Hence chi is a 2-coloring of [1,K] with no monochromatic E_c-solution, giving Rad2(a;c) >= K+1. Degenerate check: if K = 1 the interval [1,1] and any coloring works only if (1,1,1) is not a solution of E_c, i.e. 1 + a - 1 = a != c, true since c > a; and indeed the argument above covers this uniformly (a solution-free coloring of [1,1] exists since the only triple (1,1,1) solves E_a, not E_c). Numerical confirmation: for all 307 tested pairs the exact SAT computation returned Rad2(a;c) = K+1, which in particular certifies (by an explicit SAT witness coloring of [1,K]) that Rad2(a;c) > K in every case.

## Lemma 5 (Upper Bound under C1-general)

**Statement.** Assume C1-general: for every integer c* <= 0 with v2(a) <= v2(c*), Rad2(a;c*) = (a+3)(a - c*) + 1. Let a >= 1 and let c satisfy v2(a) <= v2(c) and c >= a(K+2), where K+1 = ceil((1+c(a+3))/M). Then every 2-coloring chi of [1, K+1] admits a monochromatic solution of E_c in [1, K+1]^3. In particular Rad2(a;c) exists and Rad2(a;c) <= K+1.

**Proof.** Set N := K+1, sigma(x) := N+1-x = K+2-x on [1,N], chi' := chi o sigma, and c' := a(N+1) - c = a(K+2) - c. Step 1 (applicability of C1 at c'). By hypothesis c >= a(K+2), so c' <= 0 (Lemma 2(vii)). By Lemma 3 with n = K+2, v2(c') >= v2(a), i.e. v2(a) <= v2(c'). Hence C1-general applies to the pair (a, c') and yields Rad2(a;c') = (a+3)(a - c') + 1 =: R'. (When c' = 0: d' = a - c' = a, R' = a(a+3)+1 = M; see Lemma 6 — this instance is even unconditional.) Step 2 (R' fits in [1,N]). By Lemma 2(iv), R' = (K+1) - s with s = (K+1)M - X in [0, M-1], hence R' <= K+1 = N. Also R' >= 1 trivially (indeed R' = (a+3)(a-c')+1 >= (a+3)a + 1 = M >= 5, since a - c' >= a by Lemma 2(vii)). Step 3 (find a mono c'-solution for chi'). chi' is a 2-coloring of [1,N]; restrict it to [1,R'] (N0(F2)). Since R' = Rad2(a;c') and Rad2 is exactly the least such N, every 2-coloring of [1,R'] — in particular chi'|[1,R'] — admits a monochromatic solution (y1,y2,y3) in [1,R']^3 of E_{c'}. Since R' <= N, (y1,y2,y3) is in [1,N]^3 and is chi'-monochromatic. Step 4 (reflect back). Apply Lemma 1 with this N: the triple (x1,x2,x3) := (sigma(y1), sigma(y2), sigma(y3)) is in [1,N]^3, solves E_{c^} with c^ = a(N+1) - c' = a(K+2) - (a(K+2) - c) = c, and is monochromatic for chi' o sigma = chi o sigma o sigma = chi (involutivity, Lemma 1). Hence chi has a monochromatic solution of E_c in [1,K+1]^3. Since chi was arbitrary, Rad2(a;c) <= K+1, and in particular Rad2(a;c) is finite (its existence also follows independently from v2(a) <= v2(c), but the argument delivers finiteness directly). Boundary/edge check: (a) c' = 0 (exactly when c = a(K+2)) is covered — Lemma 3 gives v2(0) = infinity >= v2(a), and C1-general includes c* = 0; (b) a = 1: v2(1) = 0 <= v2(anything), no constraint, all steps go through (verified numerically for c in {6,...,29}); (c) a = 2 and higher powers of 2: only Steps 1 uses v2, via Lemma 3 (verified for a in {2,4,8} including v2(c) > v2(a) strictly); (d) s = 0 (M | X): then R' = K+1 = N exactly and Step 2 is tight — the restriction in Step 3 is the identity; verified numerically (e.g. a=5, c=210: R' = N = 41; a=8, c=1432: R' = N = 177); (e) s = M-1: R' = K+2-M, the loosest case; verified (e.g. a=5, c=410: s = 40 = M-1).

## Theorem (Conjecture 2, conditional on Conjecture 1)

**Statement.** Assume C1-general: for every c* <= 0 with v2(a) <= v2(c*), Rad2(a;c*) = (a+3)(a - c*) + 1. Then for every a >= 1 and every integer c with v2(a) <= v2(c) and c >= a(K+2), where K := ceil((1 + c(a+3))/(1 + a(a+3))) - 1, one has Rad2(a;c) = K + 1. (Any such c automatically satisfies c >= 3a > a by Lemma 2(i), so the standing regime c > a of D-T Theorems 6-7 is implied, not additionally assumed.)

**Proof.** Upper bound: Lemma 5 shows every 2-coloring of [1, K+1] contains a monochromatic solution of E_c, so Rad2(a;c) exists and Rad2(a;c) <= K+1. Lower bound: Lemma 4 (D-T Theorem 6, independently re-derived from D-T Theorem 2 via Lemma 1) gives a 2-coloring of [1,K] with no monochromatic E_c-solution, so Rad2(a;c) >= K+1. Hence Rad2(a;c) = K+1. Non-vacuousness of the hypothesis (Lemma 7): the range { c : c >= a(K+2) } contains every admissible c >= 2aM, and also the exact-boundary family starting at c = a(M+1). Verified numerically: Rad2(a;c) = K+1 by exact SAT computation for all 307 tested pairs (a in [1,10], c up to 2630).

## Lemma 6 (Boundary Case c = a(K+2): Unconditional)

**Statement.** If c = a(K+2) exactly (equivalently c' = 0), then the instance of C1-general invoked in Lemma 5 is Rad2(a;0) = a(a+3) + 1 = M, i.e. the case c* = 0, d* = a - c* = a. This instance is UNCONDITIONALLY known: it is D-T Theorem 4 (equality Rad2(a;c) = (a+3)(a-c)+1 for all c <= 0 with a | c), applied at c = 0. Consequently, for every c with v2(a) <= v2(c) and c = a(K+2), the conclusion Rad2(a;c) = K+1 holds unconditionally (no conjecture needed). More generally, whenever the reflected constant c' = a(K+2) - c lands in a zone where Conjecture 1 is already proven (a | c' by Theorem 4; or a odd with t' = c' mod a in {0, a-1} or c' <= -a(a - t' - 2)/2 by Theorem 3 + Result R1), the conclusion for that c is unconditional. The boundary family is nonempty and structured: c0 := a(M+1) satisfies c0 = a(K+2) exactly.

**Proof.** First claim: c = a(K+2) gives c' = a(K+2) - c = 0, and Lemma 5's Step 1 then invokes Rad2(a;0) = (a+3)(a - 0) + 1 = a(a+3) + 1 = M. The pair (a, 0) satisfies a | 0 and 0 <= 0, so D-T Theorem 4 applies and gives exactly this value unconditionally (v2(0) = infinity >= v2(a), so existence holds for every a). All other steps of Lemma 5 and Lemma 4 are unconditional, hence so is the conclusion at such c. Second claim (nonemptiness): let c0 = a(M+1). Then X = 1 + a(M+1)(a+3) = 1 + (M+1)(M-1) = 1 + M^2 - 1 = M^2 (using a(a+3) = M - 1), so K+1 = ceil(M^2/M) = M, K+2 = M+1, and a(K+2) = a(M+1) = c0 exactly. Note v2(c0) = v2(a) + v2(M+1) >= v2(a), so c0 is always admissible. Sanity check against the general formula: at c = c0, s = (K+1)M - X = M*M - M^2 = 0 and R' = K+1 = M = Rad2(a;0) — the reflection is tight. Numerically, 220 exact-boundary pairs (c = a(K+2), a in [1,10]) were verified end-to-end, including the first ones c0 = a(M+1) for each a (e.g. (a,c) = (1,6), (2,24), (3,60), (5,210), (9,990), (10,1320)), with Rad2(a;c) = K+1 and Rad2(a;0) = M confirmed by SAT in every case; Rad2(a;0) = a(a+3)+1 was independently confirmed for all a in [1,10]. Caution recorded: consecutive admissible multiples of a form long runs of exact-boundary cases (for a=5, every multiple of 5 in [210, 410] has c = a(K+2), with s sweeping 0..40 = M-1), and between them the hypothesis c >= a(K+2) FAILS for non-multiples (e.g. a=5, c=211: a(K+2) = 215 > 211) — the theorem's hypothesis is genuinely a restriction below 2aM; see Lemma 7.

## Lemma 7 (Non-circularity and Non-vacuousness of the Hypothesis)

**Statement.** (i) Non-circularity: K is defined by the closed formula K = ceil((1 + c(a+3))/M) - 1 directly from (a,c); no fixed-point or iterative definition is involved, and the hypothesis 'c >= a(K+2)' is a directly decidable inequality in (a,c). The conjecture C1-general is invoked only at the pair (a, c') with c' <= 0, while the statement being proved concerns c >= a(K+2) >= 3a > 0; the reduction maps the regime c > a strictly into the disjoint regime c* <= 0, so no instance of the theorem is used in its own proof, and no instance of C1 being assumed lies in the range being concluded. (ii) Non-vacuousness / eventual coverage: if c >= 2aM = 2a(a^2 + 3a + 1), then c >= a(K+2). Hence the theorem covers all admissible c from 2aM on (in particular 'c sufficiently large' in D-T's Theorem 7 can be taken as c >= 2a(a^2+3a+1)), plus the sporadic exact-boundary family below that threshold (Lemma 6).

**Proof.** (i) Inspection of the definitions: K+1 = ceil(X/M) with X = 1 + c(a+3) and M = a(a+3) + 1 are explicit integer expressions in (a,c); c' = a(K+2) - c is then explicit; Lemma 5 uses C1-general only at (a, c') and c' <= 0 < 3a <= c, so (a,c') is never an instance of the theorem's own conclusion (whose constants are >= 3a > 0). The logical dependency graph is: [D-T T2, T6 (proven)] + [C1-general at nonpositive constants (assumed)] ==> [Theorem at c >= a(K+2)]. Acyclic. (ii) Exact computation: K+1 = ceil(X/M) <= (X + M - 1)/M, hence a(K+2) = a(K+1) + a <= a(X + M - 1)/M + a. Now a(X + M - 1) = a(c(a+3) + M) = ac(a+3) + aM = c(M-1) + aM. Therefore a(K+2) <= (c(M-1) + aM)/M + a = c - c/M + 2a. Thus a(K+2) <= c holds whenever c - c/M + 2a <= c, i.e. c/M >= 2a, i.e. c >= 2aM. (All steps are exact over the rationals; the single inequality used, ceil(z) <= (numerator + denominator - 1)/denominator for integer numerator, is the standard exact bound.) Numerically verified with zero violations for all a in [1,12] and ALL integers c in [2aM, 6aM) (not only admissible ones): c >= a(K+2) held in every case. Also verified that the bound cannot be relaxed to a(M+1) <= c in general: a=5, c=211 in [a(M+1), 2aM) = [210, 410) fails the hypothesis (a(K+2) = 215 > 211), so below 2aM the theorem applies exactly to those c satisfying the stated inequality (empirically, the multiples of a in that window, all of which are exact-boundary cases and hence unconditional by Lemma 6).

## Gaps
- The result is CONDITIONAL by design: the upper bound (Lemma 5) assumes C1-general (D-T Conjecture 1) at the single reflected constant c' = a(K+2) - c <= 0. Unconditional today exactly when c' falls in a proven zone of C1: c' = 0 or a | c' (D-T Theorem 4 — covers the entire boundary family c = a(K+2)); a odd with t' = c' mod a in {0, a-1}, or c' <= -a(a-t'-2)/2 (D-T Theorem 3 + Result R1). Once the companion mission (extending this project's q >= 1 upper-bound proof to q = 0, i.e. proving C1-general in full) is complete, this theorem becomes unconditional with no further work.
- The lower bound cites D-T Theorem 2 (and Theorem 6) as proven black boxes; I did not re-verify D-T's explicit Theorem 2 coloring line by line. Mitigations: (a) Lemma 4 re-derives Theorem 6 from Theorem 2 alone via the reflection lemma, so only Theorem 2 is load-bearing; (b) the reading of 'Rad2 >= L+1' as 'a solution-free coloring of [1,L] exists' is justified in N0(F3) including the infinite-Rad2 case; (c) the exact SAT computations independently certified Rad2(a;c) > K (via an explicit witness coloring found by the solver) for all 307 tested pairs, so the lower bound is machine-confirmed on the whole test grid.
- The theorem says nothing about a < c < a(K+2) off the boundary family (e.g. a=5, c=211, where a(K+2) = 215 > c): there the reflected constant c' is positive and the reduction to C1 does not apply. This matches the scope of D-T Conjecture 2 exactly (it is stated only for c >= a(K+2)) — not a gap in the proof, but recorded so the range is not overclaimed. D-T Theorem 6 still gives the lower bound Rad2 >= K+1 there; equality can fail below the threshold.
- Numerical verification covers a in [1,10] (plus the arithmetic-only checks for a in [1,12]) and c up to 2630; all identities proven in Lemmas 1-3 and 7(ii) are exact integer/ultrametric arguments with no asymptotic step, so no coverage gap is expected for larger parameters, but the SAT confirmations are finite by nature.
