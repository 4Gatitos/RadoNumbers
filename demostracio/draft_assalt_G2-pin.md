## Lemma M (monotonicity of family feasibility)

**Statement.** Let (a,c) be a non-degenerate middle-band pair and N >= 2. If some valid colouring chi of [1,N] has gate chi|_W(N) in F (<=3 runs), then chi' = chi|_[1,N-1] is a valid colouring of [1,N-1] with gate chi'|_W(N-1) in F.

**Proof.** Validity: every solution triple of x1+a*x2-x3=c inside [1,N-1] is a triple inside [1,N], so chi' inherits the absence of monochromatic solutions. Windows: by Lemma W(ii) (proven in draft_L3-CCL.md) each W(M) = {k in [1,M] : 1 <= |ak-c| <= M-1} is a set of consecutive integers; if k in W(N-1) then k <= N-1 <= N and |ak-c| <= N-2 <= N-1, so k in W(N). Hence W(N-1) is a contiguous subinterval of the interval W(N), and the word chi'|_W(N-1) is the restriction of chi|_W(N) to a block of consecutive coordinates. Restricting a word to an interval of coordinates cannot increase its number of maximal runs (each run of the restriction is contained in a run of the original, and their order is preserved). Since F is exactly the set of words with <= 3 runs (Lemma F of draft_L3-CCL.md), chi'|_W(N-1) is in F.

## Theorem E (extremal reduction: gen = Rad2 iff CCL at N = Rad2-1)

**Statement.** Fix a non-degenerate band pair (a,c) with Rad2 = Rad2(a;c) finite. Let fam(N) := 'there exists G in F with Psi(G,N) satisfiable'. If fam(Rad2-1) holds, then fam(N) holds iff N <= Rad2-1, and the Layer-3 search of midband_gen.py returns exactly Rad2, i.e. gen(a,c) = Rad2(a;c). Consequently gen = Rad2 on the whole band if and only if CCL holds at the single value N = Rad2(a;c)-1 for every band pair; CCL at all other N is not needed for the exactness of the generator.

**Proof.** For N <= Rad2-1: iterate Lemma M downward from Rad2-1 to N, giving fam(N). For N >= Rad2: if fam(N) held, the 2-SAT model would be a valid colouring of [1,N] (Lemma S of draft_L3-CCL.md, unconditional soundness), contradicting N >= Rad2. So fam(N) = [N <= Rad2-1], a monotone predicate with first failure exactly at Rad2. The generator's decide(N) computes fam(N) (Lemma S: decide is the disjunction of Psi(G,N)-satisfiability over G in F). Its search: decide(K1) is evaluated first, where K1 = K+1 <= Rad2 by Dwivedi-Tripathi Theorem 6 (published input); if decide(K1) is false then K1 = Rad2 and the code returns K1 = Rad2. Otherwise the doubling climb maintains the invariant decide(lo)=True, terminates with decide(hi)=False (hi exists since decide(Rad2)=False), and the binary search on an exactly monotone 0/1 predicate returns the first false point, which is Rad2. (The cap_guard can only raise an exception, never return a wrong value.) Finally, at N = Rad2-1 the interval [1,N] is colourable, so 'Psi(G,N) satisfiable for some G' is true (take the gate of any valid colouring); hence CCL at N = Rad2-1 is equivalent to fam(Rad2-1). For N >= Rad2 CCL is vacuous, and for N < Rad2-1 it is implied by fam(Rad2-1) via Lemma M as far as the generator is concerned.

## Lemma W1 (single-arm normal form: suffix window, maximal AP)

**Statement.** Let (a,c) be non-degenerate band, lam = ceil(c/a), mu = lam*a-c in [1,a-1], t = a-mu, and let N <= lam-1. Put Delta = lam-1-N >= 0 and delta = a(lam-N)-mu = t + a*Delta. If delta >= N the window is empty. Otherwise W(N) = [N-m+1, N] is a SUFFIX of [1,N] with m = floor((N-1-delta)/a) + 1, all owners are left-arm (s_k < 0), and the owner at position N-i has distance delta + a*i for i = 0..m-1 (a maximal arithmetic progression: every distance delta+a*i <= N-1 is realized).

**Proof.** For k <= N <= lam-1: s_k = ak-c <= a(lam-1)-c = -mu < 0, so every owner is left-arm and d_k = c-ak = a(lam-k)-mu. Since k <= lam-1, d_k >= a-mu = t >= 1 always, so activity is governed only by d_k <= N-1, i.e. a(lam-k) <= N-1+mu, i.e. k >= lam - floor((N-1+mu)/a) =: k0. Since N-1+mu < a*lam - ... : indeed N <= lam-1 < c = a*lam - mu gives N-1+mu <= a*lam - 1, hence floor((N-1+mu)/a) <= lam-1 and k0 >= 1; thus W(N) = [k0, N], a suffix. Writing k = N-i: d_{N-i} = a(lam-N+i)-mu = delta + a*i, and activity delta+a*i <= N-1 holds iff i <= (N-1-delta)/a, so m = floor((N-1-delta)/a)+1 and every term of the AP up to N-1 occurs.

## Lemma REF (reflection to the abstract head system)

**Statement.** In the situation of Lemma W1 (nonempty window), let rho(x) = N+1-x. Define the abstract system Sigma(a,delta,N): ground [1,N]; owners at positions j = 1..m, owner j owning distance D_j = delta + a(j-1) (m as in Lemma W1). Then chi is a valid colouring of [1,N] for x1+a*x2-x3=c if and only if chi o rho is a valid colouring of Sigma(a,delta,N). Gates reverse as words, so run counts and membership in F are preserved. In particular, for Delta = 0 (delta = t), Sigma(a,t,N) is exactly the small-c system of the equation x1+a*x2-x3 = a-t on [1,N] (this recovers and extends Lemma 3 of draft_L2-tail.md to every N <= lam-1, without deep-tail hypotheses).

**Proof.** By Lemma R (draft_L3-CCL.md), validity of chi is: for each owner k with distance d_k, the class chi^{-1}(chi(k)) contains no pair {p, p+d_k} in [1,N]. rho is a colour-preserving involution of [1,N] that maps pairs at distance d bijectively onto pairs at distance d. The owner at position N-i (distance delta+a*i) maps to position i+1; so the owner set of the reflected system is exactly positions 1..m with owner j owning D_j = delta+a(j-1) — the system Sigma(a,delta,N). Constraints correspond one-to-one, so validity transfers both ways. The gate of chi o rho is the reversal of the gate of chi; reversal preserves run counts, and F is closed under reversal. For Delta = 0: the equation x1+a*x2-x3 = a-t on [1,N] has solutions (x, k, x+g_k) with g_k = a(k-1)+t > 0, so its owners are positions k with g_k <= N-1, i.e. k = 1..m with the same distances D_k: identical system.

## Lemma A (constant gate: exact characterization)

**Statement.** In Sigma(a,delta,N) with m >= 1 owners: the constant gate (all owners colour g) is feasible if and only if delta >= m, equivalently N <= (a+1)delta. Witness: chi = g on [1,m], 1-g on [m+1,N].

**Proof.** (delta >= m <=> N <= (a+1)delta): m-1 = floor((N-1-delta)/a) <= delta-1 iff N-1-delta <= a*delta - 1 iff N <= (a+1)delta. (Sufficiency): with chi = g exactly on [1,m], the class of g is an interval of length m, whose internal distances are <= m-1 < delta <= D_j for every j, so no forbidden pair; the class of 1-g contains no owner, hence carries no constraint; all owner positions j <= m are coloured g, matching the gate. (Necessity): if delta <= m-1, then positions 1 and 1+delta are both owners (1+delta <= m), hence both coloured g under the constant gate; they form a pair at distance delta = D_1, which is owned by owner 1 of colour g — a violation. So no valid extension exists.

## Lemma C (three-block witness, delta >= a)

**Statement.** Let a >= 2, delta >= a, and delta+1 <= N <= (a+3)delta. Put p = min(delta, floor((N-delta)/(a+1))), s = N - delta - (a+1)p, q = N-p-s. Then 0 <= p <= delta, 0 <= s <= delta, q = delta + p*a >= delta >= 1, and the colouring chi = 1^p 0^q 1^s is valid for Sigma(a,delta,N); its gate is 1^{min(p,m)} 0^{max(m-p,0)}, with at most 2 runs.

**Proof.** Parameters: if floor((N-delta)/(a+1)) <= delta then p = floor((N-delta)/(a+1)) and s = (N-delta) mod (a+1) in [0,a], and a <= delta gives s <= delta. Otherwise p = delta and s = N-delta-(a+1)delta = N - (a+2)delta; here floor((N-delta)/(a+1)) >= delta+1 forces s >= a+1-... >= 0 (since N-delta >= (delta+1)(a+1) gives s >= a+1 > 0), and s <= delta iff N <= (a+3)delta, which holds by hypothesis. In both cases p(a+1)+s = N-delta, hence q = N-p-s = delta+pa. Validity: class 1 = [1,p] u [N-s+1,N], class 0 = [p+1, N-s], an interval of length q. Owners j <= min(p,m) sit in [1,p] (colour 1, matching the gate); owners p < j <= m sit in class 0 because j <= m <= (N-1-delta)/a + 1 <= N-delta <= N-s. Constraints: (i) owner j > p, colour 0, distance D_j >= D_{p+1} = delta+pa = q: an interval of length q contains no pair at distance >= q. (ii) owner j <= min(p,m), colour 1, distance D_j <= delta+(p-1)a = q-a: pairs of class 1 inside [1,p] have distance <= p-1 <= delta-1 < D_j; inside [N-s+1,N] distance <= s-1 <= delta-1 < D_j; across the two intervals the distance is at least (N-s+1)-p = q+1 > q-a >= D_j. Hence no forbidden pair exists and chi is valid. The gate word is 1^{min(p,m)} followed by 0s, at most 2 runs.

## Lemma BAND (band-membership identity and the in-band range of Lemma C)

**Proof.** With A = 1+a(a+3) and c = lam*a - mu: c < a(K+2) iff K >= lam-2 (integers, since c/a < K+2 iff lam <= K+2) iff ceil((1+c(a+3))/A) >= lam-1 iff 1+c(a+3) > A(lam-2). ## Lemma B (chi* prefix witness, Delta = 0, t >= 2)

**Statement.** Let t >= 2, and assume Rad2(a;c) exists (v2(a) <= v2(c)); then r = a mod 2t >= 1. Put rho = min(r,t) and N* = (a+3)t + a - rho = Rad2(a; a-t) - 1. For every N <= N*: the phase-defect word chi* of nota/sec_lb.tex (equivalently draft_unifier.md Lemma F: chi*(x) = P(x - r*j(x)), P the period-2t word 1^t 0^t, defects on the grid t_i = a*i+2t-rho), restricted to [1,N], is a valid colouring of Sigma(a,t,N), its window satisfies m <= 2t, and its gate is exactly 1^{min(t,m)} 0^{max(m-t,0)} — at most 2 runs.

**Proof.** (r >= 1): if 2t | a then v2(a) > v2(t); writing c = (lam-1)a + t, v2((lam-1)a) >= v2(a) > v2(t) forces v2(c) = v2(t) < v2(a), contradicting existence. (m <= 2t): m-1 = floor((N-1-t)/a) <= floor((N*-1-t)/a); N*-1-t = a(t+1) + (2t-rho-1); since 0 <= 2t-rho-1 <= 2t-2 <= 2a-4 < 2a, floor((N*-1-t)/a) <= t+2, i.e. m <= t+3; if 2t-rho-1 < a then m <= t+2 <= 2t (t >= 2); if 2t-rho-1 >= a then a <= 2t-2, which together with a > t forces t >= 3, and m <= t+3 <= 2t. (Validity): Sigma(a,t,N) is the small-c system for x1+a*x2-x3 = a-t on [1,N] (Lemma REF); chi* has no monochromatic solution of that equation on [1,N*] by the PROVEN Theorem lb of nota/sec_lb.tex (= draft_unifier Theorem LB, complete and unconditional), and restriction to [1,N] removes constraints only. ## Theorem SA (single-arm completeness — CCL proven for all N <= lam-1)

**Statement.** Let (a,c) be a non-degenerate middle-band pair with Rad2 existing, and let N <= lam-1 (this is exactly the single-arm regime; for N >= lam either m <= 3 or the window is V-shaped, see Lemma V0). If Psi(G,N) is satisfiable for some G in {0,1}^W, then Psi(G*,N) is satisfiable for some G* in F with at most 2 runs. Explicitly, with Delta = lam-1-N, delta = t+a*Delta: (a) if N <= (a+1)delta: constant gate (Lemma A + REF); (b) if Delta >= 1 and N > (a+1)delta: three-block gate 0^{(m-p)^+} 1^{min(p,m)} (Lemma C + REF, applicable by Lemma BAND); (c) if Delta = 0, t >= 2, (a+1)t < N <= N*: reversed chi* gate 0^{(m-t)^+} 1^{min(t,m)} (Lemma B + REF); (d) if Delta = 0, t >= 2, N > N*: no gate is satisfiable (vacuous); (e) if Delta = 0, t = 1: N <= a+1 is case (a); for a+2 <= N <= 2a+1, m = 2 and every gate lies in F; for N >= 2a+2 no gate is satisfiable (vacuous). Cases (a)-(c) and the m=2 branch of (e) are witness-constructive and do not even use the satisfiability hypothesis.

**Proof.** Coverage: for Delta >= 1, (a) and (b) cover all N by Lemma BAND (N <= (a+3)delta always in-band; window empty is trivial). For Delta = 0, delta = t: (a) covers N <= (a+1)t; since N* = (a+3)t+a-rho > (a+1)t, (c) covers the rest up to N*; beyond N* is (d). For t = 1: m = floor((N-2)/a)+1, so N <= 2a+1 gives m <= 2 (every word of length <= 3 has <= 3 runs, so any satisfiable gate is itself in F — the hypothesis provides one). Witnesses: Lemmas A/B/C produce valid colourings of Sigma(a,delta,N) with gates of at most 2 runs; Lemma REF transfers them to valid colourings of [1,N] for x1+a*x2-x3=c with reversed (still <=2-run) gates, which are then models of Psi(G*,N). Vacuity: (d) by Lemma REF the instance is equivalent to the small-c system for c' = a-t on [1,N] with N >= N*+1 = Rad2(a;a-t); the PROVEN upper bounds (nota/sec_ub1.tex thm:ub1-unified for r <= t, nota/sec_ub2.tex thm:ub2 for r > t) say [1, Rad2(a;a-t)] has no good colouring, and by restriction-monotonicity neither does any longer interval... precisely: [1,N] uncolourable for every N >= Rad2(a;a-t) since a good colouring of [1,N] would restrict to one of [1,Rad2(a;a-t)]. Hence no gate is satisfiable and CCL holds vacuously. (e) same argument with Rad2(a;a-1) = 2a+2 (Dwivedi-Tripathi Theorem 1, published). End-to-end machine verification (a <= 11, full band, every N <= lam-1 with nonempty window): 47,999 instances classified into exactly one case (A: 43,107; B: 427; C: 906; trivial m<=2: 35; vacuous-unsat: 3,524); every constructed witness re-verified against the RAW equation by brute force with gate of <= 2 runs; every claimed-unsat instance confirmed UNSAT by an independent SAT solver. ## Lemma V0 (trichotomy of windows)

**Statement.** For every non-degenerate band pair and every N: either (i) N <= lam-1 and the window is single-arm (Theorem SA applies); or (ii) N >= lam and m <= 3, where every gate is in F (CCL trivial); or (iii) N >= lam, m >= 4, and then BOTH valley owners lam-1 and lam belong to W (the window is genuinely V-shaped).

**Proof.** For N <= lam-1 all owners are left-arm (Lemma W1). Let N >= lam and m >= 4; note lam-1 >= 1 and lam <= N so both valley positions lie in [1,N]. If owner lam-1 were inactive then t > N-1, i.e. N <= t <= a-1; every left owner k <= lam-1 has d_k >= t > N-1 (inactive) and every right owner k >= lam has d_k = mu + a(k-lam) <= N-1 <= t-1 < a, forcing k = lam; so m <= 1, contradiction. If owner lam were inactive then mu > N-1, i.e. N <= mu <= a-1; right owners need d = mu+a(k-lam) <= N-1 < mu, impossible; left owners need t+a*i <= N-1 <= mu-1 < a, forcing i = 0; so m <= 1, contradiction. Hence both valley owners are active.

## Lemma RAIL (rigid rails under a split valley — V-shaped structure)

**Statement.** Let chi be a valid colouring of [1,N] for a band pair with both valley owners active, and suppose chi(lam-1) = g and chi(lam) = 1-g (split valley). Then: (i) chi(y) = g for EVERY y in [1,N] with y ≡ lam-1 (mod a); (ii) chi(y) = 1-g for every y in [t+1, N] with y ≡ lam-1+t (mod a). In particular chi is completely determined on two of the a residue classes mod a. (Corollary: if t = 2 and lam+1 <= N, then chi(lam+1) = 1-g; symmetric statements hold for mu = 2 via the colour/side swap.)

**Proof.** The class of g avoids the distance t (owner lam-1 has colour g) and the class of 1-g avoids mu = a-t (owner lam has colour 1-g). Upward induction: if chi(x) = g and x+a <= N, then x+t <= N, and the pair (x, x+t) at distance t forces chi(x+t) = 1-g; then the pair (x+t, x+t+mu) = (x+t, x+a) at distance mu forces chi(x+a) = g. Downward: if chi(x) = g and x-a >= 1, then x-t = (x-a)+mu >= 1, the pair (x-t, x) forces chi(x-t) = 1-g, and the pair (x-a, x-t) at distance mu forces chi(x-a) = g. Starting from chi(lam-1) = g and iterating in both directions yields (i) for the whole residue class of lam-1 inside [1,N]. For (ii): any y ≡ lam-1+t (mod a) with t+1 <= y <= N has y-t in [1,N] and y-t ≡ lam-1, so chi(y-t) = g by (i) and the distance-t constraint forces chi(y) = 1-g. Machine check: all 908 valley-split valid colourings arising from m <= 3 windows (a <= 9, N in [lam, lam+5], all gates) satisfy both rails exactly; zero violations. (For m >= 4 no valley-split valid colouring was found at all — see the gap section.)

## Lemma V-IMPL (valley-equality + arm-monotonicity imply the F-form)

**Statement.** Let the window be V-shaped (Lemma V0(iii)), with left owners k_lo..lam-1 and right owners lam..k_hi. Suppose a valid colouring chi satisfies: (VAL) chi(lam-1) = chi(lam); and (MONO) on each arm, reading owner colours from the valley outward, the colour changes at most once. Then the gate of chi is of the form b^x g^y b^z (x,z >= 0, y >= 2, b = 1-g), i.e. its g-positions form a contiguous central block containing the valley pair; in particular the gate has at most 3 runs and lies in F. Consequently: if every satisfiable V-shaped instance admits at least one valid colouring satisfying (VAL) and (MONO), then CCL holds for all V-shaped windows — and combined with Theorem SA and Lemma V0, CCL holds in full.

**Proof.** Let g = chi(lam-1) = chi(lam). By (MONO) the left-arm word read outward from the valley is g^i b^(L-i) for some i >= 1, and the right-arm word read outward is g^j b^(R-j) for some j >= 1. The gate read left-to-right is the reversal of the left word followed by the right word: b^(L-i) g^i g^j b^(R-j) = b^(L-i) g^(i+j) b^(R-j), of the stated form with y = i+j >= 2 covering both valley owners. A word of this form has at most 3 maximal runs, hence lies in F (Lemma F of draft_L3-CCL.md). The final implication is immediate since the gate of a valid colouring is a satisfiable gate.
