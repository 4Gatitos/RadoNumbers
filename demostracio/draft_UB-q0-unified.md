## Summary

The upper bound of Dwivedi–Tripathi Conjecture 1 is proved by extending the existing regime-(r<=d) proof from q>=1 to q>=0. UNIFIED THEOREM: for every d>=2 and every a=2dq+r with q>=0, 1<=r<=d and v2(a)<=v2(a-d), every 2-coloring of [1,N], N=(a+3)d+a-(r-1)=(d+1)a+3d+1-r, contains a monochromatic solution of x1+a*x2-x3=a-d; hence Rad2(a;a-d)<=N. For q=0 (equivalently c=a-d<=0) one has r=a and N=(a+3)(a-c)+1, so together with the citable lower bound Rad2(a;c)>=(a+3)(a-c)+1 (Dwivedi–Tripathi, Integers 20 (2020) #A36, Theorem 2, valid for all c<a) this yields Rad2(a;c)=(a+3)(a-c)+1 for ALL c<=0 with v2(a)<=v2(c) and a-c>=2; the single excluded pair (a,c)=(1,0) (the only c<=0 case with d=1) is classical: Rad2(1;0)=5=(1+3)(1)+1 (Schur-type equation x1+x2=x3; also Dwivedi–Tripathi Theorem 4 with a|c). Conjecture 1 is therefore proved in full. Exactly two steps of the q>=1 proof used a>=2d+1 or N-E>=4d+1 (both false at q=0) and both were repaired inside Lemma 7 (endgame Case A): (i) the far-zone range E+2d<=N now follows from N-E=2d(q+1)+1>=2d+1>2d, valid for q>=0; (ii) the window-reach inequality a+2d<=X* now follows from X*>=2a+2d+1-r and a+1-r>=1, where r<=a holds because a=2dq+r with q>=0 (this replaces the old argument via a+1-r>=a+1-d>0 from a>=2d+1, and is sharp at q=0 where r=a gives slack exactly 1). Every other lemma survives verbatim after re-deriving its range conditions with q>=0: the pairing room condition N-E-d=2dq+d+1>=d+1>d holds at q=0 with slack exactly 1, the far-zone case split threshold u>=2dq+d+2>d+1 still guarantees u-d>=1, and Case B is entirely q-free. ## Lemma 0 (Setup, normalization, basic inequalities; q >= 0)

**Statement.** Let d >= 2 and let a >= 1 be an integer. Write a = 2dq + r with q >= 0 and r in [1, 2d] (this decomposition is unique), and assume the regime condition r <= d. Set c := a - d, N := (a+3)d + a - (r-1), E := d(a+1). Then: (i) r <= a, with equality if and only if q = 0; moreover q = 0 iff a <= d iff c <= 0, and q >= 1 iff a >= 2d+1 iff c > 0. (ii) N = (d+1)a + 3d + 1 - r and N - E = a + 2d + 1 - r = 2d(q+1) + 1 >= 2d + 1; consequently E + 2d <= N - 1 and, for q = 0, N = (a+3)d + 1 = (a+3)(a-c) + 1. (iii) The solutions of x1 + a*x2 - x3 = a - d in [1,N]^3 are exactly the triples T(x,k) := (x, k, x + a(k-1) + d) with x >= 1, k in [1,N], and x + a(k-1) + d <= N (values may repeat). (iv) A coloring chi of [1,N] is valid (has no monochromatic solution triple) iff its color swap 1-chi is valid; hence in any proof by contradiction we may normalize chi(1) = 0.

**Proof.** (i) From a = 2dq + r with q >= 0 and r >= 1 we get a >= r, with a = r iff 2dq = 0 iff q = 0. If q = 0 then a = r <= d (regime), so c = a - d <= 0. If q >= 1 then a = 2dq + r >= 2d + 1 > d, so c = a - d >= d + 1 > 0. This proves both equivalences (a <= d iff q = 0 iff c <= 0, and their negations). [REPAIR vs the q>=1 draft: Lemma 0(i) there asserted q >= 1 and a >= 2d+1 from the hypothesis 0 < c; here q = 0 is admitted and the inequality r <= a replaces a >= 2d+1 as the workhorse fact.] (ii) Since r <= d, min(r-1, d-1) = r-1, so N = (a+3)d + a - (r-1) = ad + 3d + a + 1 - r = (d+1)a + 3d + 1 - r. Then N - E = (d+1)a + 3d + 1 - r - d(a+1) = a + 2d + 1 - r, and substituting a = 2dq + r gives a + 2d + 1 - r = 2dq + 2d + 1 = 2d(q+1) + 1. Since q >= 0, N - E >= 2d + 1, i.e., E + 2d <= N - 1. For q = 0: r = a, so N = (d+1)a + 3d + 1 - a = da + 3d + 1 = (a+3)d + 1, and with d = a - c this is (a+3)(a-c) + 1. (iii) x1 + a*x2 - x3 = a - d is equivalent to x3 = x1 + a(x2 - 1) + d. Given x2 = k >= 1, we have x3 = x + a(k-1) + d >= x + d > x >= 1, so x3 >= 1 and x1 <= x3 - d <= N - d are automatic; the binding constraints are exactly x >= 1, k in [1,N], x3 <= N. (iv) The condition 'chi(x1) = chi(x2) = chi(x3) for some solution triple' is invariant under the swap 0 <-> 1 of colors; hence chi is valid iff 1-chi is, and any putative valid coloring may be normalized to chi(1) = 0. QED.

## Lemma 1 (2-adic lemma; conventions covering q = 0, including a - d <= 0)

**Statement.** Adopt the conventions v2(0) := +infinity and v2(-n) := v2(n) for n > 0. Let e := v2(d). In the setting of Lemma 0, if v2(a) <= v2(a - d) (the existence condition for Rad2(a; a-d), noting a - d = c may be zero or negative when q = 0), then v2(r) <= e, and consequently gamma := gcd(r, 2d) divides d.

**Proof.** Write d = 2^e * m with m odd. Suppose for contradiction v2(r) >= e + 1. Then r != d, since v2(d) = e < e + 1; write r = 2^{e+1} r'. FIRST: v2(a) >= e+1. Indeed a = 2dq + r; the term 2dq is either 0 (when q = 0, with v2(0) = infinity >= e+1) or has v2(2dq) = e + 1 + v2(q) >= e + 1, and v2(r) >= e+1, so 2^{e+1} divides both terms, giving v2(a) >= e+1. (For q = 0 this is simply v2(a) = v2(r) >= e+1.) SECOND: v2(a - d) = e exactly. Indeed r - d = 2^{e+1} r' - 2^e m = 2^e (2r' - m) with 2r' - m odd (even minus odd) and nonzero, so v2(r - d) = e exactly — this holds whether r - d is positive or negative, by the convention v2(-n) = v2(n). Then a - d = 2dq + (r - d) is the sum of a term of 2-adic valuation >= e+1 (or the zero term, when q = 0) and a term of valuation exactly e; by the ultrametric inequality with strict disparity, v2(a - d) = e exactly — in particular a - d != 0, so v2(a-d) is an ordinary integer. (For q = 0 this is simply a - d = r - d, of valuation e.) Combining, v2(a) >= e + 1 > e = v2(a - d), contradicting the existence condition. Hence v2(r) <= e. [Boundary remark: if r = d and q = 0, i.e., a = d and c = 0, then a - d = 0 and v2(a-d) = infinity, so the existence condition holds trivially; the conclusion also holds trivially since v2(r) = v2(d) = e <= e. The contradiction branch above never encounters r = d.] CONSEQUENCE: let gamma = gcd(r, 2d). Its 2-part: v2(gamma) = min(v2(r), v2(2d)) = v2(r) <= e, since v2(r) <= e < e + 1 = v2(2d). Its odd part divides the odd part of 2d, which is m. A positive integer divides d = 2^e m iff its 2-adic valuation is <= e and its odd part divides m; gamma satisfies both, so gamma | d. QED. (Machine-checked for all 2211 admissible q = 0 pairs with d <= 80 and all 1492 admissible q >= 1 pairs with a <= 120.)

## Lemma 2 (First forced color)

**Statement.** Let chi be a valid coloring of [1,N] with chi(1) = 0, in the setting of Lemma 0 (q >= 0). Then chi(d+1) = 1.

**Proof.** The triple T(1,1) = (1, 1, 1+d) is a solution triple by Lemma 0(iii), and it lies in [1,N]^3 because 1 + d <= N: indeed N = E + 2d(q+1) + 1 >= E + 2d + 1 = d(a+1) + 2d + 1 >= d + 2d + 1 > d + 1, using only a >= 1 and q >= 0. [REPAIR vs the q>=1 draft: the old justification invoked N >= E + 4d + 1, which fails at q = 0; the bound N >= E + 2d + 1 suffices and holds for all q >= 0.] Its first two entries have color chi(1) = 0; if chi(d+1) = 0 the triple would be monochromatic, contradicting validity. Hence chi(d+1) = 1. QED.

## Lemma 3 (Transport Lemma; q-free)

**Statement.** Let chi be a valid coloring of [1,N] with chi(1) = 0. Let k0 in [1,N] satisfy chi(k0) = 1 and put D := (k0 - 1)a + d. Then for every y in [1, N - D - d] with chi(y) = 1: chi(y + d) = 0 and chi(y + 2d) = 1.

**Proof.** This lemma and its proof are entirely q-free; all four triples below are genuine solution triples inside [1,N]^3, with range checks in brackets. Step 1: T(y, k0) = (y, k0, y + D) [x1 = y >= 1; x2 = k0 in [1,N]; x3 = y + D <= N - d <= N by the hypothesis y <= N - D - d]. Since chi(y) = chi(k0) = 1, validity forces chi(y + D) = 0. Step 2: T(y + D, 1) = (y + D, 1, y + D + d) [x3 = y + D + d <= N by hypothesis]. Since chi(y + D) = chi(1) = 0, validity forces chi(y + D + d) = 1. Step 3: T(y + d, k0) = (y + d, k0, y + d + D) [x1 = y + d >= 1; x3 = y + D + d <= N]. Since chi(y + d + D) = 1 = chi(k0), validity forces chi(y + d) = 0. Step 4: T(y + d, 1) = (y + d, 1, y + 2d) [x3 = y + 2d <= y + D + d <= N, using D = (k0 - 1)a + d >= d]. Since chi(y + d) = chi(1) = 0, validity forces chi(y + 2d) = 1. (Repeated values among x1, x2, x3 are permitted by the problem statement, so no genericity conditions are needed.) QED.

## Lemma 4 (Pairing Lemma; definition of phi and p; room condition re-derived for q >= 0)

**Statement.** Let chi be valid on [1,N] with chi(1) = 0, in the setting of Lemma 0 (q >= 0). Then for every s in [1, d]: chi(s + d) = 1 - chi(s). Define phi(s) := chi(s) for s in [1,d] (so phi(1) = 0), and define p : Z -> {0,1} by p(u) = phi(s) if u == s (mod 2d) with s in [1,d], and p(u) = 1 - phi(s) if u == s + d (mod 2d) with s in [1,d]. Then p is 2d-periodic, antisymmetric (p(u + d) = 1 - p(u) for all u), p(1) = 0, and chi(u) = p(u) for all u in [1, 2d].

**Proof.** Fix s in [1,d]. NOT BOTH 0: the triple T(s, 1) = (s, 1, s + d) lies in [1,N]^3 since s + d <= 2d < N (Lemma 0(ii): N >= E + 2d + 1 > 2d); if chi(s) = chi(s+d) = 0, then together with chi(1) = 0 it is monochromatic — contradiction. NOT BOTH 1: apply the Transport Lemma (Lemma 3) with k0 = d + 1 (legitimate: chi(d+1) = 1 by Lemma 2, and d+1 <= N), so D = D_{d+1} = da + d = E, at y = s. The room condition y = s <= N - E - d must be checked for q >= 0: N - E - d = 2d(q+1) + 1 - d = 2dq + d + 1 >= d + 1 > d >= s, valid for every q >= 0 (at q = 0 the room is d + 1, exceeding the largest seed s = d by exactly one unit — this is the tightest point of the pairing step and explains why the bound drops at N - 1). If chi(s) = 1, Lemma 3 yields chi(s + d) = 0. Hence exactly one of chi(s), chi(s+d) equals 1, i.e., chi(s+d) = 1 - chi(s). PROPERTIES OF p: 2d-periodicity holds by construction. Antisymmetry: if u == s (mod 2d) with s in [1,d], then u + d == s + d and p(u+d) = 1 - phi(s) = 1 - p(u); if u == s + d (mod 2d), then u + d == s (mod 2d) and p(u+d) = phi(s) = 1 - (1 - phi(s)) = 1 - p(u). Finally p(1) = phi(1) = chi(1) = 0, and chi(u) = p(u) on [1,2d] is exactly the definition of phi together with the pairing just proved. QED.

## Lemma 5 (Window Lemma; q >= 0)

**Statement.** Let chi be valid on [1,N], chi(1) = 0, with phi, p as in Lemma 4. Define s* := min{ s in [2, d+1] : p(s) = 1 } (well-defined since p(d+1) = 1 - phi(1) = 1), D* := (s* - 1)a + d, and X* := N - D*. Then chi(u) = p(u) for every u in [1, X*]. Moreover X* >= N - E = 2d(q+1) + 1 >= 2d + 1, since D* <= D_{d+1} = E. [The q>=1 draft asserted X* >= 4d+1 here; for q = 0 only X* >= 2d+1 holds, and nothing more is needed anywhere in the proof.]

**Proof.** First, chi(s*) = 1: if s* <= d this is phi(s*) = p(s*) = 1 by the definition of phi; if s* = d+1 it is Lemma 2. Also s* <= d + 1 <= N. The Transport Lemma therefore applies with k0 = s*, gap D = D*, and room condition y <= N - D* - d = X* - d. Fix s in [1,d]; let y0 := s if phi(s) = 1, and y0 := s + d otherwise. By Lemma 4, chi(y0) = 1 and y0 in [1, 2d]. CLAIM (induction on j >= 0): (a) if y0 + 2dj <= X* + d then chi(y0 + 2dj) = 1; (b) if y0 + 2dj + d <= X* then chi(y0 + 2dj + d) = 0. Base j = 0: (a) is the seed chi(y0) = 1 (note y0 <= 2d <= X* + d since X* >= 2d + 1 > d). (b): if y0 + d <= X*, then y0 <= X* - d, so transport at y = y0 gives chi(y0 + d) = 0. Inductive step: assume (a) at j, i.e., chi(y_j) = 1 for y_j := y0 + 2dj <= X* + d (if the hypothesis of (a) at j fails, the hypotheses of (a) at all larger indices and of (b) at all indices >= j fail too, and there is nothing to prove). For (b) at j: if y_j + d <= X*, then y_j <= X* - d, and transport at y_j gives chi(y_j + d) = 0. For (a) at j+1: if y_j + 2d <= X* + d, then y_j <= X* - d, and transport at y_j gives chi(y_j + 2d) = 1. This proves the claim. COVERAGE: let u in [1, X*]. Its residue mod 2d equals that of y0 or of y0 + d for the appropriate s in [1,d], since {y0, y0 + d} == {s, s + d} (mod 2d). Case u == y0 (mod 2d): since y0 in [1,2d], the least positive representative of its class is y0 itself, so u = y0 + 2dj for some j >= 0 with u <= X* < X* + d; by (a), chi(u) = 1 = p(u), since p(u) = p(y0) = chi(y0) = 1 by construction. Case u == y0 + d (mod 2d): if y0 = s (the case phi(s) = 1), the least positive representative is s + d = y0 + d, so u = y0 + d + 2dj with u <= X*; by (b), chi(u) = 0 = p(u) (p(u) = p(s+d) = 1 - phi(s) = 0). If y0 = s + d (the case phi(s) = 0), the least positive representative is s: either u = s, a seed with chi(s) = phi(s) = 0 = p(u) (Lemma 4), or u = s + 2dj = y0 + d + 2d(j-1) for some j >= 1 with u <= X*, and (b) gives chi(u) = 0 = p(u). In all cases chi(u) = p(u). Every step above is independent of q; the only q-sensitive quantity is the guaranteed window length X* >= 2d(q+1) + 1, which at q = 0 equals 2d + 1 and in particular covers [1, 2d] and the cell 2d + 1. QED.

## Lemma 6 (Far-Zone Lemma; q >= 0)

**Statement.** Let chi be valid on [1,N], chi(1) = 0, with p, s*, X* as above, and E = d(a+1). Then chi(z) = 1 - p(z - E) for every z in [E+1, N].

**Proof.** Let u := z - E in [1, N - E], where N - E = 2d(q+1) + 1 (Lemma 0(ii)). Since u <= N - E <= X* (Lemma 5), the value chi(u) = p(u) is known. CASE 1: p(u) = 1. Then chi(u) = 1, and the triple T(u, d+1) = (u, d+1, u + E) = (u, d+1, z) lies in [1,N]^3 (x2 = d+1 <= N; x3 = z <= N by hypothesis). Since chi(u) = chi(d+1) = 1 (Lemma 2), validity forces chi(z) = 0 = 1 - p(u). CASE 2: p(u) = 0. Sub-case u <= N - E - d: then u + d <= N - E and p(u + d) = 1 by antisymmetry, so Case 1 applied to u + d (legitimate: u + d <= N - E, so z + d = E + (u+d) <= N) gives chi(z + d) = chi(E + u + d) = 0. The triple T(z, 1) = (z, 1, z + d) lies in [1,N]^3 (x3 = z + d <= N as just checked). If chi(z) = 0, then chi(z) = chi(1) = chi(z + d) = 0 would be monochromatic — contradiction; hence chi(z) = 1 = 1 - p(u). Sub-case u > N - E - d, i.e., u >= N - E - d + 1 = 2dq + d + 2: since 2dq + d + 2 >= d + 2 > d + 1 for every q >= 0, we have u - d >= 2 >= 1, and p(u - d) = 1 - p(u) = 1 by antisymmetry. Moreover u - d <= N - E - d... more simply u - d <= N - E, so Case 1 applied to u - d gives chi(z - d) = chi(E + (u - d)) = 0; and z - d = E + (u - d) >= E + 1, so z - d lies in [E+1, N] and the triple T(z - d, 1) = (z - d, 1, z) lies in [1,N]^3 (x3 = z <= N). If chi(z) = 0, then chi(z - d) = chi(1) = chi(z) = 0 would be monochromatic — contradiction; hence chi(z) = 1 = 1 - p(u). [q = 0 check of the sub-case split: the threshold is u >= d + 2, and u ranges in [1, 2d+1]; both sub-cases are nonempty and exhaust the range; at u = 2d + 1 == 1 (mod 2d) we have p(u) = p(1) = 0 and u - d = d + 1 with p(d+1) = 1, as used.] QED. (Logical remark, relevant to the assembly: when X* >= E + 1 the window and far-zone ranges overlap, and on the overlap the two forced formulas may disagree; both lemmas are implications from the validity of chi, so a disagreement simply refutes validity outright — the endgame below provides a uniform contradiction in all cases regardless. The machine replay observed such early refutations frequently; they only shorten the argument.)

## Lemma 7 (Endgame Case A: phi not identically zero; the two q = 0 repairs)

**Statement.** Assume the setting of Lemmas 0-6, with the existence condition v2(a) <= v2(a-d), and suppose s* <= d (i.e., phi(s) = 1 for some s in [2,d]; this requires d >= 2). Set epsilon := phi(d). Then there exists u in [1, 2d] with p(u + r) = epsilon and p(u) = 1 - epsilon, and for any such u the solution triple (a + u, d, E + u) is monochromatic of color epsilon. This contradicts validity, so no such chi exists.

**Proof.** EXISTENCE OF u (q-free). View p as a function on Z/2d (residues represented by [1, 2d]) and let P := { v in Z/2d : p(v) = epsilon }. Suppose no u as stated exists; then for every u in Z/2d, p(u + r) = epsilon implies p(u) = epsilon, i.e., P - r is a subset of P, where P - r := { v : v + r in P }. Translation by r is a bijection of the finite group Z/2d, so |P - r| = |P|, forcing P - r = P. Iterating, P - jr = P for all j >= 0, so P is invariant under translation by every element of the subgroup <r> = gamma(Z/2d), where gamma = gcd(r, 2d). By Lemma 1, gamma | d, so d is a member of <r> and P - d = P. Now d is in P, because p(d) = phi(d) = epsilon. Hence 2d = d + d is in P as well, i.e., p(2d) = epsilon. But antisymmetry (Lemma 4) gives p(2d) = 1 - p(d) = 1 - epsilon — a contradiction (0 != 1). The required u exists. THE TRIPLE. Fix such a u in [1, 2d] and consider T(a + u, d) = (a + u, d, (a + u) + (d - 1)a + d) = (a + u, d, E + u), a genuine solution triple by Lemma 0(iii). Ranges: x1 = a + u >= a + 1 >= 2 >= 1; x2 = d in [1, N]; x3 = E + u <= E + 2d <= N. [REPAIR 1 vs the q>=1 draft: the old range check for x3 invoked N - E >= 4d + 1, false at q = 0; the correct and sufficient bound is N - E = 2d(q+1) + 1 >= 2d + 1 > 2d for every q >= 0 (Lemma 0(ii)).] Forced colors: (1) x1 = a + u <= a + 2d <= X*. Indeed s* <= d gives D* = (s*-1)a + d <= (d-1)a + d, so X* = N - D* >= N - (d-1)a - d = (d+1)a + 3d + 1 - r - (d-1)a - d = 2a + 2d + 1 - r, and (2a + 2d + 1 - r) - (a + 2d) = a + 1 - r >= 1 > 0, because r <= a (Lemma 0(i), valid for all q >= 0, with equality r = a exactly at q = 0 — where the slack is exactly one unit). [REPAIR 2 vs the q>=1 draft: the old justification was a + 1 - r >= a + 1 - d > 0 via a >= 2d + 1, which is false at q = 0 (there a <= d); the inequality r <= a from the decomposition a = 2dq + r, q >= 0, replaces it and is sharp.] Hence by the Window Lemma chi(a + u) = p(a + u) = p(u + r), because a - r = 2dq == 0 (mod 2d) — an identity valid for q = 0 as well (there a = r exactly). By the choice of u, chi(a + u) = epsilon. (2) x2 = d lies in [1, d], so chi(d) = phi(d) = epsilon by definition. (3) x3 = E + u lies in [E+1, E+2d], a subset of [E+1, N] by Repair 1, so by the Far-Zone Lemma chi(E + u) = 1 - p(u) = 1 - (1 - epsilon) = epsilon. Thus all three entries have color epsilon: the triple is monochromatic, contradicting the validity of chi. QED. (Machine-checked: the u-existence and the full triple instantiation — ranges, window reach, forced colors — verified for every phi-pattern on all tested instances, exhaustively for d <= 13 and by structured plus random sampling up to d = 206, zero failures.)

## Lemma 8 (Endgame Case B: phi identically zero; entirely q-free)

**Statement.** Assume the setting of Lemmas 0-6 and suppose s* = d + 1 (i.e., phi(s) = 0 for all s in [1, d]). Then the solution triple (d + 1 - r, 2, a + 2d + 1 - r) is monochromatic of color 0, contradicting validity; so no such chi exists.

**Proof.** Here D* = D_{d+1} = E, so X* = N - E = a + 2d + 1 - r = 2d(q+1) + 1 (Lemma 0(ii)). Consider T(d + 1 - r, 2) = (d + 1 - r, 2, (d + 1 - r) + a + d) = (d + 1 - r, 2, a + 2d + 1 - r), a genuine solution triple by Lemma 0(iii). Ranges: x1 = d + 1 - r >= 1 since r <= d (regime hypothesis; at q = 0, r = a and x1 = d + 1 - a >= 1 since a <= d); x2 = 2 in [1, N]; x3 = a + 2d + 1 - r = X* <= N since D* >= 0. Forced colors: (1) chi(d + 1 - r) = phi(d + 1 - r) = 0, since d + 1 - r is in [1, d] (as 1 <= r <= d) and phi is identically 0. (2) chi(2) = phi(2) = 0, using d >= 2 so that 2 is in [1, d]. (3) x3 = X* lies in the window [1, X*], and X* - 1 = a + 2d - r = 2dq + 2d = 2d(q+1) == 0 (mod 2d) — an identity valid verbatim for q = 0, where X* - 1 = 2d — so X* == 1 (mod 2d) and, by the Window Lemma and 2d-periodicity of p, chi(X*) = p(X*) = p(1) = phi(1) = 0. All three entries have color 0: the triple is monochromatic, contradicting validity. Every inequality in this proof is independent of q. QED.

## Theorem (unified upper bound, regime r <= d, all q >= 0)

**Statement.** Let d >= 2 and let a >= 1 satisfy v2(a) <= v2(a - d) (conventions: v2(0) = infinity, v2(-n) = v2(n)); write a = 2dq + r with q >= 0, r in [1, 2d], and assume r <= d. Set N = (a+3)d + a - (r-1) = (d+1)a + 3d + 1 - r. Then every 2-coloring of [1, N] contains a monochromatic solution of x1 + a*x2 - x3 = a - d. Consequently Rad2(a; a - d) <= (a+3)d + a - (r-1) = (a+3)d + a - min(r-1, d-1).

**Proof.** Suppose some 2-coloring chi of [1, N] has no monochromatic solution. By Lemma 0(iv) normalize chi(1) = 0. Lemma 2 gives chi(d+1) = 1; Lemma 4 defines phi on [1,d] and its 2d-periodic antisymmetric extension p, with chi = p on [1, 2d]. The index s* = min{ s in [2, d+1] : p(s) = 1 } is well defined since p(d+1) = 1 - phi(1) = 1 (Lemma 5). If s* <= d, Lemma 7 — whose two q-sensitive steps were repaired to hold for all q >= 0 — produces a monochromatic solution triple (a + u, d, E + u): contradiction. If s* = d + 1, then p(s) = 0 for all s in [2, d] and phi(1) = 0, so phi is identically 0 on [1, d]; Lemma 8 produces the monochromatic triple (d + 1 - r, 2, a + 2d + 1 - r): contradiction. Hence no valid chi exists and Rad2(a; a-d) <= N; since r <= d gives min(r-1, d-1) = r - 1, this is the stated bound. The chain of lemmas used only: N - E = 2d(q+1) + 1 >= 2d + 1 (Lemmas 2, 4, 6, 7), r <= a (Lemma 7), r <= d (Lemma 8), d >= 2 (Lemma 8 and the nonvacuity of Case A), and gcd(r, 2d) | d from the existence condition (Lemmas 1, 7) — all established for every q >= 0. QED.

## Corollary (Dwivedi-Tripathi Conjecture 1, in full)

**Statement.** For all integers a >= 1 and c <= 0 with v2(a) <= v2(c): Rad2(a; c) = (a+3)(a-c) + 1.

**Proof.** Set d := a - c, so d >= a >= 1 (as c <= 0) and c = a - d. CASE d = 1: this forces a = 1 and c = 0, the equation x1 + x2 - x3 = 0, i.e., the Schur-type equation x1 + x2 = x3. Its 2-color Rado number is the classical value 5 (Beutelspacher-Brestovansky; also Dwivedi-Tripathi Theorem 4, which proves equality Rad2(a;c) = (a+3)(a-c)+1 whenever a | c and c <= 0, applies since 1 | 0), and (1+3)(1-0) + 1 = 5. Machine-recomputed: Rad2(1;0) = 5. CASE d >= 2: since 1 <= a <= d < 2d, the unique decomposition a = 2dq + r with r in [1, 2d] has q = 0 and r = a, and the regime condition r <= d holds because a <= d. The existence condition v2(a) <= v2(c) = v2(a - d) is exactly the theorem's hypothesis (for c = 0, v2(0) = infinity and it holds trivially; note d = a there, covered with r = a = d). The unified Theorem gives Rad2(a; a - d) <= (a+3)d + a - (a - 1) = (a+3)d + 1 = (a+3)(a-c) + 1. For the reverse inequality, Dwivedi-Tripathi (Integers 20 (2020) #A36), Theorem 2, states Rad2(a; c) >= (a+3)(a-c) + 1 for ALL c < a; here c <= 0 < 1 <= a, so it applies. Hence Rad2(a; c) = (a+3)(a-c) + 1. Both previously open windows of Conjecture 1 are covered: (i) a even with a not dividing c (e.g., (a,c) = (6,-2), (6,-4), (10,-2), (12,-4), all machine-confirmed), since the argument never uses parity of a or divisibility of c; and (ii) a odd with t = c mod a in [1, a-3] and -a(a-t-2)/2 < c < 0 (e.g., (5,-4), (7,-6), (9,-7), (17,-4), machine-confirmed). QED. (Remark on tightness: at length N - 1 the derivations lose exactly one unit of room — at q = 0 the pairing room condition s <= N - E - d and the Case-A window slack a + 1 - r are both tight with margin exactly 1 — and the machine-verified extremal colorings at N - 1, which realize phi identically 0, show the bound cannot be improved; SAT at N - 1 was re-confirmed for all 20 tested q = 0 instances.)

## Gaps
- No mathematical gaps remain in the upper-bound proof: every lemma was re-derived with q >= 0 and the two (and only two) steps that used a >= 2d+1 or N - E >= 4d+1 — both inside Lemma 7 (Case A range E + 2d <= N, and window-reach slack a + 1 - r) — were repaired with proofs valid for all q >= 0. The following are honest caveats, not gaps in the argument itself:
- The lower-bound half of Conjecture 1 is cited from the literature (Dwivedi-Tripathi, Integers 20 (2020) #A36, Theorem 2, valid for all c < a) rather than re-proven here, exactly as the task prescribes; it was nevertheless corroborated by SAT witnesses at N-1 on all 20 tested q = 0 instances.
- The excluded pair (a,c) = (1,0) (the unique c <= 0 case with d = 1; Case B of the endgame needs d >= 2 for the cell chi(2) = phi(2)) is handled by citation: Dwivedi-Tripathi Theorem 4 (a | c, c <= 0) and the classical Schur-type value Rad2(1;0) = 5, machine-recomputed. Note: contrary to the task statement's parenthetical, (a,c) = (2,0) is NOT excluded — it has d = 2, q = 0, r = 2 and is covered by the unified theorem (machine-confirmed: Rad2(2;0) = 11 = N, UNSAT at 11, SAT at 10).
- Machine corroboration of the per-phi endgame and replay is exhaustive over all 2^(d-1) patterns only for d <= 13 (up to 4096 patterns per instance); for the larger tested instances ((12,16), (9,16), (17,21), (3,203), (3,206), and the d = a boundary cases up to d = 13 exhaustive) it used all structured patterns (identically zero, every single-one position, alternating, dense) plus 300 seeded random patterns. The proof itself is unconditional and does not rely on these checks.
- A logical subtlety recorded for the verification record (not a gap): when X* >= E + 1 the window and far-zone ranges overlap and their forced formulas can disagree on the overlap (frequently observed in the replay, e.g., always for a = 1 with phi == 0); since both lemmas are implications from the assumed validity of chi, any disagreement refutes validity immediately, and the endgame triple provides a uniform contradiction in the remaining cases — the written proof is correcte in either eventuality.
