## Lemma R (reflection duality)

**Statement.** Let a>=1, N>=1, c in Z. Set sigma(x)=N+1-x and c_hat = a(N+1)-c. Then (x1,x2,x3) in [1,N]^3 solves x1+a*x2-x3=c iff (sigma(x1),sigma(x2),sigma(x3)) solves x1+a*x2-x3=c_hat. Consequently chi:[1,N]->{0,1} has no monochromatic E(c)-solution iff chi∘sigma has no monochromatic E(c_hat)-solution; [1,N] is E(c)-colorable iff it is E(c_hat)-colorable. Specialization: for c = lam*a - mu with mu in [1,a-1], t=a-mu, taking N=lam gives c_hat = 2a-t, hence Rad2(a;c) <= lam <=> Rad2(a;2a-t) <= lam, and [1,lam] is E(c)-colorable <=> lam <= Rad2(a;2a-t)-1. (Taking N=lam-1 gives c_hat = a-t, recovering draft_L2-tail Lemma 3: one duality subsumes both reflections.)

**Proof.** sigma is an involution of [1,N]. Compute sigma(x1)+a*sigma(x2)-sigma(x3) = (N+1-x1)+a(N+1-x2)-(N+1-x3) = a(N+1) - (x1+a*x2-x3). This equals c_hat iff x1+a*x2-x3 = c; membership in [1,N]^3 is preserved since sigma maps [1,N] onto itself; repeated entries map to repeated entries. Monochromaticity is preserved because (chi∘sigma)(sigma(x)) = chi(x). Applying this bijection to all solution triples gives the validity equivalence; sigma being an involution makes it two-sided. For the specialization: a(lam+1)-c = a*lam + a - c = mu + a = 2a - t since a*lam - c = mu and mu = a - t. The Rad2 equivalence follows since Rad2(a;c) <= lam iff [1,lam] admits no valid colouring (definition plus monotonicity: a valid colouring of [1,M] restricts to any [1,M']<=M). Note a | c_hat iff a | c (c_hat ≡ -c mod a), and existence transfers by Lemma V. Verified computationally: raw SAT (no window theory) confirms [1,lam] E(c)-colorable == [1,lam] E(2a-t)-colorable on 294 instances, a in 8..14, lam straddling all thresholds, zero mismatches.

## Lemma V (existence transfer)

**Statement.** If v2(a) <= v2(c) and t = c mod a, then v2(a) <= v2(t); consequently v2(a) <= v2(2a-t) and v2(a) <= v2(a-t). In particular, for t=1 this forces a odd, and the hypotheses of Theorems T1/T2 below are implied by the existence of Rad2(a;c). Moreover v2(a)<=v2(d) <=> v2(a)<=v2(a-d) <=> v2(a)<=v2(2a-d), so the project's 2-adic lemma (nota/sec_ub1.tex, Lemma 2adic-a) applies verbatim: with a = 2dq+r, r in [1,2d], one has gamma := gcd(r,2d) | d; in particular r != 2d, i.e. 2d does not divide a.

**Proof.** Let alpha = v2(a) and write c = qa + t. From 2^alpha | a we get 2^alpha | qa, and 2^alpha | c by hypothesis, so 2^alpha | c - qa = t, i.e. v2(t) >= alpha. Then 2^alpha divides 2a and t, hence 2a-t; and divides a and t, hence a-t. Conversely if 2^alpha | a-d (resp. 2a-d) then 2^alpha | d since 2^alpha | a (resp. 2a); so the three conditions v2(a)<=v2(d), v2(a)<=v2(a-d), v2(a)<=v2(2a-d) are equivalent. The cited 2-adic lemma has hypothesis v2(a)<=v2(a-d) and conclusion v2(r)<=v2(d), whence gcd(r,2d) | d; it is proven in full in sec_ub1.tex (verified). If r = 2d then 2d | a and v2(a) >= v2(2d) = v2(d)+1 > v2(d), contradicting v2(a)<=v2(d); so r in [1,2d-1].

## Theorem T1 (bottom-band pin source, t=1)

**Statement.** Let a >= 3 be odd. Every 2-colouring of [1, 2a-2] contains a monochromatic solution of x1 + a*x2 - x3 = 2a-1. Hence Rad2(a; 2a-1) <= 2a-2. (This is sharp: computed values give Rad2(a;2a-1) = 2a-2 for all odd a <= 19, so the t=1 deep-tail threshold lam >= 2a-2 cannot be weakened.)

**Proof.** Solutions in [1,N]^3, N=2a-2, are the triples T(x,k) = (x, k, x + a(k-2) + 1). Suppose chi is a valid colouring; by colour swap assume chi(2)=0. If chi(1)=0, the triple T(1,2)=(1,2,2) has colours (chi(1),chi(2),chi(2))=(0,0,0), monochromatic — contradiction. So chi(1)=1. T(2,2)=(2,2,3) with colours (0,0,chi(3)) forces chi(3)=1. T(a,1)=(a,1,1) with colours (chi(a),1,1) forces chi(a)=0 [a <= N]. Descent claim: chi(a-2k)=0 for k=0,...,(a-3)/2. Induction step, given chi(a-2k)=0 with a-2k >= 5: (i) T(a-2k-1, 2, a-2k), colours (chi(a-2k-1),0,0), forces chi(a-2k-1)=1; (ii) T(2a-2-2k, 1, a-1-2k), colours (chi(2a-2-2k),1,chi(a-1-2k))=(·,1,1), forces chi(2a-2-2k)=0 [x1 = 2a-2-2k <= N, x3 = a-1-2k >= 1]; (iii) T(2a-3-2k, 2, 2a-2-2k), colours (·,0,0), forces chi(2a-3-2k)=1; (iv) T(2a-3-2k, 1, a-2-2k), colours (1,1,·), forces chi(a-2-2k)=0 [a-2-2k >= 3 >= 1]. Since a is odd the cells a-2k are the odd numbers descending from a; the chain reaches chi(3)=0, contradicting chi(3)=1 (for a=3, already chi(a)=chi(3)=0 contradicts directly). All triples and ranges verified mechanically for every odd a < 200 (verify_g2.py, zero failures).

## Lemma P (T2 setup, first colours, pairing)

**Statement.** Fix d >= 2, a >= d+1 with v2(a) <= v2(d); let N = d(a+3)+2, E = d(a+1) (so N-E = 2d+2), and consider E'': x1+a*x2-x3 = 2a-d, whose solutions in [1,N]^3 are the triples T(x,k) = (x,k,x+a(k-2)+d). Let chi be a valid colouring of [1,N] with chi(2)=0 (WLOG by colour swap). Then: (a) chi(d+2)=1; (b) chi(s+d) = 1-chi(s) for every s in [1, d+2]; (c) defining phi := chi|[1,d] and p: Z -> {0,1} as the 2d-periodic antisymmetric extension (p ≡ phi(s) on s+2dZ and ≡ 1-phi(s) on s+d+2dZ, s in [1,d]; so p(u+d)=1-p(u) for all u), one has chi = p on [1, 2d+2]; and p(2)=0.

**Proof.** (a) T(2,2)=(2,2,d+2) has colours (0,0,chi(d+2)); validity forces chi(d+2)=1. (b) Not both 0: T(s,2)=(s,2,s+d) has colours (chi(s),0,chi(s+d)); if chi(s)=chi(s+d)=0 it is monochromatic [s+d <= 2d+2 <= N]. Not both 1: suppose chi(s)=1 with s <= d+2. Step 1: T(s,d+2)=(s,d+2,s+E), colours (1,1,chi(s+E)), forces chi(s+E)=0 [s+E <= N iff s <= 2d+2, true]. Step 2: T(s+E,2)=(s+E,2,s+E+d), colours (0,0,·), forces chi(s+E+d)=1 [s+E+d <= N iff s <= d+2 — this is the tight step, consuming exactly the room N-E=2d+2]. Step 3: T(s+d,d+2)=(s+d,d+2,s+d+E), colours (chi(s+d),1,1), forces chi(s+d)=0. Hence exactly one of chi(s),chi(s+d) is 1. (c) For u in [1,d]: chi(u)=phi(u)=p(u). For u in [d+1,2d+2]: u-d in [1,d+2] and (b) gives chi(u)=1-chi(u-d); iterating once more when u-d > d yields chi(u)=chi(u-2d)=phi(u-2d); in all cases this equals p(u) by antisymmetry and 2d-periodicity. p(2)=phi(2)=chi(2)=0. All triple identities and range inequalities verified mechanically for all admissible (a,d), d<=8, a<=60.

## Lemma W (transport and window, Case A)

**Statement.** In the setting of Lemma P, suppose s** := min{ s in [3, d+1] : p(s)=1 } exists (Case A). Put D** := a(s**-2)+d and X** := N - D**. Then chi(u) = p(u) for every u in [1, X**], and X** >= a+2d+2.

**Proof.** X** >= N - ((d-1)a + d) = d(a+3)+2 - da + a - d = a + 2d + 2 since s** <= d+1. chi(s**) = p(s**) = 1 by Lemma P(c) (s** <= d+1 <= 2d+2). Transport: if chi(y)=1 and y <= X**-d then chi(y+d)=0 and chi(y+2d)=1. Indeed: T(y,s**)=(y,s**,y+D**), colours (1,1,·), forces chi(y+D**)=0 [y+D** <= N-d]; T(y+D**,2), colours (0,0,·), forces chi(y+D**+d)=1 [y+D**+d <= N iff y <= X**-d]; T(y+d,s**)=(y+d,s**,y+d+D**), colours (chi(y+d),1,1), forces chi(y+d)=0; T(y+d,2)=(y+d,2,y+2d), colours (0,0,·), forces chi(y+2d)=1 [y+2d <= y+D**+d <= N since D** >= a+d > d, using s** >= 3]. Window induction: for each s in [1,d] set y0 := s if phi(s)=1, else y0 := s+d; then chi(y0)=1 (Lemma P) and y0 in [1,2d]. By induction on j: (a) if y0+2dj <= X**+d then chi(y0+2dj)=1; (b) if y0+2dj+d <= X** then chi(y0+2dj+d)=0 — the base is the seed, and each step applies transport at y = y0+2dj <= X**-d. Coverage: every u in [1,X**] is congruent mod 2d to y0 or y0+d for the appropriate s; writing u via the least positive representative (y0, or s when y0=s+d) expresses u as a chain cell, and (a)/(b) give chi(u)=p(u) in every case, exactly as in the verified window lemma of sec_ub1.tex with (1, d+1, E) replaced by (2, s**, D**). Range inequalities machine-verified at the extreme y = X**-d for all admissible (a,d), d<=8, a<=60.

## Lemma F (far zone)

**Statement.** In the setting of Lemma P (no case assumption): chi(E+u) = 1 - p(u) for every u in [1, 2d+2].

**Proof.** Case p(u)=1: then chi(u)=1 (u <= 2d+2, Lemma P(c)), and T(u,d+2)=(u,d+2,u+E) has colours (1,1,chi(E+u)), forcing chi(E+u)=0 [E+u <= N iff u <= 2d+2]. Case p(u)=0, u <= d+2: p(u+d)=1 and u+d <= 2d+2, so the previous case gives chi(E+u+d)=0; T(E+u,2)=(E+u,2,E+u+d) has colours (chi(E+u),0,0), forcing chi(E+u)=1 [E+u+d <= N iff u <= d+2]. Case p(u)=0, u in [d+3,2d+2]: p(u-d)=1 with u-d in [3,d+2], so chi(E+u-d)=0 by the first case; T(E+u-d,2)=(E+u-d,2,E+u) has colours (0,0,chi(E+u)), forcing chi(E+u)=1. The two sub-cases of p(u)=0 cover [1,2d+2].

## Lemma U (translation lemma, 2-adic input)

**Statement.** In the setting of Lemma P, let r = a mod 2d in [1,2d] and eps := p(d+1). There exists u in [1,2d] with p(u+r) = eps and p(u) = 1 - eps.

**Proof.** Suppose not; view p on Z/2d and let P := { v : p(v) = eps }. The failure means P - r ⊆ P; translation by r is a bijection of Z/2d, so |P-r| = |P| and P - r = P. Iterating, P is invariant under the subgroup generated by r, which is gamma*(Z/2d) with gamma = gcd(r,2d). By Lemma V (the project's 2-adic lemma), gamma | d, so d lies in that subgroup and P - d = P, i.e. v in P iff v+d in P. Now d+1 in P since p(d+1) = eps; hence (d+1)+d = 2d+1 ≡ 1 in P, i.e. p(1) = eps. But antisymmetry gives eps = p(d+1) = 1 - p(1), so p(1) = 1 - p(1) — absurd. Note this argument needs NO case hypothesis and no knowledge of chi(1). Verified exhaustively over every antisymmetric pattern p with p(2)=0 for all residue classes of a, d <= 12 (216 (a,d) classes, zero failures).

## Lemma A (endgame, Case A)

**Statement.** In the setting of Lemmas P, W (Case A: s** exists), F, U: chi is not valid — the triple (a+u, d+1, E+u), with u from Lemma U, is a monochromatic solution of colour eps = p(d+1).

**Proof.** It is a solution: (a+u) + a(d+1) - (E+u) = a(d+2) - d(a+1) = 2a - d. Ranges: x1 = a+u <= a+2d <= X**-2 <= N (Lemma W); x2 = d+1 <= N; x3 = E+u <= E+2d = N-2, and E+u >= E+1. Colours: chi(a+u) = p(a+u) by the window lemma (a+u <= X**), and p(a+u) = p(u+r) by 2d-periodicity since a ≡ r (mod 2d); by choice of u this equals eps. chi(d+1) = p(d+1) = eps (d+1 <= 2d+2, Lemma P(c)). chi(E+u) = 1 - p(u) = 1 - (1-eps) = eps (Lemma F, u <= 2d <= 2d+2). All three entries have colour eps — monochromatic, contradicting validity.

## Lemma B (endgame, Case B: ladder and arcs)

**Statement.** In the setting of Lemma P, suppose s** does not exist, i.e. p(s)=0 for all s in [3, d+1] (Case B). Then chi(1)=1 and chi(s)=0 for all s in [2, d+1]; and chi is not valid: there are s, s' in [2, d+1] with s+a+d ≡ s' (mod 2d), the ladder forces chi(s+a+d) = 0, and the triple (s, 3, s+a+d) is a monochromatic solution of colour 0.

**Proof.** Case B data: for s in [3,d], p(s)=phi(s)=chi(s)=0; p(d+1)=1-phi(1)=0 forces chi(1)=1; then the pairing (Lemma P(b), s=1) gives chi(d+1)=1-chi(1)=0; chi(2)=0 by normalization. So chi(1)=1, chi([2,d+1])=0 — in particular chi(3)=0 (for d=2, 3=d+1). Three pair rules, each one triple family: R0: no x with chi(x)=chi(x+d)=0, via T(x,2)=(x,2,x+d) and chi(2)=0; R1: no x with chi(x)=chi(x+a-d)=1, via T(x+a-d,1)=(x+a-d,1,x) and chi(1)=1; R3: no x with chi(x)=chi(x+a+d)=0, via T(x,3)=(x,3,x+a+d) and chi(3)=0. U-move: if chi(x)=0 and x <= N-a-d, then T(x,3) with colours (0,0,·) forces chi(x+a+d)=1, and then T(x+a+d,1)=(x+a+d,1,x+2d) with colours (1,1,·) forces chi(x+2d)=0 [ranges: x+a+d <= N; x+2d >= 1]. Fill: iterating U from each seed chi(s)=0, s in [2,d+1]: chi(x)=0 for every x ≡ s (mod 2d) with s <= x <= N-a+d (each rung x satisfies x-2d <= N-a-d). Arcs: S := [2,d+1] mod 2d and S+(a+d) are two arcs of d consecutive residues in Z/2d; two such arcs are disjoint iff the shift is ≡ d (mod 2d), i.e. iff a ≡ 0 (mod 2d) — excluded by Lemma V (r != 2d). Hence there are s, s' in [2,d+1] with s+a+d ≡ s' (mod 2d). Since s' <= d+1 < s+a+d, the multiplicity K = (s+a+d-s')/2d is a nonnegative integer, and s+a+d <= (d+1)+a+d <= N-a+d [equivalent to (d-2)a + 2d + 1 >= 0, true for d >= 2], so the fill gives chi(s+a+d)=0. Then T(s,3)=(s,3,s+a+d) is a solution [s+3a-(s+a+d)=2a-d] with colours (chi(s),chi(3),chi(s+a+d)) = (0,0,0) — monochromatic, contradicting validity. Constructive existence of (s,s',K) and all ranges machine-verified for all admissible residue classes d<=10; the whole Case B independently confirmed refutable by pure unit propagation on 108 instances d in [2,6], a <= 35.

## Theorem T2 (bottom-band pin source, t>=2)

**Statement.** Let d >= 2, a >= d+1 with v2(a) <= v2(d). Then every 2-colouring of [1, d(a+3)+2] contains a monochromatic solution of x1 + a*x2 - x3 = 2a-d. Hence Rad2(a; 2a-d) <= d(a+3)+2.

**Proof.** Suppose chi is a valid colouring of [1,N], N = d(a+3)+2. By colour swap normalize chi(2)=0. Lemma P yields the pairing pattern p on [1,2d+2] with p(2)=0. Either s** = min{s in [3,d+1]: p(s)=1} exists — then Lemmas W, F, U, A produce the monochromatic triple (a+u, d+1, E+u), contradiction — or p vanishes on [3,d+1] — then Lemma B produces the monochromatic triple (s, 3, s+a+d), contradiction. The two cases are exhaustive by definition. Hence no valid colouring exists. Cross-checked against exact SAT-computed values Rad2(a;2a-t) for every existing pair with a <= 20 and spot checks to a = 40: the bound holds with margin 6..225 for t >= 2 (the true value is near t(a+1)), and the theorem's hypotheses (d >= 2, a >= d+1, v2(a) <= v2(d)) are exactly the admissible range.

## Theorem G2 (the deep-tail pin) + sharp onset

**Statement.** Let a >= 2, c > a with a not dividing c and v2(a) <= v2(c); put lam = ceil(c/a), t = c mod a in [1,a-1]. (i) If t >= 2 and lam >= t(a+3)+2, or t = 1 and lam >= 2a-2, then every 2-colouring of [1, lam] contains a monochromatic solution of x1+a*x2-x3 = c; i.e. Rad2(a;c) <= lam. (ii) Unconditionally in lam: Rad2(a;c) <= lam if and only if lam >= Rad2(a; 2a-t). So the exact onset of the pin is lam = Rad2(a;2a-t), which is <= the deep-tail threshold, with equality precisely in the t=1 family (where Rad2(a;2a-1) = 2a-2).

**Proof.** By Lemma V, v2(a) <= v2(c) implies v2(a) <= v2(t) (so a is odd when t=1) and Rad2(a;2a-t) exists. By Theorem T1 (t=1) or T2 (t>=2, d=t), Rad2(a; 2a-t) <= 2a-2 resp. t(a+3)+2 <= lam. Hence [1, Rad2(a;2a-t)] admits no valid colouring for E(2a-t), and by restriction (monotonicity) neither does [1, lam]. By Lemma R with N = lam (c_hat = a(lam+1)-c = 2a-t), [1, lam] admits no valid colouring for E(c) either, i.e. Rad2(a;c) <= lam. This proves (i). For (ii): Lemma R gives [1,lam] E(c)-colorable <=> [1,lam] E(2a-t)-colorable <=> lam <= Rad2(a;2a-t) - 1; negating both sides gives the equivalence. Combined with draft_L2-tail's reduction — whose input G2 was exactly statement (i) — the deep-tail closed form Rad2(a;c) = lam - 1 + [lam <= E1] is now a theorem modulo the inputs draft_L2 already lists as proven (DT Theorem 6 for the lower bound; E1 = Rad2(a; a-t) from nota Theorems 1-2). Structurally, the deep tail is governed by exactly two bottom-band values of the same equation: Rad2(a; a-t) = E1 decides WHERE in {lam-1, lam} the value sits (duality at N = lam-1), and Rad2(a; 2a-t) decides WHEN the pin bites (duality at N = lam).
