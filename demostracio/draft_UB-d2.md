# Draft: UB-d2

**status**: complete

## Summary

PROVED: the upper bound of the conjecture for d=2. For every a >= 3 with v2(a) <= v2(a-2) (equivalently r = a mod 4 in {1,2,3}), every 2-coloring of [1,M] with M = 3a+6-min(r-1,1) (i.e. M = 3a+6 if r=1, M = 3a+5 if r in {2,3}; this equals the conjectured value (a+3)*2 + a - min(r-1,1)) contains a monochromatic triple (x,k,x+a(k-1)+2). Hence Rad2(a;a-2) <= M, matching the mechanically verified lower-bound colorings of length M-1, so Rad2(a;a-2) = M for all existing a. Proof method: after normalizing chi(1)=0, only multiplier values k in {1,2,3,4} can occur inside [1,M]; the triple (1,1,3) forces chi(3)=1, and the proof branches on chi(2). Branch A (chi(2)=1) is killed using only k=1 and k=2 triples: a forcing wave 2 -> a+4 -> 2a+4 -> 3a+4 reflects at the top (3a+2 forces 2a and 2a-2), lands at chi(a-4)=0 / chi(a-3)=0, and a descent with step -4 drives a forced 0 down to position 6 or 7, where one final k=2 triple collides with the wave's own forced values near a+6 -- a contradiction that already materializes below M (consistent with the computed fact that ALL extremal colorings of [1,M-1] have chi(2)=0). Branch B (chi(2)=0) forces chi(4)=1, and the single k=4 triple ending exactly at M forces chi(M)=0; a zig-zag cascade alternating k=1 triples (distance 2) and k=3 triples (distance 2a+2) then drives forced zeros down the residue class of a+1 (or a+2) modulo 4 until it forces chi(3)=0 (r in {1,2}) or chi(4)=0 (r=3), a contradiction. Every derivation step of every lemma was machine-verified (triple arithmetic, range bounds, cited facts, and the exact terminal step of the contradiction) for all 298 admissible a in [3,399], the constraint model was cross-validated against the repo checker on 64,000 random colorings, and the theorem itself was confirmed by exhaustive search at N=M for 13 values of a. No gaps remain for the d=2 upper bound.

## Lemma 1 (Setup, admissibility, and constraint inventory)

**Statement.** Fix d=2, c=a-2, a>=3. (0) Rad2(a;a-2) exists iff a is not divisible by 4; write r = a mod 4 in {1,2,3} and q = floor(a/4), so a=4q+r, and the admissible a are: a>=5 when r=1, a>=6 when r=2, a>=3 when r=3. Define M = 3a+6 if r=1 and M = 3a+5 if r in {2,3}; this equals the conjectured value (a+3)d + a - min(r-1,d-1) for d=2. Let chi:[1,M]->{0,1} be a coloring with NO monochromatic solution of x1 + a*x2 - x3 = a-2, normalized (WLOG) so that chi(1)=0. Then: (a) the solutions with all entries in [1,M] are exactly the triples T_k(x) = (x, k, x+a(k-1)+2) with k in {1,2,3,4} and 1 <= x <= M-a(k-1)-2; explicitly the x-ranges are: k=1: x <= M-2; k=2: x <= M-a-2, which equals 2a+4 (r=1) or 2a+3 (r in {2,3}); k=3: x <= M-2a-2, which equals a+4 (r=1) or a+3 (r in {2,3}); k=4: x <= M-3a-2, which equals 4 (r=1) or 3 (r in {2,3}). (b) Blocking rule: if in a triple T_k(x) two of the three coordinates (counted as a multiset of positions {x, k, x+a(k-1)+2}) are known to have the same color v, then the third has color 1-v. (c) chi(3)=1. (d) For every u with chi(u)=0: chi(u-2)=1 whenever u-2>=1, and chi(u+2)=1 whenever u+2<=M.

**Proof.** (0) The general existence criterion is v2(a) <= v2(a-d); here d=2. If a is odd, v2(a)=0 <= v2(a-2), so Rad2 exists. If a == 2 (mod 4), then v2(a)=1 while a-2 == 0 (mod 4) gives v2(a-2)>=2, so Rad2 exists. If a == 0 (mod 4), then v2(a)>=2 while a-2 == 2 (mod 4) gives v2(a-2)=1 < v2(a), so Rad2 does not exist. The zone 0 < c < a with c=a-2 requires a>=3; combined with the residue classes this gives a>=5 (r=1), a>=6 (r=2), a>=3 (r=3). For d=2: (a+3)d + a - min(r-1,1) = 3a+6 - min(r-1,1), which is 3a+6 for r=1 and 3a+5 for r in {2,3}. (WLOG) The property 'chi has a monochromatic solution' is invariant under swapping the two colors, so we may assume chi(1)=0. (a) A solution of x1 + a*x2 - x3 = a-2 with x2=k has x3 = x1 + a(k-1) + 2, so solutions are exactly the triples T_k(x); note x3 > x1 always, and x3 > k: for k=1, x3=x+2>=3>1; for k in {2,3,4}, x3 >= 1+a(k-1)+2 >= a+3 > 4 >= k since a>=3. If k>=5 then x3 >= 1 + 4a + 2 = 4a+3; and 4a+3 > M in all cases: for r=1 (a>=5), 4a+3-(3a+6) = a-3 >= 2 > 0; for r in {2,3} (a>=3), 4a+3-(3a+5) = a-2 >= 1 > 0. Hence no triple with k>=5 fits inside [1,M], and for k<=4 the triple fits iff 1 <= x <= M-a(k-1)-2. The stated values of the ranges are direct arithmetic: M-a-2 = 2a+4 or 2a+3; M-2a-2 = a+4 or a+3; M-3a-2 = 4 or 3. (b) A triple T_k(x) is monochromatic iff chi(x)=chi(k)=chi(x+a(k-1)+2) (this reads correctly also when x=k, in which case being monochromatic means the two distinct positions agree). If two of the coordinates carry color v and the third carried v as well, T_k(x) would be monochromatic, contradicting the hypothesis on chi; so the third carries 1-v. (c) The triple T_1(1) = (1,1,3) has coordinates {1,1,3}; chi(1)=0 at both occurrences, so by (b) chi(3)=1. (d) The triple T_1(u-2) = (u-2,1,u) has chi(1)=0 and chi(u)=0, so chi(u-2)=1 by (b) (range: 1 <= u-2 and u <= M). Similarly T_1(u) = (u,1,u+2) has chi(u)=0 and chi(1)=0, forcing chi(u+2)=1 (range: u+2 <= M).

## Lemma 4 (Branch A, descent and contradiction, a >= 9)

**Statement.** Let chi be as in Lemma 1 with chi(2)=1 and a>=9. Set (v0, v*) = (a-3, 6) if r=1; (a-4, 6) if r=2; (a-4, 7) if r=3. Then v0 == v* (mod 4), v0 >= v*, chi(v)=0 for every v in D := {v* , v*+4, ..., v0}, and a contradiction follows. Consequently Branch A (chi(2)=1) is impossible for every admissible a (combining with Lemma 2).

**Proof.** Congruence and size: r=1: v0=a-3=4q-2 == 2 == 6 (mod 4), and v0>=6 iff a>=9: true. r=2: v0=a-4=4q-2 == 2 == 6 (mod 4), v0>=6 iff a>=10: true since a>=9 and a==2 (mod 4). r=3: v0=a-4=4q-1 == 3 == 7 (mod 4), v0>=7 iff a>=11: true since a>=9 and a==3 (mod 4). Downward induction on v in D. Base v=v0: chi(v0)=0 is fact V9 (r=1) or W9 (r in {2,3}) of Lemma 3. Step: assume chi(v)=0 with v in D, v>v*; then v>=v*+4>=10. [S1] chi(v-2)=1 by Lemma 1(d) at u=v (v-2>=8>=1). [S2] chi(v+a)=0 by T_2(v-2)=(v-2,2,v+a): cited x-coordinate v-2 (color 1 by S1) and k=2 (color 1); ranges: x=v-2<=v0-2<=a-5<=M-a-2 (indeed a-5<=2a+3 always), and z=v+a<=v0+a<=2a-3<=M. [S3] chi(v+a-2)=1 by Lemma 1(d) at u=v+a (v+a-2>=1). [S4] chi(v-4)=0 by T_2(v-4)=(v-4,2,v+a-2): cited k=2 and z-coordinate v+a-2, both color 1 (by hypothesis chi(2)=1 and S3); ranges: x=v-4>=v*>=6>=1, and z=(v-4)+(a+2)=v+a-2<=M. Since v-4 in D, the induction proceeds down to v*. Terminal contradiction: from chi(v*)=0, Lemma 1(d) gives chi(v*-2)=1 (v*-2 in {4,5}). Consider the triple T_2(v*-2)=(v*-2, 2, v*+a); its z-coordinate is v*+a = a+6 (r in {1,2}) or a+7 (r=3), and z=(v*-2)+(a+2) confirms the arithmetic; ranges: x=v*-2<=5<=M-a-2 and z<=a+7<=M (a+7<=3a+5 iff a>=1). Together with Lemma 2 (a in {3,5,6,7}) this rules out chi(2)=1 for every admissible a: indeed every admissible a is in {3,5,6,7} or satisfies a>=9. (Machine-verified for all admissible a in [3,399]: the contradiction fires at exactly this terminal step, and at the exact steps stated in Lemma 2 for a in {3,5,6,7}.)

## Lemma 5 (Branch B, boundary forcing)

**Statement.** Let chi be as in Lemma 1 (so chi(1)=0, chi(3)=1) and suppose chi(2)=0. Then: (a) chi(4)=1; (b) chi(M)=0; (c) chi(M-2)=1, where M-2 = 3a+4 (r=1) or 3a+3 (r in {2,3}); (d) chi(u0)=0 where u0 := a+2 if r=1 and u0 := a+1 if r in {2,3}. Moreover, if a=3, part (d) is already a contradiction, because u0=4 while chi(4)=1 by (a); hence Branch B is impossible for a=3.

**Proof.** (a) The triple T_1(2)=(2,1,4) has chi(2)=0 and chi(1)=0, so by the blocking rule chi(4)=1 (range: 4<=M). (b) Case r=1 (M=3a+6): the triple T_4(4)=(4,4,3a+6) is inside [1,M] because the k=4 x-range bound is M-3a-2=4, and its z-coordinate is 4+3a+2=3a+6=M; its x- and k-coordinates are both position 4 with chi(4)=1, so chi(M)=0. Case r in {2,3} (M=3a+5): the triple T_4(3)=(3,4,3a+5) is inside [1,M] because M-3a-2=3, with z=3+3a+2=3a+5=M; its coordinates 3 and 4 have chi(3)=1 and chi(4)=1, so chi(M)=0. (c) By Lemma 1(d) at u=M: chi(M-2)=1 (M-2>=1). (d) Case r=1: the triple T_3(a+2)=(a+2,3,3a+4) has z=(a+2)+(2a+2)=3a+4=M-2; range: x=a+2 <= M-2a-2=a+4. Its k-coordinate 3 and z-coordinate M-2 both have color 1 (by Lemma 1(c) and part (c)), so chi(a+2)=0. Case r in {2,3}: the triple T_3(a+1)=(a+1,3,3a+3) has z=(a+1)+(2a+2)=3a+3=M-2; range: x=a+1 <= M-2a-2=a+3. Again chi(3)=1 and chi(3a+3)=1 force chi(a+1)=0. Final claim: for a=3 (r=3), u0=a+1=4, so (d) says chi(4)=0 while (a) says chi(4)=1 -- equivalently, the triple T_3(4)=(4,3,12) would be monochromatic in color 1 (chi(4)=1, chi(3)=1, chi(12)=chi(3a+3)=1). ## Lemma 6 (Branch B, zig-zag cascade and contradiction)

**Statement.** Let chi be as in Lemma 1 with chi(2)=0, and let a>=5 (so q=floor(a/4)>=1). With u0 as in Lemma 5(d), define u_j = u0 - 4j. Consequently Branch B (chi(2)=0) is impossible for every admissible a (combining with Lemma 5 for a=3).

**Proof.** First the arithmetic of the endpoint: r=1: u0=a+2=4q+3, so u_q=u0-4q=3. r=2: u0=a+1=4q+3, so u_q=3. r=3: u0=a+1=4q+4, so u_q=4. Also for 0<=j<=q-1 we have u_j >= u_{q-1} = u_q+4 in {7,8}. Induction on j; base j=0 is Lemma 5(d). Step (0<=j<=q-1), assuming chi(u_j)=0 with u_j>=7: [L1] chi(u_j-2)=1 by Lemma 1(d) at u=u_j (u_j-2>=5>=1). [L2] chi(u_j+2a)=0 by the k=3 triple T_3(u_j-2)=(u_j-2, 3, u_j+2a): indeed z=(u_j-2)+(2a+2)=u_j+2a; its x-coordinate u_j-2 has color 1 (L1) and its k-coordinate 3 has color 1, forcing chi(u_j+2a)=0. Ranges: x=u_j-2 <= u0-2 <= a <= M-2a-2 (since M-2a-2 >= a+3), and z=u_j+2a <= u0+2a <= 3a+2 <= M, and x>=5>=1. [L3] chi(u_j+2a-2)=1 by Lemma 1(d) at u=u_j+2a. [L4] chi(u_j-4)=0 by the k=3 triple T_3(u_j-4)=(u_j-4, 3, u_j+2a-2): z=(u_j-4)+(2a+2)=u_j+2a-2; its k-coordinate 3 and z-coordinate u_j+2a-2 both have color 1 (Lemma 1(c) and L3), forcing chi(u_j-4)=0. Ranges: x=u_j-4>=u_q>=3>=1 and z<=M as in L2. Since u_j-4=u_{j+1}, the induction closes, and after the step with j=q-1 we obtain chi(u_q)=0. Contradiction: for r in {1,2}, u_q=3 and chi(3)=1 (Lemma 1(c)) -- explicitly, the final triple T_3(3)=(3,3,2a+5) is monochromatic in color 1, since its coordinates are the position 3 twice (chi(3)=1) and the position u_{q-1}+2a-2=2a+5 with color 1 by the last L3. For r=3, u_q=4 and chi(4)=1 (Lemma 5(a)) -- explicitly, the final triple T_3(4)=(4,3,2a+6) is monochromatic in color 1 (chi(4)=1, chi(3)=1, and chi(2a+6)=1 by the last L3, since u_{q-1}=8 gives u_{q-1}+2a-2=2a+6). Together with Lemma 5's a=3 case, Branch B is impossible for every admissible a. (Machine-verified for all admissible a in [3,399]: every L-step is a valid in-range triple citing established facts, no position is ever assigned twice inconsistently before the terminal step, and the contradiction fires exactly at the last L4.)

## Theorem (Upper bound of the conjecture for d=2)

**Statement.** Let a>=3 with v2(a) <= v2(a-2), i.e. r = a mod 4 in {1,2,3}, and let M = 3a+6 - min(r-1,1) = (a+3)*2 + a - min(r-1, 2-1) be the conjectured value of Rad2(a; a-2). Then EVERY 2-coloring of [1,M] contains a monochromatic triple (x, k, x+a(k-1)+2), i.e. a monochromatic solution of x1 + a*x2 - x3 = a-2. Hence Rad2(a; a-2) <= M; combined with the (previously and mechanically verified) validity of the extremal family coloring of [1, M-1], Rad2(a; a-2) = (a+3)*2 + a - min(r-1,1) for every existing a, establishing the conjecture for d=2.

**Proof.** Suppose, for contradiction, that chi:[1,M]->{0,1} has no monochromatic triple. Since the property is invariant under swapping colors, assume WLOG chi(1)=0. By Lemma 1, chi(3)=1, and every solution triple inside [1,M] is T_k(x) with k<=4, so the blocking rule of Lemma 1(b) applies to exactly the triples enumerated there. Branch on chi(2). If chi(2)=1: for a in {3,5,6,7}, Lemma 2 produces a monochromatic triple; for a>=9 (which together with a in {3,5,6,7} covers all admissible a, since the admissible values are 3,5,6,7,9,10,11,13,...), Lemmas 3 and 4 produce a monochromatic triple. Contradiction. If chi(2)=0: for a=3, Lemma 5 produces a monochromatic triple; for a>=5, Lemmas 5 and 6 do. Contradiction. Hence no such chi exists and Rad2(a;a-2) <= M. The reverse inequality Rad2(a;a-2) >= M is given by the extremal family coloring of [1,M-1] (the family stated in the project description, mechanically verified valid for all existing pairs with d=2 in the tested range, and re-verified here by independent exhaustive search which found these colorings among the survivors at length M-1); therefore Rad2(a;a-2) = M = (a+3)*2 + a - min(r-1,1).
